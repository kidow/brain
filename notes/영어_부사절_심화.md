## as의 다중 용법 — 시간 / 이유 / 비례 / 방식

as 하나로 4가지 부사절을 만듦. 문맥으로 구분.

- 시간: As the sun set, the temperature dropped. (동시 발생)
- 이유: As it was raining, we stayed inside. (알려진 배경)
- 비례: As you get older, time passes faster. (함께 변화)
- 방식: Do it as I showed you. (방법 지정)

시간 vs 비례 구분:
- 시간 = 특정 순간 단발 이벤트 ("그때")
- 비례 = 두 변수가 함께 점진적으로 변화 (그래프 두 선)
- 판별: "~할수록"으로 번역되면 비례, 아니면 시간

---

## 시간 부사절 — 세밀한 의미 차이

| 접속사 | 초점 | 예 |
|---|---|---|
| when | 단순 시점 | When he arrived, we started. |
| while | 진행 중인 배경 | While I was sleeping, she called. |
| as | 찰나의 동시 | As I opened the door, the phone rang. |
| once | 전환점 이후 | Once you understand it, it becomes easy. |
| by the time | 기준 시점 이전에 완료 | By the time he arrived, we had finished. |

by the time: 결과절에 완료 시제(had p.p. / will have p.p.) 자연스러움.

---

## 조건 부사절 — if 너머의 조건 표현

| 접속사 | 뉘앙스 | 예 |
|---|---|---|
| unless | 부정 조건 (= if not) | Unless you hurry, you'll be late. |
| as long as | ~하는 한 (제한) | Stay as long as you're quiet. |
| provided (that) | 격식 조건 | You can borrow it, provided you return it. |
| in case | 예방 (지금 행동) | Take an umbrella in case it rains. |
| suppose / supposing | 가설 (구어) | Suppose it rains — should we cancel? |

if = 중립 조건. 나머지는 각각 부정·제한·격식·예방·가설 뉘앙스 추가.

---

## 양보 부사절 — "~임에도 불구하고" 세분화

**사실 양보** (실제로 그러함):
- although (격식) / though (비격식) / even though (강조)
- Although it was cold, she went out.

**가상 양보** (사실인지 모름):
- even if — Even if it rains, I'll go. (올지 모르지만 와도)

even though vs even if: though = 사실 인정 / if = 가상·불확실

**기타:**
- whether or not: Whether or not you agree, we're moving forward.
- however/whatever + 복합관계사: However hard it is, don't give up.

---

## 이유 부사절 — because / since / as / for

| | because | since | as | for |
|---|---|---|---|---|
| 강도 | 강 | 중 | 약 | 중 |
| 위치 | 앞·뒤 | 주로 앞 | 주로 앞 | 반드시 뒤 |
| 격식 | 중립 | 중립 | 비격식 | 격식·문어 |

- because: 이유가 핵심 정보 ("Why?" 답할 때)
- since: 알려진 배경 이유
- as: since보다 가볍고 구어적
- for: 주절 보충 설명, 절 앞 불가

since vs as: 대부분 교환 가능. since는 시간("~이후로")과 이중 의미 가능 → 모호할 때 because로 대체.

---

## 목적 부사절 — so that / in order that / lest

- so that + 조동사: She studied so that she could pass. (구어·격식)
- in order that: 격식·문어체 버전
- in order to / so as to: 주어 같을 때 부정사로 축약
- lest + should + 원형: "~하지 않도록" (부정 목적, 격식·문어, 현대에 드묾)
  - He whispered lest anyone should hear. → 구어: so that nobody would hear

---

## 결과 부사절 — so...that / such...that

"너무 ~해서 …하다"

- so + 형용사/부사 + that: She was so tired that she fell asleep.
- such + (a/an) + 형용사 + 명사 + that: It was such a good movie that we watched it twice.

so vs such: so 뒤 = 형용사·부사 단독 / such 뒤 = (형용사+)명사
- so kind / such a kind woman

that절 구어에서 생략 가능. so much/many/little/few + that도 동일 패턴.

---

## 방식 부사절 — as / as if / as though

**as** — 실제 방식 "~대로, ~처럼"
- Do it as I showed you.
- Leave things as they are.

**as if / as though** — 비현실 "마치 ~인 것처럼" (가정법 시제)
- He talks as if he knew everything. (사실은 모름 — 2차)
- She looked as if she had seen a ghost. (본 적 없음 — 3차)

as vs as if: as = 실제 방식(직설법) / as if = 비현실(가정법)
구어에서 실제 가능성 있을 때 직설법도 허용: It looks as if it's going to rain.
