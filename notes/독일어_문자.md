# 독일어 문자

## 움라우트 세 글자: ä, ö, ü

독일어에만 있는 특수 모음 3개. 글자 위에 점 두 개(¨)가 표시.

| 글자 | 소리 | 예시 |
|------|------|------|
| ä | "에"처럼 [ɛː] | Mädchen (소녀) → "메트헨" |
| ö | "에" 입모양 + 입술 오므리기 [øː] | schön (아름다운) → "쇤" |
| ü | "이" 입모양 + 입술 오므리기 [yː] | über (~위에) → "위버" |

- ö, ü는 한국어에 없는 소리 — 입술 모양으로 만듦
- 점 두 개 못 쓸 때: ä→ae, ö→oe, ü→ue (예: Müller → Mueller)

## ß (에스체트, Eszett)

- 발음: "ss"와 동일 [s]. 특별한 소리 아님.
- 기원: 중세 필기체의 장s(ſ) + z 합자(ligature). 필기 편의에서 탄생.
- 사용 규칙: 긴 모음·이중모음 뒤 → ß / 짧은 모음 뒤 → ss
  - Straße (거리), groß (크다), weiß (하얗다) → ß
  - Wasser (물) → ss
- 대문자: SS 또는 ẞ (2017년 공식 추가)
- 스위스 독일어는 ß 미사용, 항상 ss로 대체

## 영어와 다른 알파벳 읽기

26자는 동일하지만 이름(읽기)이 다름. 핵심 함정 6개:

| 글자 | 독일어 이름 | 실제 소리 | 예시 |
|------|-----------|---------|------|
| W | 베(veh) | 영어 'v' | Wasser(물) = "바써" |
| V | 파우(fau) | 영어 'f' | Vater(아버지) = "파터" |
| Z | 체트(zet) | 'ts' | Zeit(시간) = "차이트" |
| J | 요트(jot) | 영어 'y' | Jahr(년) = "야어" |
| Q | 쿠(ku) | 'kv' | qu = 항상 kv |
| S | 에스(es) | 모음 사이서 'z' | — |

- W·V·Z는 영어 지식이 오히려 방해됨 — 반드시 새로 덮어써야 함
- V는 외래어에선 영어처럼 'v' 소리 유지

## ch 발음 — 두 가지

앞에 오는 모음에 따라 소리가 갈림.

| 조건 | 소리 | 설명 | 예시 |
|------|------|------|------|
| a·o·u·au 뒤 | [x] 뒤쪽 ch | 목 안쪽 긁기. 한국어에 없는 소리. | Bach(시내), Buch(책), Nacht(밤) |
| e·i·ä·ö·ü·ei·eu 뒤 | [ç] 앞쪽 ch | 혀 앞쪽 마찰. 'ㅎ'에 가까움. | ich(나), Mädchen(소녀), Licht(빛) |

- 기억법: **a·o·u → 목 뒤 / 나머지 → 목 앞**
- 앞쪽 ch = 'ㅎ'으로 근사 가능 (통함)
- 뒤쪽 ch = 'ㅎ'보다 훨씬 거침. 처음엔 ㅎ으로 시작하고 나중에 교정.
- 단어 시작 ch: 대부분 [ç] 또는 [k] (Chemie=케미, Charakter=카락터)

## ei와 ie — 뒤 글자를 길게 읽는다

독일어에서 **ei**와 **ie**는 글자 순서가 비슷하지만 소리가 완전히 다르다. 핵심 규칙은 **뒤에 있는 모음 이름 쪽으로 읽는다**는 것.

| 철자 | 읽기 | 예시 |
|------|------|------|
| ei | 아이 [aɪ] | Wein(포도주) = "바인" |
| ie | 이 [iː] | Bier(맥주) = "비어" |

한국어식으로 기억하면 `ei`는 "에이"가 아니라 **아이**, `ie`는 "이에"가 아니라 **긴 이**다.

- mein(나의) = "마인"
- Liebe(사랑) = "리베"
- viel(많은) = "필"

핵심: **ei는 아이, ie는 이. 독일어에서는 이 두 철자를 절대 영어식으로 읽지 않는다.**

## eu와 äu — 둘 다 오이처럼 읽는다

독일어에서 **eu**와 **äu**는 철자가 다르지만 발음은 같다. 둘 다 대략 **오이** [ɔʏ]처럼 읽는다.

| 철자 | 읽기 | 예시 |
|------|------|------|
| eu | 오이 [ɔʏ] | Deutsch(독일어) = "도이치" |
| äu | 오이 [ɔʏ] | Häuser(집들) = "호이저" |

`äu`는 보통 단어의 변화형에서 나온다. 예를 들어 `Haus`(집)가 복수형이 되면 `Häuser`가 되고, 이때 `au`가 `äu`로 바뀌지만 소리는 **오이**가 된다.

- neun(아홉) = "노인"
- heute(오늘) = "호이테"
- Bäume(나무들) = "보이메"

핵심: **eu = äu = 오이. `äu`를 "애우"처럼 나눠 읽지 않는다.**

## sch — 한 덩어리로 슈/쉬

독일어의 **sch**는 `s + c + h` 세 글자를 따로 읽지 않고, 한 덩어리로 **sh** [ʃ] 소리를 낸다. 한국어로는 대략 **슈/쉬**에 가깝다.

| 철자 | 읽기 | 예시 |
|------|------|------|
| sch | 슈/쉬 [ʃ] | Schule(학교) = "슐레" |
| sch | 슈/쉬 [ʃ] | schön(아름다운) = "쇤" |

영어의 `school`처럼 `s`와 `k`가 나뉘는 느낌이 아니다. 독일어 `Schule`은 **스쿨레**가 아니라 **슐레**다.

- schreiben(쓰다) = "슈라이벤"
- Fisch(물고기) = "피쉬"
- Mensch(사람) = "멘쉬"

핵심: **sch는 언제나 한 소리 [ʃ]. 세 글자지만 발음은 한 덩어리다.**
