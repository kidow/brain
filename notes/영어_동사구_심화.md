## 구동사 분리 가능 vs 불가 (Separable vs Inseparable)

**분리 가능**: 목적어가 동사와 입자 사이에 들어갈 수 있음.
- turn off the light → turn the light off ✅
- 대명사는 반드시 가운데: turn it off ✅ / turn off it ❌

**분리 불가**: 목적어는 반드시 뒤.
- look after the kids ✅ / look the kids after ❌
- run into a friend ✅ / run a friend into ❌

대표 분리 불가: look after / look into / look for / run into / come across / get over / go through

불확실할 때는 목적어를 뒤에 두는 것이 안전.

---

## 3단어 구동사 (Three-part Phrasal Verbs)

동사 + 입자 + 전치사. 항상 분리 불가, 목적어 반드시 맨 뒤.

- put up with: ~을 참다 (I can't put up with this noise.)
- look forward to: ~을 기대하다 (+ 동명사: look forward to seeing)
- run out of: ~이 떨어지다
- come up with: ~을 생각해내다
- get along with: ~와 잘 지내다
- look up to: ~을 존경하다
- look down on: ~을 무시하다

## 자동사형 구동사 — 목적어 없는 구동사

- break down: 고장 나다 / 무너지다
- give in: 항복하다
- show up / turn up: 나타나다
- break out: 발생하다 (전쟁·화재·질병)
- go off: 폭발하다 / 알람 울리다
- set off: 출발하다
- fall apart: 무너지다

같은 구동사가 자/타동사 둘 다 가능:
- give up smoking (타동사) vs she gave up (자동사)
- turn up the music (타동사) vs he turned up late (자동사)

---

## 같은 동사, 다른 입자 — get / look 계열

**get 계열:**
- get up(일어나다) / get over(극복·회복) / get along(잘 지내다) / get away(도망·떠나다)
- get through(통과·연락) / get back(돌아오다·되찾다) / get into(빠지다·탑승) / get off(내리다)

**look 계열:**
- look up(찾아보다·상황 나아지다) / look into(조사하다) / look after(돌보다) / look for(찾다)
- look forward to(기대하다) / look down on(무시하다) / look up to(존경하다)

학습 전략: 동사 하나씩 묶어 공략 — get → look → take → put 순.

---

## 구동사 vs 단일 동사 — 격식 차이

같은 의미지만 구동사 = 구어·비격식, 단일 동사 = 격식·문어.

| 구동사 | 단일 동사 |
|---|---|
| give up | abandon / quit |
| find out | discover |
| look into | investigate |
| put off | postpone |
| come up with | devise / propose |
| turn down | reject / decline |
| carry out | execute / implement |

이메일·대화 → 구동사 OK. 논문·계약서 → 단일 동사 선호.

## take / put 계열

**take:**
- take off: 이륙하다 / 옷 벗다 / 급성장하다
- take on: 떠맡다 / 고용하다
- take over: 인수하다
- take out: 꺼내다 / 포장하다
- take up: 공간·시간 차지하다 / 취미 시작하다
- take in: 받아들이다 / 이해하다 / 속이다

**put:**
- put off: 미루다
- put on: 입다 / 켜다
- put out: 끄다(불) / 출판하다
- put up with: 참다
- put down: 내려놓다 / 비하하다
- put through: 전화 연결하다 / 겪게 하다
