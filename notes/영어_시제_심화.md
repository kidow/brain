## 영어 시제 전체 지도

|  | 단순 | 진행 | 완료 | 완료진행 |
|---|---|---|---|---|
| **현재** | do | is doing | has done | has been doing |
| **과거** | did | was doing | had done | had been doing |
| **미래** | will do | will be doing | will have done | will have been doing |

심화 포커스: 완료·완료진행 계열.

---

## 현재완료의 핵심 감각 — 과거 행위, 현재 결과

`have + p.p.` = 과거 사건 + 현재 상태·결과 연결.

| 용법 | 키워드 | 예 |
|---|---|---|
| 결과 | — | I have broken my leg. (지금 부러진 상태) |
| 경험 | ever / never / before | Have you ever been to Paris? |
| 계속 | for / since | I have lived here for 10 years. |
| 완료 | just / already / yet | She has just arrived. |

yesterday / ago / last~ → 단순과거만. 현재완료와 함께 불가.
```
I saw him yesterday.        ✅
I have seen him yesterday.  ❌  (과거 시점 명시 → 현재와 단절)
```

---

## 현재완료 진행 — has been doing

`have been + -ing` = 과거부터 지금까지 계속 진행 중. 누적·지속 강조.

```
I have been waiting for an hour.     (한 시간째 기다리는 중)
She has been studying since morning. (아침부터 지금도 공부 중)
I have been crying.                  (for 없어도 자연스러움 — 누적 뉘앙스)
```

현재완료 vs 현재완료진행:
```
I have read the book.          → 다 읽었다 (완료)
I have been reading the book.  → 읽는 중이다 (아직 진행)
```

현재진행 vs 현재완료진행:
```
I am reading.           → 지금 이 순간 스냅샷
I have been reading.    → 과거부터 지금까지 누적·지속
```

상태동사(know / like / believe) → 진행형 불가 → 현재완료만 씀.
```
I have known her for years.      ✅
I have been knowing her.         ❌
```

## 과거완료 — had done

`had + p.p.` = 과거의 어떤 시점보다 더 이전에 일어난 일. "과거 속의 과거".

```
When I arrived, she had already left.
(내가 도착했을 때 → 그녀는 이미 떠난 상태)
```

비교:
```
When I arrived, she left.      → 도착하자 떠남 (동시·직후)
When I arrived, she had left.  → 이미 떠난 상태 (더 이전)
```

```
He had never seen snow before he moved to Canada.
I realized I had forgotten my wallet.
```

---

## 과거완료 진행 — had been doing

`had been + -ing` = 과거 시점 이전부터 그 시점까지 계속 진행 중이었던 동작.

```
She had been crying when I found her.
He had been working for 12 hours before he collapsed.
They had been dating for 3 years before they broke up.
```

비교:
```
I have been running.                        → 지금 기준
I had been running when it started to rain. → 비 오기 시작한 시점 기준
```

---

## 미래완료 — will have done

`will have + p.p.` = 미래 특정 시점까지 완료되어 있을 것. `by + 시점`과 자주 쓰임.

```
By next year, I will have graduated.
By the time you arrive, I will have finished cooking.
By 2030, humans will have landed on Mars.
```

| | 구조 | 완료 기준 |
|---|---|---|
| 현재완료 | `has done` | 지금 |
| 과거완료 | `had done` | 과거의 한 시점 |
| 미래완료 | `will have done` | 미래의 한 시점 |

---

## 미래완료 진행 — will have been doing

`will have been + -ing` = 미래 특정 시점까지 계속 진행 중이었을 것. 지속 기간 강조.

```
By next month, I will have been working here for 10 years.
By the time she wakes up, it will have been raining for 6 hours.
```

실용 빈도: 현재완료 > 현재완료진행 > 과거완료 > 미래완료 > 과거완료진행 > 미래완료진행(매우 낮음)
