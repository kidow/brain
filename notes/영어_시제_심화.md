## 영어 시제 전체 지도

|  | 단순 | 진행 | 완료 | 완료진행 |
|---|---|---|---|---|
| **현재** | do | is doing | has done | has been doing |
| **과거** | did | was doing | had done | had been doing |
| **미래** | will do | will be doing | will have done | will have been doing |

심화 포커스: 완료·완료진행 계열.

---

## 현재완료의 핵심 감각 — 과거 행위, 현재 결과

`have + p.p.` = 과거 사건 + 현재 상태·결과 연결.

| 용법 | 키워드 | 예 |
|---|---|---|
| 결과 | — | I have broken my leg. (지금 부러진 상태) |
| 경험 | ever / never / before | Have you ever been to Paris? |
| 계속 | for / since | I have lived here for 10 years. |
| 완료 | just / already / yet | She has just arrived. |

yesterday / ago / last~ → 단순과거만. 현재완료와 함께 불가.
```
I saw him yesterday.        ✅
I have seen him yesterday.  ❌  (과거 시점 명시 → 현재와 단절)
```
