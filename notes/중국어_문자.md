# 중국어 문자

## 한 글자 = 한 음절 = 한 뜻 (표어문자)

중국어 한자(漢字)는 **표어문자**(logogram)다. 한글·알파벳 같은 표음문자와 달리, 한자 한 글자는 동시에 세 가지를 담는다:

1. **소리** — 딱 한 음절 (두 음절짜리 한자는 없다)
2. **뜻** — 의미 단위 하나
3. **모양** — 고유 형태

예시:
- `山` → `shān`, "산"
- `人` → `rén`, "사람"
- `好` → `hǎo`, "좋다"

한국 한자어(산=山, 인=人)와 같은 원리지만, 중국어는 한글 같은 표음 보조문자 없이 **한자만으로** 문장 전체를 쓴다. 상용한자 약 3,000자를 통째로 외워야 한다.

## 병음(拼音, pīnyīn) — 발음 표기 체계

한자만 봐선 발음을 알 수 없다. 그래서 중국은 1958년에 **병음(拼音, pīnyīn)** 을 제정했다. 알파벳에 성조 기호를 얹는 로마자 표기 체계다.

역할:
- 한자의 발음을 적는다 → `北京` = `Běi jīng`
- 키보드에서 한자 입력할 때 사용
- 교재에서 한자 위/옆에 발음 표시

구조:
```
성모(초성) + 운모(중·종성) + 성조부호
  b       +   ei          +  ˇ   → běi (북녘)
  j       +   īng         +  ¯   → jīng (수도)
```

병음은 한자를 **대체하지 않는다** — 한자와 별도로 존재하는 발음 도우미다.

## 4성(四声) — 표준 중국어의 네 가지 성조

표준 중국어(보통화/普通话)엔 **4성 + 경성** 총 5가지 성조가 있다.

| 성조 | 부호 | 모양 | 설명 | 예시 |
|------|------|------|------|------|
| 1성 | ā | ¯ 평평 | 높고 길게 유지 | `mā` 엄마 妈 |
| 2성 | á | ╱ 올라감 | 낮→높, 의문 어조처럼 | `má` 삼 麻 |
| 3성 | ǎ | ∨ 내려갔다 올라옴 | 낮게 내렸다가 올림 | `mǎ` 말 马 |
| 4성 | à | ╲ 내려감 | 높→낮, 단호하게 뚝 | `mà` 꾸짖다 骂 |
| 경성 | a | 없음 | 빠르고 가볍게 | `ma` 의문 어기조사 吗 |

핵심: **같은 발음 + 성조만 다르면 완전히 다른 단어**. 한국어엔 없는 개념.  
기억법: 1~4 번호 쓸 때 획 방향 = 성조 기호 모양과 일치.

프랑스어 악센트와 다름 — 악센트는 모음 **소리(질)** 변경, 성조는 **피치 곡선** 변경. 성조는 음운적 변별 기능(phonemic): 틀리면 다른 단어가 됨.

## 간체자 vs 번체자

1949년 건국 후 문맹률 낮추려 복잡한 한자를 단순화 → 간체자(简体字) 1956~1964년 공식 제정.

| 번체자 | 간체자 | 뜻 | 변화 |
|--------|--------|-----|------|
| 愛 | 爱 | 사랑 | 안의 心 제거 |
| 龍 | 龙 | 용 | 획 대폭 축소 |
| 車 | 车 | 수레/차 | 획 단순화 |
| 國 | 国 | 나라 | 안 채우기 단순화 |

지역별: 중국 대륙·싱가포르 = 간체자 / 대만·홍콩·마카오 = 번체자.  
한국 한자는 번체자 기반 → 번체자 배우면 한국 한자 지식 활용 가능.  
발음은 동일 — `愛` = `爱` = `ài`. 글자 모양만 다름.  
대륙 중국인은 번체자 낯선 경우 많음. 반대로 번체자 사용자는 간체자 어느 정도 유추 가능.

## 부수(部首, bùshǒu) — 한자의 구성 요소

한자는 반복되는 구성 요소(부수)로 이루어진다. 총 214개 부수가 기준.

역할:
1. **의미 힌트** — 부수가 뜻 영역 암시
   - `氵`(물 수 변) → 물 관련: 海(바다) / 河(강) / 泳(헤엄치다) / 洗(씻다)
   - `木`(나무 목) → 나무 관련: 树(나무) / 桌(책상) / 椅(의자)
   - `口`(입 구) → 입·말 관련: 吃(먹다) / 喝(마시다) / 唱(노래하다)
2. **사전 분류** — 병음 모를 때 부수로 한자 검색

한국 한자 교육의 "부수" 개념과 동일. 부수 알면 모르는 한자도 뜻 영역 유추 가능.

## 형성자(形聲字) — 한자의 80%가 이 구조

형성자 = **의미부(형부)** + **발음부(성부)** 조합.

예: `清(qīng)` = `氵`(물, 뜻) + `青(qīng)`(발음 힌트)  
같은 성부 青을 공유: 請(qǐng 요청) / 情(qíng 감정) / 晴(qíng 맑음) → 발음 비슷

성부로 발음 **추측**, 병음으로 발음 **확인·표기** — 역할 다름:
- 성부 = 글자 **안에** 내장된 발음 힌트 (수천 년 전 설계)
- 병음 = 글자 **밖에** 붙는 로마자 표기 시스템 (1958년 발명)

주의: 성부 발음은 힌트지 정답 아님 — 성조 다르거나 역사적으로 변한 경우 있음.

## 획순(筆順, bǐshùn) — 한자 쓰는 순서

기본 규칙:
- 위→아래 / 왼→오른
- 가로→세로 (교차 시): 十은 ─ 먼저
- 바깥→안→닫기: 国은 테두리→내용→아랫선
- 왼쪽 삐침→오른쪽 삐침: 人은 ノ 먼저
- 중앙→좌우: 小는 중간 │ 먼저
- 관통 세로획은 마지막: 中은 口 쓴 뒤 │

획순 맞아야 빠르게 쓰고 흘려 쓸 때 자연스럽게 이어짐. 디지털 필기 인식도 획순 기반.  
한글 획순(ㄱ = 가로→세로)과 같은 원리 — 기본 규칙 몇 개 외우면 대부분 유추 가능.

## 갑골문(甲骨文) → 현재 — 한자 3,000년 진화

| 시기 | 이름 | 특징 |
|------|------|------|
| ~기원전 1200년 | 갑골문(甲骨文) | 거북 껍질·뼈에 새김. 그림 그대로 |
| ~기원전 1000년 | 금문(金文) | 청동기에 주조. 조금 단순화 |
| ~기원전 200년 | 소전(小篆) | 진시황 문자 통일. 둥근 곡선 |
| ~기원전 200년~ | 예서(隸書) | 빠른 필기로 직선화·평탄화 |
| ~기원후 200년~ | 해서(楷書) | 현재 한자의 직접 조상. 정방형 |

처음엔 진짜 그림: `日`=태양, `山`=봉우리 3개, `木`=나무 형태.  
수천 년 쓰면서 직선화·추상화 → 지금 모양.  
부수 보면 원래 뭘 그렸는지 유추 가능한 이유가 여기 있음.

## 한 글자 ≠ 한 단어 — 현대 중국어 단어 구조

한 글자 = 한 음절 = 한 뜻이지만, 실제 단어(词, cí)는 대부분 **2개 이상 한자 조합**.

예시:
- 电(전기) + 话(말) → 电话(diànhuà) 전화
- 电(전기) + 脑(뇌) → 电脑(diànnǎo) 컴퓨터
- 手(손) + 机(기계) → 手机(shǒujī) 휴대폰
- 学(배우다) + 生(사람) → 学生(xuéshēng) 학생

한국 한자어와 완전히 같은 원리(電話·電腦·手機). 글자 뜻 알면 조합 단어 유추 가능.  
글자(字) 2,000~3,000개 알면 → 조합으로 수만 개 단어 읽기 가능.
