# 중국어 문자

## 한 글자 = 한 음절 = 한 뜻 (표어문자)

중국어 한자(漢字)는 **표어문자**(logogram)다. 한글·알파벳 같은 표음문자와 달리, 한자 한 글자는 동시에 세 가지를 담는다:

1. **소리** — 딱 한 음절 (두 음절짜리 한자는 없다)
2. **뜻** — 의미 단위 하나
3. **모양** — 고유 형태

예시:
- `山` → `shān`, "산"
- `人` → `rén`, "사람"
- `好` → `hǎo`, "좋다"

한국 한자어(산=山, 인=人)와 같은 원리지만, 중국어는 한글 같은 표음 보조문자 없이 **한자만으로** 문장 전체를 쓴다. 상용한자 약 3,000자를 통째로 외워야 한다.

## 병음(拼音, pīnyīn) — 발음 표기 체계

한자만 봐선 발음을 알 수 없다. 그래서 중국은 1958년에 **병음(拼音, pīnyīn)** 을 제정했다. 알파벳에 성조 기호를 얹는 로마자 표기 체계다.

역할:
- 한자의 발음을 적는다 → `北京` = `Běi jīng`
- 키보드에서 한자 입력할 때 사용
- 교재에서 한자 위/옆에 발음 표시

구조:
```
성모(초성) + 운모(중·종성) + 성조부호
  b       +   ei          +  ˇ   → běi (북녘)
  j       +   īng         +  ¯   → jīng (수도)
```

병음은 한자를 **대체하지 않는다** — 한자와 별도로 존재하는 발음 도우미다.

## 4성(四声) — 표준 중국어의 네 가지 성조

표준 중국어(보통화/普通话)엔 **4성 + 경성** 총 5가지 성조가 있다.

| 성조 | 부호 | 모양 | 설명 | 예시 |
|------|------|------|------|------|
| 1성 | ā | ¯ 평평 | 높고 길게 유지 | `mā` 엄마 妈 |
| 2성 | á | ╱ 올라감 | 낮→높, 의문 어조처럼 | `má` 삼 麻 |
| 3성 | ǎ | ∨ 내려갔다 올라옴 | 낮게 내렸다가 올림 | `mǎ` 말 马 |
| 4성 | à | ╲ 내려감 | 높→낮, 단호하게 뚝 | `mà` 꾸짖다 骂 |
| 경성 | a | 없음 | 빠르고 가볍게 | `ma` 의문 어기조사 吗 |

핵심: **같은 발음 + 성조만 다르면 완전히 다른 단어**. 한국어엔 없는 개념.  
기억법: 1~4 번호 쓸 때 획 방향 = 성조 기호 모양과 일치.

프랑스어 악센트와 다름 — 악센트는 모음 **소리(질)** 변경, 성조는 **피치 곡선** 변경. 성조는 음운적 변별 기능(phonemic): 틀리면 다른 단어가 됨.

## 간체자 vs 번체자

1949년 건국 후 문맹률 낮추려 복잡한 한자를 단순화 → 간체자(简体字) 1956~1964년 공식 제정.

| 번체자 | 간체자 | 뜻 | 변화 |
|--------|--------|-----|------|
| 愛 | 爱 | 사랑 | 안의 心 제거 |
| 龍 | 龙 | 용 | 획 대폭 축소 |
| 車 | 车 | 수레/차 | 획 단순화 |
| 國 | 国 | 나라 | 안 채우기 단순화 |

지역별: 중국 대륙·싱가포르 = 간체자 / 대만·홍콩·마카오 = 번체자.  
한국 한자는 번체자 기반 → 번체자 배우면 한국 한자 지식 활용 가능.  
발음은 동일 — `愛` = `爱` = `ài`. 글자 모양만 다름.  
대륙 중국인은 번체자 낯선 경우 많음. 반대로 번체자 사용자는 간체자 어느 정도 유추 가능.

## 부수(部首, bùshǒu) — 한자의 구성 요소

한자는 반복되는 구성 요소(부수)로 이루어진다. 총 214개 부수가 기준.

역할:
1. **의미 힌트** — 부수가 뜻 영역 암시
   - `氵`(물 수 변) → 물 관련: 海(바다) / 河(강) / 泳(헤엄치다) / 洗(씻다)
   - `木`(나무 목) → 나무 관련: 树(나무) / 桌(책상) / 椅(의자)
   - `口`(입 구) → 입·말 관련: 吃(먹다) / 喝(마시다) / 唱(노래하다)
2. **사전 분류** — 병음 모를 때 부수로 한자 검색

한국 한자 교육의 "부수" 개념과 동일. 부수 알면 모르는 한자도 뜻 영역 유추 가능.

## 형성자(形聲字) — 한자의 80%가 이 구조

형성자 = **의미부(형부)** + **발음부(성부)** 조합.

예: `清(qīng)` = `氵`(물, 뜻) + `青(qīng)`(발음 힌트)  
같은 성부 青을 공유: 請(qǐng 요청) / 情(qíng 감정) / 晴(qíng 맑음) → 발음 비슷

성부로 발음 **추측**, 병음으로 발음 **확인·표기** — 역할 다름:
- 성부 = 글자 **안에** 내장된 발음 힌트 (수천 년 전 설계)
- 병음 = 글자 **밖에** 붙는 로마자 표기 시스템 (1958년 발명)

주의: 성부 발음은 힌트지 정답 아님 — 성조 다르거나 역사적으로 변한 경우 있음.

## 획순(筆順, bǐshùn) — 한자 쓰는 순서

기본 규칙:
- 위→아래 / 왼→오른
- 가로→세로 (교차 시): 十은 ─ 먼저
- 바깥→안→닫기: 国은 테두리→내용→아랫선
- 왼쪽 삐침→오른쪽 삐침: 人은 ノ 먼저
- 중앙→좌우: 小는 중간 │ 먼저
- 관통 세로획은 마지막: 中은 口 쓴 뒤 │

획순 맞아야 빠르게 쓰고 흘려 쓸 때 자연스럽게 이어짐. 디지털 필기 인식도 획순 기반.  
한글 획순(ㄱ = 가로→세로)과 같은 원리 — 기본 규칙 몇 개 외우면 대부분 유추 가능.

## 갑골문(甲骨文) → 현재 — 한자 3,000년 진화

| 시기 | 이름 | 특징 |
|------|------|------|
| ~기원전 1200년 | 갑골문(甲骨文) | 거북 껍질·뼈에 새김. 그림 그대로 |
| ~기원전 1000년 | 금문(金文) | 청동기에 주조. 조금 단순화 |
| ~기원전 200년 | 소전(小篆) | 진시황 문자 통일. 둥근 곡선 |
| ~기원전 200년~ | 예서(隸書) | 빠른 필기로 직선화·평탄화 |
| ~기원후 200년~ | 해서(楷書) | 현재 한자의 직접 조상. 정방형 |

처음엔 진짜 그림: `日`=태양, `山`=봉우리 3개, `木`=나무 형태.  
수천 년 쓰면서 직선화·추상화 → 지금 모양.  
부수 보면 원래 뭘 그렸는지 유추 가능한 이유가 여기 있음.

## 한 글자 ≠ 한 단어 — 현대 중국어 단어 구조

한 글자 = 한 음절 = 한 뜻이지만, 실제 단어(词, cí)는 대부분 **2개 이상 한자 조합**.

예시:
- 电(전기) + 话(말) → 电话(diànhuà) 전화
- 电(전기) + 脑(뇌) → 电脑(diànnǎo) 컴퓨터
- 手(손) + 机(기계) → 手机(shǒujī) 휴대폰
- 学(배우다) + 生(사람) → 学生(xuéshēng) 학생

한국 한자어와 완전히 같은 원리(電話·電腦·手機). 글자 뜻 알면 조합 단어 유추 가능.  
글자(字) 2,000~3,000개 알면 → 조합으로 수만 개 단어 읽기 가능.

## 성모(聲母)와 운모(韻母) — 병음의 내부 구조

병음 한 음절 = **성모(초성)** + **운모(중·종성)** + **성조**

성모(聲母) 21개: b p m f / d t n l / g k h / j q x / zh ch sh r / z c s  
운모(韻母) 36개: a o e i u ü / ai ei ao ou / an en ang eng er / ia iao ie iu ian in iang ing iong / ua uo uai ui uan un uang ueng / üe üan ün

성모 없이 운모만으로도 음절 가능 (a, e, ou 등).  
한국어 초성+중성+종성 구조와 같은 원리 — 다만 중국어 종성은 ng 또는 모음으로만 끝남.

→ 참고: `sheets/pinyin-table.html` (성모×운모 전체 조합표, 성조 전환+발음 재생)

## 회의자(會意字) — 뜻+뜻=새 뜻

회의자 = 의미부 둘 이상을 합쳐 새 뜻을 만드는 방식 (전체 한자의 약 13%).

| 한자 | 구성 | 의미 조합 | 뜻 |
|------|------|----------|-----|
| 明 | 日+月 | 해+달=밝음 | míng 밝다 |
| 休 | 人+木 | 사람이 나무에 기댐 | xiū 쉬다 |
| 森 | 木×3 | 나무 세 개=숲 | sēn 숲 |
| 好 | 女+子 | 여자+아이=좋음 | hǎo 좋다 |
| 男 | 田+力 | 밭에서 힘씀=남자 | nán 남자 |

형성자 vs 회의자: 한쪽이 발음 힌트면 형성자, 양쪽 다 의미 기여면 회의자.

## 상형자(象形字) — 그림에서 문자로

전체 한자 약 4%. 사물 모양을 직접 그린 원조 유형. 다른 모든 한자의 재료(부수)가 됨.

갑골문 → 해서 변화:
日(태양 원) / 月(초승달) / 山(봉우리 3개) / 木(줄기+가지+뿌리) / 水(물결) / 人(서있는 사람 측면) / 口(입) / 目(눈)

이 기본 상형자들이 부수로 들어가 수천 개 글자를 구성:
- 水→氵 : 海·河·泳·清
- 木 : 树·桌·椅·森
- 口 : 吃·喝·唱·吗
- 目 : 看·眼·睛·盲

상형자 수십 개 인식 → 처음 보는 글자도 뜻 영역 유추 가능.

## 가차(假借) — 소리만 빌린 한자

뜻은 무관, 발음만 빌려 외래 개념 표기. 외래어 음역에서 주로 등장.

| 중국어 | 병음 | 표기 대상 |
|--------|------|---------|
| 咖啡 | kāfēi | coffee |
| 沙发 | shāfā | sofa |
| 巧克力 | qiǎokèlì | chocolate |

영리한 의미+음역 동시 성공 사례:
- 可口可乐(kěkǒu kělè) = Coca-Cola → "맛있고 즐겁다"
- 百事可乐(bǎishì kělè) = Pepsi → "모든 일이 즐겁다"

한국어는 외래어를 한글로 소리 그대로("커피") — 중국어는 한자 체계라 반드시 한자 선택 필요.  
→ 외래 브랜드·지명·인명 볼 때 한자 뜻보다 **발음 우선** 봐야 함.

## 한국 한자어 = 중국어 문자 직독 가능

한국어 어휘의 약 60~70%가 한자어 → 중국어 한자와 직접 연결됨.

예시 (뜻 동일, 발음만 다름):
- 학교(學校) = 学校 xuéxiào
- 전화(電話) = 电话 diànhuà
- 문화(文化) = 文化 wénhuà
- 경제(經濟) = 经济 jīngjì

한국인은 어휘 의미를 새로 외울 필요 거의 없음 — 배울 건 **발음(병음)+성조**뿐.

주의: 완전 1:1 아님. 한국어 한자어가 중국에서 안 쓰이는 경우 있음(예: 편지↔信).  
그래도 다른 언어 화자 대비 압도적으로 유리한 출발점.

## 중국어 방언 — 같은 글자, 다른 발음

한자는 방언이 달라도 뜻을 전달하는 통일 문자 체계. 주요 방언:
- 普通话(보통화/Mandarin): 대륙 표준, 약 10억 화자
- 粤语(광둥어/Cantonese): 광둥·홍콩·마카오, 약 8천만
- 吴语(우어/Shanghainese): 상하이·저장, 약 8천만
- 闽南语(민남어/Hokkien): 푸젠·대만·동남아, 약 4천만

"我爱你" 발음 비교:
- 보통화: Wǒ ài nǐ
- 광둥어: Ngo oi nei
- 상하이어: Wu e nong

한자·일본·한국·베트남도 같은 글자를 각자 다른 발음으로 읽음 (學: 학/がく/học/xué).  
→ 한자는 수천 년간 동아시아를 하나로 묶은 문자 인프라.

## 현대 중국어 입력법 — 병음 IME

키보드에 한자 수천 개 올릴 수 없음 → 병음 입력법(拼音输入法) 사용.

작동: 병음 타이핑(예: xue) → IME가 동음이의자 목록 팝업 → 맞는 한자 선택
- xue 입력 시 후보: 学 雪 血 穴 靴 ...

같은 발음 한자가 수십 개씩 존재 (yī: 一医衣依咦伊...).

결과:
- 错别字(cuòbiézì): 발음 맞는데 잘못된 한자 고르는 오타 — SNS에서 흔히 보임
- 성조 없이도 입력 가능 — IME가 후보 목록으로 해결, 타이핑 속도에 병목 없음

폰 추가 입력: 병음 스와이프 / 필기 인식 / 부수 입력(주로 대만)  
→ 병음 = 디지털 중국어 리터러시의 기반. 손글씨 못 써도 병음만 알면 OK.

## HSK — 중국어 능력 등급과 필요 한자 수

HSK(汉语水平考试) = 중국 공인 중국어 능력 시험, 현재 9등급.

| 등급 | 어휘 수 | 실용 능력 |
|------|--------|---------|
| HSK 1 | 500 | 기초 생존 회화 |
| HSK 2 | 1272 | 일상 대화 |
| HSK 3 | 2245 | 사회생활 가능 |
| HSK 4 | 3245 | 유창한 소통 |
| HSK 5 | 4316 | 신문·방송 이해 |
| HSK 6 | 5456 | 고급 표현 |
| HSK 7-9 | 11092 | 원어민 수준 |

중국 초등 졸업 ≈ 2500자 / 고등 졸업 ≈ 3500자 / 상용 2000자 = 대부분 실용 텍스트 커버.  
한국인은 HSK 3-4 어휘 중 절반 이상이 한자어 겹침 → 뜻 암기 부담 절반 이하.  
실용 목표: 여행 → HSK 2-3 / 비즈니스 → HSK 4-5.

## 성어(成語, chéngyǔ) — 4글자에 담긴 고사

성어 = 4글자 관용구. 고대 역사·고전에서 온 압축 표현. 약 5만 개 존재, 실용 빈출은 500~1000개.

예시:
- 一石二鸟(yī shí èr niǎo): 돌 하나로 새 둘 = 일석이조
- 一举两得(yī jǔ liǎng dé): 한 번에 둘 얻음 = 일거양득
- 亡羊补牢(wáng yáng bǔ láo): 양 잃고 우리 고침 = 소 잃고 외양간 고치기
- 半途而废(bàntú ér fèi): 중간에 그만둠 = 용두사미

한국 한자성어 대부분 = 중국 성어 그대로 (일석이조·일거양득 한국에도 동일).  
성어 구사 = 중국에서 교양의 표시. 신문·정치 연설·고급 SNS 표현에 자주 등장.

## 동음이의자와 문화 금기

같은 발음 한자가 많아 숫자·선물이 문화적 금기·선호로 직결됨.

숫자:
- 4(sì) ≈ 死(sǐ 죽음) → 불길, 건물에 4층 없는 경우 많음
- 8(bā) ≈ 发(fā 부자 되다) → 최고 길수 (베이징 올림픽: 2008.8.8. 오후 8:08 개막)
- 6(liù) ≈ 流/顺(순조롭다) → 길함

선물 금기:
- 梨(lí 배) ≈ 离(lí 이별) → 연인·부부에게 금지
- 伞(sǎn 우산) ≈ 散(sàn 흩어지다) → 이별 암시
- 钟(zhōng 시계) ≈ 送终(장례 치르다) → 가장 심각한 금기

발음과 의미가 문화·비즈니스·외교 결정에까지 영향. 한자 문화권에서 소리와 의미의 깊은 연결.
