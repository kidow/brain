## 時(시) — 조수사 원리 그대로

한자어 숫자 + 時. 앞서 조수사에서 배운 발음 변화 패턴 일부 재사용.

- 1時(いちじ) / 2時(にじ) / 3時(さんじ) / 4時(よじ, し 회피) / 5時(ごじ)
- 6時(ろくじ) / 7時(しちじ) / 8時(はちじ) / 9時(くじ, きゅうじ 아님) / 10時(じゅうじ)
- 11時(じゅういちじ) / 12時(じゅうにじ)

핵심 규칙: 4時(よじ)는 조수사에서 배운 "4의 발음 함정"(し 회피)이 그대로 적용. 9時(くじ)는 새 예외 — きゅう가 아니라 く로 줄어드는 건 時에서만 나타나는 관용적 규칙.

오전/오후: 午前(ごぜん) 오전 / 午後(ごご) 오후
- 午前9時 — 오전 9시 / 午後3時 — 오후 3시

## 分(분) — ふん/ぷん 발음 변화

규칙: っ(촉음) 또는 ん으로 끝나는 숫자 뒤에서 ぷん.

- 1分(いっぷん,っ) / 2分(にふん) / 3分(さんぷん,ん) / 4分(よんぷん,ん) / 5分(ごふん)
- 6分(ろっぷん,っ) / 7分(ななふん) / 8分(はっぷん,っ) / 9分(きゅうふん) / 10分(じゅっぷん,っ)

패턴 감각: 숫자가 っ(1・6・8・10, 조수사 촉음화 규칙)나 ん(3・4)으로 끝나면 → ぷん. 그 외(2・5・7・9)는 → ふん. 個(こ)의 촉음화 규칙이 分에도 적용, 여기에 ん으로 끝나는 경우(3・4)가 추가.

30분 특수 표현: 30分(さんじゅっぷん) / 半(はん, "~시 반")
- 3時30分 = 3時半 — 3시 30분 = 3시 반

시각 전체 조립:
- 午後3時15分です。오후 3시 15분입니다.
- 7時半に起きます。7시 반에 일어납니다.
