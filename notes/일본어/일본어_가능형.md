## ~ことができる — 가능형 (범용, 문형 방식)

"~할 수 있다"를 만드는 가장 쉬운 방법. 동사 기본형 + ことができる.

- 私は日本語を話すことができます。— 저는 일본어를 말할 수 있습니다.
- 漢字を読むことができます。— 한자를 읽을 수 있습니다.

구조 분해: こと(것, 명사화) + が(주어 표지) + できる(가능하다, 동사). 직역 "말하는 것이 가능하다" — こと가 동사를 명사화(話す→話すこと), が로 주어 삼아 できる로 서술.

장점: 그룹 구분 없이 모든 동사에 동일 적용(기본형 그대로).
- する → することができる / 来る → 来ることができる / 食べる → 食べることができる

단점: 문장이 길고 딱딱함(문어체·격식체). 회화에서는 동사 가능형(食べられる)이 더 자연스러움.

## 동사 가능형 — 2그룹·3그룹 규칙

동사 자체를 활용해 가능형 생성. ことができる보다 짧고 자연스러움.

**2그룹** — る 떼고 + られる: 食べる → 食べられる (먹을 수 있다) / 見る → 見られる (볼 수 있다). ない형(食べる→食べない)과 같은 자리에서 붙임.

**3그룹** — 통암기:
- する → できる (할 수 있다) — 규칙적 활용 아니라 통째로 다른 단어. ことができる의 できる와 같은 어원.
- 来る(くる) → 来られる(こられる) (올 수 있다)

정중형: られる·できる 자체가 2그룹 동사처럼 활용 → ます 붙이면 끝. 食べられます。/ できます。

## 1그룹 가능형 — 어미 う단 → え단 + る

- 書く → 書ける (쓸 수 있다) / 飲む → 飲める (마실 수 있다) / 話す → 話せる (말할 수 있다)
- 乗る → 乗れる (탈 수 있다) / 買う → 買える (살 수 있다)

패턴: く→け, む→め, す→せ, る→れ, う→え (어미를 え단으로 바꿈).

가능형은 만들어지면 전부 2그룹처럼 활용 → ます 붙이면 끝. 書けます。/ 飲めます。

동사 활용 3형태 비교:
| 형태 | 1그룹 규칙 | 예 |
|---|---|---|
| ます형 | う단 → い단 | 飲む → 飲みます |
| ない형 | う단 → あ단 | 飲む → 飲まない |
| 가능형 | う단 → え단 | 飲む → 飲める |

う단 기준으로 い(정중)·あ(부정)·え(가능) 세 방향으로 갈라지는 구조.

가능형 문형: 대상은 が(を 아님). ✗ 日本語を話せます / ✓ 日本語が話せます。일본어를 말할 수 있습니다. — 가능형은 형용사처럼 "능력이 있다"는 상태를 서술하므로 が(상태·능력 서술의 조사).
