## とても — 매우/아주

정도부사 중 가장 기본. 형용사·동사 앞에서 강조.

- とても大きい。— 아주 크다.
- とても好きです。— 매우 좋아합니다.

긍정문 전용 — 부정문에서 "매우 안 ~" 뜻으로 쓰려면 다른 표현(全然·あまり) 필요.

어순: 부사는 항상 수식하는 말(형용사·동사) 바로 앞. 한국어 "아주 크다"와 어순 동일.

## あまり — 별로/그다지 (부정문 전용)

반드시 부정문과 짝. 혼자 못 씀. あまり + 긍정문 = 비문법적.

- あまり好きじゃない。— 별로 안 좋아한다.
- あまり大きくない。— 별로 안 크다.
- ✗ あまり好きです。(오류)

한국어 "별로"도 비슷한 극성 제약("별로 좋아해" 대신 "별로 안 좋아해").

とても vs あまり:
| | 극성 | 뜻 |
|---|---|---|
| とても | 긍정문 전용 | 매우 |
| あまり | 부정문 전용 | 별로 |

## よく — 자주/잘

뜻 둘, 문맥으로 구분. 긍정문에서 자유롭게 씀(극성 제약 없음).

1. **빈도** ("자주") — よく映画を見ます。자주 영화를 봅니다. (동작 동사 + 반복 빈도)
2. **정도·완성도** ("잘") — よく分かりました。잘 알겠습니다. / よく聞いてください。잘 들어주세요. (이해·듣기 동사 + 능숙함)

구별: "몇 번"이면 빈도, "얼마나 잘"이면 정도.

어원: い형용사 良い(よい, 좋다)의 부사형. 良い→よく (い→く 규칙, ~くない와 같은 뿌리).

## 빈도부사 심화 — いつも·時々·全然

빈도 스펙트럼(높음→낮음): いつも(항상, 100%) → よく(자주, 70%) → 時々(때때로, 50%) → あまり(별로) → 全然(전혀, 0%)

- いつも運動します。— 항상 운동합니다.
- 時々映画を見ます。— 때때로 영화를 봅니다.

全然 — "전혀" (부정문 전용, あまり보다 강함): 全然分かりません。전혀 모르겠습니다.
주의: 구어체에서 全然이 긍정문 강조로도 쓰임(全然大丈夫です — 완전 괜찮아요). 격식체 글쓰기에선 부정문 전용 규칙 유지 권장.

빈도부사 전체 정리:
| 부사 | 뜻 | 극성 |
|---|---|---|
| いつも | 항상 | 제약 없음 |
| よく | 자주 | 제약 없음 |
| 時々 | 때때로 | 제약 없음 |
| あまり | 별로 | 부정 전용 |
| 全然 | 전혀 | 부정 전용(구어체 예외) |
