## ~てもいい — 허가 ("~해도 된다")

て형 + も + いい = "~해도 좋다/된다".

- 写真を撮ってもいいですか。— 사진을 찍어도 됩니까?
- ここに座ってもいいですか。— 여기 앉아도 됩니까?

구조 분해: て(연결) + も(조사 も, "도") + いい(좋다, い형용사). 직역 "찍어도 좋다" — 한국어 "~해도 되다"와 어순·구조까지 일치.

대답: はい、いいですよ。(네, 좋아요) / いいえ、だめです。(아니요, 안 됩니다)

핵심 감각: 항상 의문문(~か)으로 물어보고, 답은 い형용사 활용(いいです·よくないです)으로 함 — 형용사 활용 규칙이 그대로 적용.

## ~てはいけない — 금지 ("~하면 안 된다")

て형 + は + いけない = "~하면 안 된다".

- ここで写真を撮ってはいけません。— 여기서 사진을 찍으면 안 됩니다.
- たばこを吸ってはいけません。— 담배를 피우면 안 됩니다.

구조 분해: て(연결) + は(조사 は, 화제 표지·대조) + いけない(관용 표현, 直訳 "가서는 안 된다"). は의 대조 기능이 작동 — "~하는 건 안 된다" 뉘앙스. てもいい의 も와 대비되는 자리.

정중형 いけません: いける(가능동사)의 부정형이 관용화된 것, 통암기.

대체 표현: だめです(더 회화체) — ここで写真を撮ってはだめです。

てもいい vs てはいけない:
| | 조사 | 뜻 | 뉘앙스 |
|---|---|---|---|
| ~てもいい | も | 해도 된다 | 허가 |
| ~てはいけない | は | 하면 안 된다 | 금지 |

같은 て형 뒤에 조사 하나(も/は)만 바뀌어 허가↔금지가 뒤집힘 — も(추가)/は(대조) 조사 성격 차이가 문법적으로 이어짐.

## ~てから — 순서 ("~하고 나서")

て형 + から = "~하고 나서" — 동작 순서를 명확히 함.

- 手を洗ってから食べます。— 손을 씻고 나서 먹습니다.
- 宿題をしてから遊びます。— 숙제를 하고 나서 놉니다.

구조 분해: から는 "~부터/~때문에"의 기본 뜻을 가진 조사. て형과 결합하면 "그 동작부터 시작해서" → "~한 다음에"로 순서 표현.

て만 쓸 때와 비교: て형 자체도 "~하고"로 순서를 나타낼 수 있지만(起きて、シャワーを浴びます — 일어나서 샤워합니다), から를 붙이면 순서를 훨씬 명확하고 의도적으로 강조.
- 起きてシャワーを浴びます。(그냥 나열)
- 起きてからシャワーを浴びます。(순서 강조 — 반드시 일어난 다음에)

## て형 — 원인·이유 ("~해서")

て형 자체가 원인·이유도 나타냄. 별도 조사 없이 て만으로 "~해서/~때문에".

- 雨が降って、試合が中止になりました。— 비가 와서 시합이 취소되었습니다.
- 風邪をひいて、学校を休みました。— 감기에 걸려서 학교를 쉬었습니다.

핵심 규칙: て형 이유 용법은 자연발생적·통제 불가능한 상황에 주로 씀(날씨·감정·신체 반응 등).
- 疲れて、早く寝ました。— 피곤해서 일찍 잤습니다.

주의: から(이유, 순서의 から와 다른 별개 용법)와 뉘앙스 차이.
- 雨が降ったので、試合が中止になりました。(から/ので — 논리적 인과관계, 격식체)
- 雨が降って、試合が中止になりました。(て — 자연스러운 흐름 서술, 회화체)

て는 강한 의지·명령문과는 못 씀 — 명령·요청의 이유를 말할 땐 から/ので 사용.
- ✗ 疲れて、早く寝てください。(어색)
- ✓ 疲れたから、早く寝てください。(자연스러움)

て형 전체 용법 정리:
| 용법 | 형태 | 뜻 |
|---|---|---|
| 연결 | Aて、B | A하고 B (단순 나열) |
| 진행/상태 | ~ている | ~하고 있다 |
| 요청 | ~てください | ~해 주세요 |
| 허가 | ~てもいい | ~해도 된다 |
| 금지 | ~てはいけない | ~하면 안 된다 |
| 순서 | ~てから | ~하고 나서 |
| 이유(자연발생) | Aて、B | A해서 B |
