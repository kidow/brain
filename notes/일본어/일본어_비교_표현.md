## より — 비교 기준 조사 ("~보다")

비교 대상을 표시하는 전용 조사. 한국어 "보다"와 1:1 대응.

- 東京は大阪より大きいです。— 도쿄는 오사카보다 큽니다.

문형 구조: A は B より 형용사
- A = 비교당하는 대상(주어) / B より = "B보다" / 뒤에 형용사(い형용사·な형용사 모두 가능)

- この本はあの本より面白いです。— 이 책은 저 책보다 재미있습니다. (い형용사)
- 日本語は英語より簡単です。— 일본어는 영어보다 간단합니다. (な형용사)

핵심 감각: より는 항상 비교당하는 기준점(원래 "덜한 쪽") 뒤에 붙음. 한국어 "보다" 위치와 완전히 동일해서 어순 그대로 옮기면 됨.

## ~のほうが — "~쪽이 더" (선택·강조)

より만으로 비교 가능하나, 어느 쪽이 더 나은지 강조할 때 ほう(方, "쪽") 사용.

문형: A のほうが B より 형용사
- りんごのほうがバナナより好きです。— 사과 쪽이 바나나보다 좋아합니다.

より 이하 생략 가능(문맥상 명확하면): りんごのほうが好きです。— 사과 쪽을 더 좋아합니다.

핵심 규칙: のほうが의 が는 조사 が(주어 표지) — "어느 쪽이"에 대한 답으로 강조되는 주체 표시. "모르는 것 물을 때 が" 공식과 일치.
- どちらのほうが好きですか。어느 쪽을 더 좋아합니까? → りんごのほうが好きです。사과 쪽이 더 좋아합니다.

より vs のほうが:
| | より | のほうが |
|---|---|---|
| 역할 | 비교 기준점 표시 | 우세한 쪽 강조 |
| 생략 가능? | 생략 불가(비교 대상 필수) | より 이하 생략 가능 |
| 예 | AはBより大きい | Aのほうが(Bより)大きい |
