# 독일어 요일 시간대 표현

## jeden Montag — 반복 표현(Akkusativ)

독일어_기초_어휘의 요일(Montag, Dienstag...)에 jeden을 붙이면 "매주 ~요일"이 된다.

jeden+요일 = "매주 ~요일에"(Akkusativ, 전치사 없이):

| 표현 | 뜻 |
|------|-----|
| jeden Montag | 매주 월요일에 |
| jeden Tag | 매일 |
| jede Woche | 매주 |
| jedes Wochenende | 매주말 |

> Ich gehe jeden Montag ins Fitnessstudio.  
> Jeden Tag trinke ich Kaffee.

핵심 문법 — der-Wörter(jeder)의 Akkusativ 남성 어미(-en), 전치사 없이 시간부사로 사용:
> jeder(주격 남성) → jeden(Akkusativ 남성) — 대명사 노트의 dieser 패턴과 동일  
> ❌ an jeden Montag / ✅ jeden Montag

성별에 따라 어미 다름(der-Wörter 규칙 그대로):
> jede Woche(여성) / jedes Wochenende(중성) / jeden Montag(남성)

구어체 강조: Immer jeden Montag gehe ich schwimmen.

핵심: **jeden/jede/jedes+시간명사 = "매~"(전치사 없이 Akkusativ). 성에 따라 어미 다름, der-Wörter 규칙 그대로.**

## -s 접미사 시간부사 — morgens, abends, montags

요일·시간대 명사+-s = "습관적으로/일반적으로 ~에". 관사·전치사 없이 그 자체로 부사.

하루 시간대+-s:

| 부사 | 뜻 |
|------|-----|
| morgens | 아침마다 |
| vormittags | 오전마다 |
| mittags | 점심마다 |
| nachmittags | 오후마다 |
| abends | 저녁마다 |
| nachts | 밤마다 |

> Ich trinke morgens Kaffee. / Abends lese ich gern.

요일+-s — jeden Montag과 같은 뜻, 더 간결:
> montags gehe ich ins Fitnessstudio. = Jeden Montag gehe ich ins Fitnessstudio.

핵심 구분 — der Morgen(명사) vs morgens(부사):
> Der Morgen ist schön. (명사, 관사 필요) / Morgens ist es kalt. (부사, 관사 없음)

조합 가능: montags morgens(월요일 아침마다)

핵심: **시간대·요일 명사+s = 습관적 부사(morgens, montags). 관사 없이 그 자체로 부사. Der Morgen(명사)과 morgens(부사)는 다른 품사.**
