# 독일어 존댓말과 격식체

## du vs Sie — 언제 어느 쪽을 쓰나

독일어_대명사에서 형태(du/Sie)는 배웠지만, 언제 어느 쪽을 쓰는지는 여기서 다룬다.

Sie(존댓말) — 처음 만나는 성인, 낯선 사람, 공식적 관계:
> 낯선 사람, 가게 점원, 관공서 직원, 직장 상사, 처음 만난 어른

du(반말) — 가족, 친구, 어린이, 같은 또래·캐주얼한 직장:
> 가족, 친구, 아이들(자동 du), 대학생끼리, SNS(요즘은 du가 표준)

핵심 원칙 — 나이·지위가 높은 쪽이 먼저 "du로 하자"고 제안(Duzen anbieten), 아랫사람이 먼저 제안하면 실례:
> Wollen wir uns duzen? (우리 말 놓을까요?)  
> Wir können uns gerne duzen. (윗사람이 하는 말)

한국어와 다른 점 — 나이보다 "관계의 공식성"이 기준:
> 동갑이라도 낯선 사람이면 Sie, 나이 차이 나도 가족이면 du

요즘 트렌드 — 스타트업·IT 업계는 나이 상관없이 du가 표준(du-Kultur, IKEA 등).

핵심: **Sie=낯선 사람·공식적 관계, du=가족·친구·아이·캐주얼한 곳. 손윗사람이 먼저 duzen 제안. 한국어와 달리 나이보다 관계의 공식성이 기준.**

## 정중함의 단계 — Können Sie...? vs Könnten Sie...? vs Würden Sie...?

독일어_접속법의 Konjunktiv II(könnte, würde)가 정중함의 단계를 만드는 도구.

정중도 낮음 → 높음:

| 표현 | 정중도 | 뜻 |
|------|--------|-----|
| Kannst du...? | 반말 | ~해줄래? |
| Können Sie...? | 존댓말, 기본 | ~해주시겠어요? |
| Könnten Sie...? | 존댓말+Konjunktiv II | ~해주실 수 있을까요? |
| Würden Sie...? | 최상급 정중 | ~해주시겠습니까? |
| Wären Sie so freundlich...? | 극존칭(편지·공문) | ~해주시면 감사하겠습니다 |

> Können Sie mir helfen? / Könnten Sie mir helfen? / Würden Sie mir helfen?

최고 존칭 — 독일어_접속법의 zu부정사 구문과 결합:
> Wären Sie so freundlich, mir die Unterlagen zu schicken?

패턴: Präsens→Konjunktiv II 단순형→강조형 순으로 정중도 상승. "würde보다 wäre/hätte/könnte가 더 흔하다"는 원칙이 실전 적용.

핵심: **정중도 순서 = Können Sie? < Könnten Sie? < Würden Sie? < Wären Sie so freundlich...? 접속법 노트의 Konjunktiv II가 정중함의 핵심 도구.**
