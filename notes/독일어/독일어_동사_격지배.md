# 독일어 동사 격지배

## 기본값은 Akkusativ, Dativ 동사는 예외 목록

대부분 동사는 목적어를 Akkusativ로 받는다. 일부는 예외적으로 Dativ를 요구한다.

Dativ 요구 동사 — 대표 그룹:

| 동사 | 뜻 |
|------|-----|
| helfen | ~을 돕다 |
| danken | ~에게 감사하다 |
| gefallen | ~의 마음에 들다 |
| gehören | ~에게 속하다 |
| folgen | ~을 따르다 |
| glauben | ~을 믿다 |
| antworten | ~에게 대답하다 |

> Ich helfe ihm. (영어라면 "help him"=목적격인데 독일어는 Dativ)  
> Ich danke dir.

핵심 함정 — 영어식 감각으로 Akkusativ라 착각하기 쉬움:
> ❌ Ich helfe ihn. / ✅ Ich helfe ihm.

역사적으로 "~에게 이익/영향을 주는" 의미의 동사들이 Dativ로 굳어짐: helfen="그에게 도움을 주다"—수혜자 느낌이라 Dativ.

전략 — Dativ 동사는 규칙이 아니라 목록이므로 개별 암기(단어장에 "동사, Dat" 표시).

핵심: **대부분 동사=Akkusativ(기본값). helfen·danken·gefallen·gehören·folgen·glauben·antworten 등 소수는 Dativ 요구(암기 목록).**

## Dativ+Akkusativ 동시 요구 동사 — "누구에게 무엇을"

일부 동사는 목적어를 두 개 갖는다 — Dativ(누구에게)+Akkusativ(무엇을).

| 동사 | 뜻 |
|------|-----|
| geben | ~에게 ~을 주다 |
| schenken | ~에게 ~을 선물하다 |
| zeigen | ~에게 ~을 보여주다 |
| erklären | ~에게 ~을 설명하다 |
| schreiben | ~에게 ~을 쓰다 |
| kaufen | ~에게 ~을 사주다 |

> Ich gebe dir das Buch. (dir=Dativ 사람, das Buch=Akkusativ 사물)

어순은 독일어_대명사 규칙 그대로:
> 명사+명사: Dativ→Akkusativ / 대명사+명사: 대명사 먼저 / 대명사+대명사: Akkusativ→Dativ

패턴: "누구에게 무엇을 ~하다" = 수혜자(Dativ)+대상물(Akkusativ) 구조.

독일어_수동태의 bekommen-Passiv가 이 동사군에서 나옴:
> Ich gebe dir das Buch. → Du bekommst das Buch geschenkt. (Dativ→주어 전환)

핵심: **geben·schenken·zeigen·erklären 등은 Dativ(사람)+Akkusativ(사물) 동시 요구. "누구에게 무엇을" 패턴. 어순은 대명사 노트 규칙 그대로.**
