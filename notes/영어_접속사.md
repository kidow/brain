## 접속사란 — 단어·절·문장을 잇는 연결고리

두 요소를 연결하는 품사.

| 종류 | 역할 | 예 |
|---|---|---|
| **등위접속사** | 대등한 두 요소 연결 | and, but, or, so |
| **종속접속사** | 주절에 종속절을 붙임 | although, because, when, if |

등위 = 양쪽이 같은 무게.
종속 = 한쪽이 다른 쪽에 의존.

---

## 등위접속사 FANBOYS

| 글자 | 접속사 | 뜻 |
|---|---|---|
| F | for | 이유 |
| A | and | 추가 |
| N | nor | 둘 다 아님 |
| B | but | 대조 |
| O | or | 선택 |
| Y | yet | 강한 대조 |
| S | so | 결과 |

두 절 연결 시 앞에 쉼표:
```
I was tired, but I kept working.
She studied, so she passed.
```

---

## 종속접속사 — 부사절 유형별

| 유형 | 접속사 |
|---|---|
| 시간 | when, while, after, before, until, as soon as |
| 이유 | because, since, as |
| 조건 | if, unless, as long as |
| 양보 | although, though, even though, even if |
| 목적 | so that, in order that |
| 결과 | so … that, such … that |
| 대조 | while, whereas |

종속절이 앞 → 쉼표 필요. 뒤 → 쉼표 없음.
```
Because she was tired, she went home.
She went home because she was tired.
```

---

## while vs whereas — 대조 접속사 구분

`while` = 시간(~하는 동안) 또는 대조(~인 반면에, 비격식)
`whereas` = 대조만 (격식·문어체)

```
While I was cooking, she watched TV.      (시간)
While I like coffee, she prefers tea.     (대조 — 비격식)
I like coffee, whereas she prefers tea.   (대조 — 격식)
```

구분: while 앞 절에 진행 동작 → 시간. 상태·의견·사실 → 대조.

---

## 상관접속사 — 쌍으로 쓰는 접속사

| 상관접속사 | 뜻 |
|---|---|
| `both A and B` | A와 B 둘 다 |
| `either A or B` | A 또는 B |
| `neither A nor B` | A도 B도 아님 |
| `not only A but also B` | A뿐만 아니라 B도 |
| `not A but B` | A가 아니라 B |

수일치: 동사는 B(뒤쪽 명사)에 맞춤.
```
Either she or I am right.
Neither he nor they are wrong.
```

---

## 접속부사 — 접속사처럼 보이지만 부사

`however / therefore / moreover / nevertheless / thus` = 접속사 아님. 두 절을 쉼표로 연결 불가.

```
I was tired, however I kept working.   ❌
I was tired. However, I kept working.  ✅
I was tired; however, I kept working.  ✅
```

| 접속부사 | 뜻 | 비고 |
|---|---|---|
| however | 그러나 | |
| therefore | 따라서 | 일반 격식 |
| thus | 따라서 | 고격식·논문 |
| moreover | 게다가 | 단순 추가 |
| furthermore | 더 나아가 | 앞 주장 강화 |
| nevertheless | 그럼에도 불구하고 | |

패턴: `문장. 접속부사, 문장.` 또는 `문장; 접속부사, 문장.`

---

## because vs since vs as — 이유 접속사 구분

| | `because` | `since` | `as` |
|---|---|---|---|
| 강도 | 강함 — 직접 이유 | 중간 | 약함 |
| 이유 | 새 정보 | 이미 알려진 정보 | 이미 알려진 정보 |
| 격식 | 중립 | 격식 | 고격식·문어 |

```
I stayed home because I was sick.    (이유 = 핵심 새 정보)
Since you're here, let's start.      (이미 알려진 배경)
As it was raining, we stayed inside. (당연히 아는 상황)
```

`since`는 시간("~이후로")도 되니 문맥으로 구분.

---

## although vs though vs even though vs even if

| | 특징 |
|---|---|
| `although` | 격식·중립 |
| `though` | 비격식, 문장 끝 가능 |
| `even though` | 사실 강조 |
| `even if` | 가정·불확실 상황 |

핵심 구분:
```
Even though I was tired, I finished.   (실제로 피곤했음 — 사실)
Even if I am tired, I will finish.     (피곤할 수도, 아닐 수도 — 가정)
```

`though`만 문장 끝 가능:
```
It was cold, though.  ✅
```

---

## if vs unless — 조건 접속사

`unless` = `if … not` = "~하지 않으면" (예외 조건)

```
I'll go unless it rains.       = I'll go if it doesn't rain.
Unless you study, you'll fail. = If you don't study, you'll fail.
```

`unless` + 부정 = 이중 부정 → 금지.
```
Unless you don't come.  ❌
If you don't come.       ✅
```

---

## so that vs so … that — 목적 vs 결과

**`so that`** = 목적 (~하기 위해서)
```
She studied hard so that she could pass.
```

**`so + 형용사/부사 + that`** = 결과 (너무 ~해서 …하다)
```
It was so cold that I stayed home.
```

**`such … that`** = 결과 (명사 앞)
```
It was such a cold day that I stayed home.
```

구분:
- 뒤에 조동사(can/could/will/would) → 목적 `so that`
- 형용사·부사 강조 → 결과 `so … that`
- 명사 앞 → 결과 `such … that`

---

## 명사절 접속사 — that / whether / if

**`that`** — 사실·생각·감정 전달.
```
I know [that she is right].      (목적어, that 생략 가능)
[That he lied] surprised me.     (주어)
```

**`whether / if`** — "~인지 아닌지" (불확실·의문)
```
I don't know whether/if she will come.
```

`whether`만 가능한 경우:
```
Whether she comes or not doesn't matter.  ✅ (주어 자리)
If she comes or not doesn't matter.       ❌
```

---

## as — 다중 용법 접속사

| 용법 | 뜻 | 예 |
|---|---|---|
| 시간 | ~할 때 (동시) | As I walked in, she left. |
| 이유 | ~이므로 (배경) | As it was late, we left. |
| 방식 | ~대로 | Do as I say. |
| 비례 | ~할수록 | As you get older, time flies faster. |

구분: 동시 진행 → 시간 / 알려진 배경 → 이유 / 동작 따라 함 → 방식.

---

## 접속사 vs 전치사 혼동 쌍

| 의미 | 접속사 (뒤에 절) | 전치사 (뒤에 명사구) |
|------|-----------------|-------------------|
| 양보 | although / though | despite / in spite of |
| 이유 | because / since | because of / due to / owing to |
| 시간 | while | during |

- Although he was tired, he worked. / Despite his tiredness, he worked.
- Because she studied hard, she passed. / Because of her hard work, she passed.
- While he was sleeping, I left. / During his sleep, I left.

실수 주의: despite he was tired ❌ → despite his tiredness ✅

## now that / once / by the time

now that — "이제 ~했으니까" (변화된 상황이 원인)
- Now that you're here, we can start.
- because와 유사하지만 새로운 상황 변화에 초점.

once — "일단 ~하면" (전환점 조건)
- Once you understand it, it's easy.
- after보다 "이 단계를 넘으면"이라는 느낌.

by the time — "~할 때쯤이면 이미" (기준 시점보다 먼저 완료)
- By the time he arrived, she had already left.
- 뒤 절은 완료 시제(had p.p. / will have p.p.) 자연스러움.
