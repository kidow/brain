# 중국어 시간 날짜 표현

## 시각 읽기 — 点(diǎn)과 分(fēn)

点 = "시"(시각 단위), 分 = "분"이다. 시각을 물을 땐 几点(jǐ diǎn, 몇 시)?

기본 형식: [숫자]点[숫자]分.
- 三点(sān diǎn) = 3시
- 三点十分(sān diǎn shí fēn) = 3시 10분
- 三点半(sān diǎn bàn) = 3시 반(半=반, 30분과 같은 뜻이라 分 안 붙임)
- 三点一刻(sān diǎn yí kè) = 3시 15분(刻=15분 단위, 영어 quarter와 같은 개념)

주의: "2시"는 二点(èr diǎn)이 아니라 两点(liǎng diǎn)이다 — 숫자 2가 단위 앞에 올 때 二 대신 两을 쓰는 규칙이 시각에서도 그대로 적용된다.

한국어와 어순이 완전히 같다 — "3시 10분"처럼 "시"가 먼저, "분"이 나중(큰 단위→작은 단위)이다. 영어(ten past three, 분이 먼저 나오는 경우도 있음)보다 오히려 한국어 화자에게 직관적이다.

## 날짜 읽기 — 年(nián)/月(yuè)/号(hào)

年=년, 月=월, 号(구어체)/日(문어체)=일이다. 순서는 큰 단위→작은 단위: 년-월-일.

예:
- 二零二五年(èr líng èr wǔ nián) = 2025년 — 년도는 숫자를 하나씩 따로 읽는다(二零二五, "이공이오"처럼), 二千二十五처럼 통째로 안 읽는다
- 三月(sān yuè) = 3월
- 十号(shí hào, 구어체) / 十日(shí rì, 문어체) = 10일

전체 조합: 二零二五年三月十号(2025년 3월 10일) — 년월일 순서가 한국어와 완전히 같다(영어는 월-일-년이라 반대).

요일은 星期(xīngqī) + 숫자다: 星期一(월)~星期六(토), 星期日/星期天(일요일만 특별히 숫자 대신 日/天). 오늘=今天(jīntiān), 어제=昨天(zuótiān), 내일=明天(míngtiān).

핵심: 년도만 숫자를 하나씩 따로 읽고, 월·일·시각은 일반 숫자 그대로 읽는다는 게 헷갈리기 쉬운 포인트다.

## 시간 접속 표현 — 以前(yǐqián)/以后(yǐhòu)/的时候(de shíhou)

시각·날짜 자체가 아니라 "~하기 전/~한 후/~할 때"처럼 시점을 사건에 연결하는 표현이다.

- **以前**(yǐqián, ~하기 전/이전) — 문장·구 뒤에 붙는다: 吃饭以前(chī fàn yǐqián, 밥 먹기 전), 三年以前(3년 전)
- **以后**(yǐhòu, ~한 후/이후) — 吃饭以后(밥 먹은 후), 三点以后(3시 이후)
- **的时候**(de shíhou, ~할 때) — 문장 뒤에 붙어 "그 시점에": 吃饭的时候(밥 먹을 때), 小时候(어릴 때, 관용적 축약형)

구조 공통점: 앞에 오는 절/구는 시제 표시(了 등) 없이 동사원형 그대로 쓰고, 시간 관계는 以前/以后/的时候 자체가 담당한다 — 중국어 특유의 "동사가 안 변한다"([중국어_특징](중국어_특징.md)에서 배운 고립어 특성)가 여기서도 그대로 적용된다. 시제·시점 정보는 오직 이 조사성 표현들이 문법 역할을 짊어진다.
