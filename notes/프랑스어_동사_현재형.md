# 프랑스어 동사 현재형

## 동사 활용 개념

프랑스어는 주어마다 동사 어미가 다름. 주어 생략 불가.

한국어 = 주어 달라도 어미 같음 (시제·높임만 바뀜).
프랑스어 = 주어가 바뀌면 어미도 바뀜.

예: parler (말하다) — -er 규칙동사

| 주어 | 동사 | 주어 | 동사 |
|---|---|---|---|
| je | parle | nous | parlons |
| tu | parles | vous | parlez |
| il/elle | parle | ils/elles | parlent |

발음: parle/parles/parlent 전부 /파흘/ — 어미 -e/-es/-ent 묵음. 귀로 구분 불가 → 주어 생략 불가 이유.

## être 활용 (이다/있다) — 완전 불규칙

| 주어 | être | 발음 |
|---|---|---|
| je | suis | /쒸이/ |
| tu | es | /에/ |
| il/elle | est | /에/ |
| nous | sommes | /솜/ |
| vous | êtes | /에트/ |
| ils/elles | sont | /송/ |

예: Je suis étudiant. / Il est grand. / Nous sommes amis.
형용사·직업·국적 연결 시 필수. il est / elle est + 형용사 패턴이 가장 빈번.

## avoir 활용 (가지다) — 불규칙

| 주어 | avoir | 발음 |
|---|---|---|
| je | ai | /에/ |
| tu | as | /아/ |
| il/elle | a | /아/ |
| nous | avons | /아봉/ |
| vous | avez | /아베/ |
| ils/elles | ont | /옹/ |

소유: J'ai un chat. / Tu as une voiture?
나이: J'ai 25 ans. (영어 I am 25 ≠ 프랑스어 J'ai 25)
감각: avoir faim(배고프다), avoir soif(목마르다), avoir peur(무섭다)

être vs avoir: être=상태·정체성, avoir=소유·보유.

## -er 규칙동사 활용 패턴

프랑스어 동사 80%가 -er 동사. -er 떼고 어미 추가.

| 주어 | 어미 | parler |
|---|---|---|
| je | -e | parle |
| tu | -es | parles |
| il/elle | -e | parle |
| nous | -ons | parlons |
| vous | -ez | parlez |
| ils/elles | -ent | parlent |

자주 쓰는 -er 동사: aimer, habiter, travailler, manger, chanter, écouter, regarder, arriver, chercher, penser.

발음: je/tu/il/elle/ils/elles 중 4개(je,il/elle,ils/elles 겹침 포함)가 /파흘/로 동일. nous/vous만 다르게 들림.
