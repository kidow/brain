# 프랑스어 동사 현재형

## 동사 활용 개념

프랑스어는 주어마다 동사 어미가 다름. 주어 생략 불가.

한국어 = 주어 달라도 어미 같음 (시제·높임만 바뀜).
프랑스어 = 주어가 바뀌면 어미도 바뀜.

예: parler (말하다) — -er 규칙동사

| 주어 | 동사 | 주어 | 동사 |
|---|---|---|---|
| je | parle | nous | parlons |
| tu | parles | vous | parlez |
| il/elle | parle | ils/elles | parlent |

발음: parle/parles/parlent 전부 /파흘/ — 어미 -e/-es/-ent 묵음. 귀로 구분 불가 → 주어 생략 불가 이유.

## être 활용 (이다/있다) — 완전 불규칙

| 주어 | être | 발음 |
|---|---|---|
| je | suis | /쒸이/ |
| tu | es | /에/ |
| il/elle | est | /에/ |
| nous | sommes | /솜/ |
| vous | êtes | /에트/ |
| ils/elles | sont | /송/ |

예: Je suis étudiant. / Il est grand. / Nous sommes amis.
형용사·직업·국적 연결 시 필수. il est / elle est + 형용사 패턴이 가장 빈번.
