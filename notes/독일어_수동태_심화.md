# 독일어 수동태 심화

## Vorgangspassiv vs Zustandspassiv — 두 종류의 수동태

독일어_동사에서 배운 수동태(werden+과거분사)는 정확히는 Vorgangspassiv(과정수동). 다른 종류로 Zustandspassiv(상태수동, sein+과거분사)가 있다.

| 종류 | 구조 | 의미 |
|------|------|------|
| Vorgangspassiv | werden+과거분사 | 동작이 진행되는 과정 |
| Zustandspassiv | sein+과거분사 | 동작이 끝난 뒤의 결과 상태 |

> Vorgangspassiv: Die Tür wird geöffnet. (문이 열리고 있다.)  
> Zustandspassiv: Die Tür ist geöffnet. (문이 열려 있다.)

한국어 "~되고 있다"(과정) vs "~되어 있다"(상태)와 대응.

독일어_형용사의 서술적 용법(sein+형용사)과 헷갈리기 쉬움:
> Zustandspassiv(동사 과거분사): Die Tür ist geöffnet.  
> 형용사 서술(순수 형용사): Die Tür ist offen.

뜻은 비슷하지만 문법적으로 다름: 하나는 동사 과거분사, 하나는 형용사.

핵심: **werden+과거분사=Vorgangspassiv(과정, ~되는 중). sein+과거분사=Zustandspassiv(상태, ~되어 있음). 조동사가 werden이냐 sein이냐로 구분.**

## 자동사 수동태 — 비인칭 수동태(Unpersönliches Passiv)

목적어 없는 자동사도 수동태 가능. 이땐 es를 형식 주어로 사용.

> 능동: Man tanzt hier viel. → 수동: Es wird hier viel getanzt.

대명사 노트의 형식 주어 es와 같은 원리(가리키는 대상 없음).

es는 1번 자리에서만 필수, 다른 요소가 1번이면 생략(V2 법칙):
> Es wird hier viel getanzt. / Hier wird viel getanzt. (es 사라짐)

행위자를 지우고 "일반적으로 일어나는 일" 강조:
> Es wird geraucht. / Hier wird nicht geraucht.(흡연 금지 표지판)

핵심: **자동사도 수동태 가능, 형식 주어 es 사용(진짜 주어 없음). es는 1번 자리에서만 필수, 다른 요소가 1번이면 생략. 행위자 지우고 일반적 행위 강조.**

## 화법조동사 수동태 — müssen/können + 과거분사 + werden

화법조동사와 수동태 결합 = "~되어야 한다/~될 수 있다".

구조: 화법조동사(2번째) + ... + 과거분사 + werden(원형, 문장 끝)
> Das muss gemacht werden. / Die Tür kann nicht geöffnet werden.

과거분사+werden(원형)이 문장 끝에 하나의 "수동 원형"처럼 붙음:
> 일반: Ich kann das machen. (원형 하나)  
> 수동: Das kann gemacht werden. (과거분사+werden)

능동→수동 전환(목적어→주어) 그대로 적용:
> 능동: Man muss das Fenster schließen. → 수동: Das Fenster muss geschlossen werden.

회화 예: Das darf nicht vergessen werden. / Hausaufgaben müssen bis Freitag abgegeben werden.

핵심: **화법조동사 수동태 = 화법조동사(2번째)+과거분사+werden(문장 끝, 원형). 능동→수동 전환 규칙(목적어→주어) 그대로 적용.**

## 수동태 대체 표현 — sich lassen + 원형

"~될 수 있다"는 sich lassen+원형으로도 자주 표현. 문법상 능동이지만 뜻은 수동.

구조: 주어(원래 목적어) + lässt sich + 원형(문장 끝)
> Die Tür lässt sich öffnen. = Die Tür kann geöffnet werden.  
> Das Problem lässt sich lösen. = Das Problem kann gelöst werden.

재귀동사(sich)+화법조동사(können) 구조가 결합된 것. können+수동보다 간결, 구어체 선호.

의무·허가엔 안 됨(가능성 표현만 대체 가능):
> ❌ Das lässt sich machen müssen. / ✅ Das muss gemacht werden.

비교:

| 표현 | 뉘앙스 |
|------|--------|
| kann + 과거분사 + werden | 표준 화법조동사 수동태(가능) |
| lässt sich + 원형 | 더 간결한 대체(가능, 구어체 선호) |

핵심: **sich lassen+원형 = kann+과거분사+werden의 간결한 대체(가능성 표현). 의무·허가는 대체 안 되고 화법조동사 수동태 그대로 씀.**
