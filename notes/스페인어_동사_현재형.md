# 스페인어 동사 현재형

## 동사 원형과 세 그룹: -ar/-er/-ir

스페인어 동사 원형(사전형)은 항상 이 세 어미 중 하나로 끝남 — 이게 활용 그룹을 결정.

| 그룹 | 예시 | 뜻 |
|------|------|-----|
| -ar | hablar | 말하다 |
| -er | comer | 먹다 |
| -ir | vivir | 살다 |

활용 = 어미(-ar/-er/-ir)를 떼고 인칭별 어미를 붙이는 것. 스페인어는 인칭(나/너/그/우리/너희/그들)마다 다른 어미가 붙음(한국어는 인칭 안 바뀜, 높임말만 바뀜).

-ar 그룹: `hablar`(말하다)의 어간은 `habl-`.

| 인칭 | 어미 | 활용형 |
|------|------|--------|
| yo(나) | -o | hablo |
| tú(너) | -as | hablas |
| él/ella/usted(그/그녀/당신) | -a | habla |
| nosotros(우리) | -amos | hablamos |
| vosotros(너희, 스페인) | -áis | habláis |
| ellos/ellas/ustedes(그들/당신들) | -an | hablan |

`hablo español.`(나는 스페인어를 말한다)

## -er 동사 활용

`comer`(먹다)의 어간은 `com-`.

| 인칭 | 어미 | 활용형 |
|------|------|--------|
| yo | -o | como |
| tú | -es | comes |
| él/ella/usted | -e | come |
| nosotros | -emos | comemos |
| vosotros | -éis | coméis |
| ellos/ellas/ustedes | -en | comen |

-ar와 비교: yo는 둘 다 `-o`로 동일(모든 그룹 공통). 나머지 인칭은 -ar의 `a`가 -er에선 `e`로 바뀜 — `hablas→comes`, `habla→come`, `hablamos→comemos`, `habláis→coméis`, `hablan→comen`.

이 대응 패턴만 기억하면 -er 동사는 -ar에서 모음만 치환하면 됨.

## -ir 동사 활용

`vivir`(살다)의 어간은 `viv-`. -er와 거의 같고 딱 두 곳만 다름.

| 인칭 | 어미 | 활용형 |
|------|------|--------|
| yo | -o | vivo |
| tú | -es | vives |
| él/ella/usted | -e | vive |
| nosotros | -imos | vivimos |
| vosotros | -ís | vivís |
| ellos/ellas/ustedes | -en | viven |

-er와 비교: yo/tú/él/ellos 네 인칭은 완전히 동일(`como=vivo`, `comes=vives`, `come=vive`, `comen=viven`). 다른 곳은 nosotros(`-emos`→`-imos`)와 vosotros(`-éis`→`-ís`) 딱 두 자리뿐.

정리: -ar/-er/-ir 세 그룹 전부 yo=`-o`로 공통. -er/-ir는 tú·él·ellos까지 공통이고, nosotros·vosotros에서만 갈림.

## yo형만 불규칙한 동사

많은 동사는 규칙 활용을 따르되 yo형만 예외적으로 바뀜. 나머지 인칭은 전부 규칙대로.

| 원형 | yo형(불규칙) | tú형(규칙) |
|------|-------------|-----------|
| hacer(하다) | hago | haces |
| poner(놓다) | pongo | pones |
| salir(나가다) | salgo | sales |
| tener(가지다) | tengo | tienes |
| venir(오다) | vengo | vienes |
| conocer(알다, 사람·장소) | conozco | conoces |
| dar(주다) | doy | das |
| ver(보다) | veo | ves |
| saber(알다, 사실) | sé | sabes |

패턴 그룹:
- `-go` 패턴: hacer→hago, poner→pongo, salir→salgo, tener→tengo, venir→vengo
- `-zco` 패턴: conocer→conozco(모음+cer/cir로 끝나는 동사 다수 공통)
- 완전 불규칙: dar→doy, ver→veo, saber→sé

## 어간모음변화 동사: e→ie

일부 동사는 강세 받는 어간 모음이 인칭에 따라 바뀜. `querer`(원하다)의 어간 `quer-`에서 강세 받는 `e`가 `ie`로 변함.

| 인칭 | 활용형 | 비고 |
|------|--------|------|
| yo | quiero | e→ie |
| tú | quieres | e→ie |
| él/ella/usted | quiere | e→ie |
| nosotros | queremos | 변화 없음 |
| vosotros | queréis | 변화 없음 |
| ellos/ellas/ustedes | quieren | e→ie |

패턴: nosotros·vosotros만 원래 모음 유지, 나머지 넷(yo/tú/él/ellos)은 전부 변화 — "부츠 모양(boot verb)"이라 부르는 이유가 이 넷을 표에서 묶으면 부츠 윤곽이 나오기 때문.

다른 e→ie 동사: `pensar`(생각하다)→pienso, `entender`(이해하다)→entiendo, `empezar`(시작하다)→empiezo.

## 어간모음변화 동사: o→ue

같은 부츠 패턴, `o`가 `ue`로 변함. `poder`(할 수 있다)의 어간 `pod-`에서 강세 받는 `o`가 `ue`로.

| 인칭 | 활용형 | 비고 |
|------|--------|------|
| yo | puedo | o→ue |
| tú | puedes | o→ue |
| él/ella/usted | puede | o→ue |
| nosotros | podemos | 변화 없음 |
| vosotros | podéis | 변화 없음 |
| ellos/ellas/ustedes | pueden | o→ue |

다른 o→ue 동사: `dormir`(자다)→duermo, `volver`(돌아가다)→vuelvo, `encontrar`(찾다)→encuentro.

예외: `jugar`(놀다/경기하다)는 어간이 `u`인데 `ue`로 변함(u→ue, 유일한 사례) → juego, juegas, juega, jugamos, jugáis, juegan.

## 어간모음변화 동사: e→i

세 번째 부츠 패턴, -ir 동사에서만 나타남. `pedir`(요청하다)의 어간 `ped-`에서 강세 받는 `e`가 `i`로 변함.

| 인칭 | 활용형 | 비고 |
|------|--------|------|
| yo | pido | e→i |
| tú | pides | e→i |
| él/ella/usted | pide | e→i |
| nosotros | pedimos | 변화 없음 |
| vosotros | pedís | 변화 없음 |
| ellos/ellas/ustedes | piden | e→i |

다른 e→i 동사: `servir`(제공하다)→sirvo, `repetir`(반복하다)→repito, `seguir`(따르다)→sigo.

세 가지 어간모음변화 요약: e→ie(querer), o→ue(poder), e→i(pedir, -ir만) — 셋 다 nosotros·vosotros는 항상 원형 유지가 공통 규칙.

## 완전 불규칙 동사: ser, estar, ir

이 셋은 어떤 규칙에도 안 맞는 완전 불규칙 — 통째로 암기.

| 인칭 | ser(본질) | estar(상태/위치) | ir(가다) |
|------|-----------|------------------|----------|
| yo | soy | estoy | voy |
| tú | eres | estás | vas |
| él/ella/usted | es | está | va |
| nosotros | somos | estamos | vamos |
| vosotros | sois | estáis | vais |
| ellos/ellas/ustedes | son | están | van |

- `estar`는 얼핏 규칙(-ar) 같지만 yo형 `estoy`가 불규칙, 나머지는 강세 위치(estás, está, están)만 예외.
- `ir`는 원형 `i-`가 활용형 어디에도 안 남을 만큼 완전 대체.

`ir a + 동사원형` = 가까운 미래("~할 것이다"): `Voy a comer.`(나는 먹을 것이다) — 영어 "going to"와 같은 구조.
