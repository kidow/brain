## 기본형 (じしょけい, dictionary form)

일본어 동사는 항상 "u" 소리로 끝남. 사전에 실리는 형태.

- 食べる (たべる) — 먹다
- 飲む (のむ) — 마시다
- 行く (いく) — 가다
- 見る (みる) — 보다

한국어 동사가 항상 "다"로 끝나는 것과 같은 원리.

## 동사 3그룹 분류

활용 규칙이 그룹마다 달라서 분류가 중요.

**1그룹 (u동사)** — 가장 많음. る 이외의 う단으로 끝나거나, る로 끝나도 바로 앞이 a·u·o 소리.
- 書く (かく) — 쓰다
- 飲む (のむ) — 마시다
- 話す (はなす) — 말하다
- 乗る (のる) — 타다 (る 앞이 "o")

**2그룹 (ru동사)** — る로 끝나고 바로 앞이 i·e 소리.
- 食べる (たべる) — 먹다 (る 앞이 "e")
- 見る (みる) — 보다 (る 앞이 "i")
- 起きる (おきる) — 일어나다

**3그룹 (불규칙)** — 딱 2개, 통째로 암기.
- する — 하다
- 来る (くる) — 오다

판별 공식:
1. る로 안 끝남 → 1그룹
2. る로 끝남 → 앞 소리 확인: i·e → 2그룹 / a·u·o → 1그룹
3. する·くる → 3그룹

## ます形 — 정중형 (polite form)

기본형은 반말. ます形은 존댓말 — 한국어 "~습니다"에 해당.

**1그룹** — 어미 う단 → い단으로 바꾸고 + ます
- 書く → 書きます (씁니다)
- 飲む → 飲みます (마십니다)
- 話す → 話します (말합니다)

**2그룹** — る 떼고 + ます
- 食べる → 食べます (먹습니다)
- 見る → 見ます (봅니다)

**3그룹** — 통암기
- する → します (합니다)
- 来る (くる) → 来ます (きます) (옵니다)

감각: 1그룹은 어미 소리 한 단 올림(く→き, む→み, す→し) / 2그룹은 る만 자름 / 3그룹은 암기.

조사 + ます형 결합 예: 私は本を読みます。나는 책을 읽습니다.

## ません — ます형 부정

"~하지 않습니다". 규칙: ます → ません (그룹 구분 없음).

- 食べます → 食べません (안 먹습니다)
- 飲みます → 飲みません (안 마십니다)
- します → しません (안 합니다)
- 来ます (きます) → 来ません (きません) (안 옵니다)

예: 私は肉を食べません。저는 고기를 안 먹습니다.

## ました / ませんでした — ます형 과거

**ました** ("~했습니다"): ます → ました
- 食べます → 食べました (먹었습니다)
- します → しました (했습니다)

**ませんでした** ("~하지 않았습니다"): ません → ませんでした
- 食べません → 食べませんでした (안 먹었습니다)

ます형 4형태:
| | 긍정 | 부정 |
|---|---|---|
| 현재/미래 | ~ます | ~ません |
| 과거 | ~ました | ~ませんでした |

예: 昨日、映画を見ました。어제 영화를 봤습니다.

## て形 — 연결형 (te-form)

여러 문형(진행형·요청·허가 등)의 기반이 되는 핵심 형태.

**2그룹** — る 떼고 + て: 食べる → 食べて / 見る → 見て
**3그룹** — 통암기: する → して / 来る → 来て(きて)

**1그룹** — 어미별 음편 규칙:
| 어미 | 변화 | 예시 |
|---|---|---|
| う・つ・る | → って | 買う→買って, 待つ→待って, 乗る→乗って |
| む・ぶ・ぬ | → んで | 飲む→飲んで, 遊ぶ→遊んで, 死ぬ→死んで |
| く | → いて | 書く→書いて |
| ぐ | → いで | 泳ぐ→泳いで |
| す | → して | 話す→話して |

예외: 行く(가다) → 行って (く 규칙 안 따름, 통암기)

암기: う・つ・る(입 앞쪽 발음)→って / む・ぶ・ぬ(콧소리)→んで

## ~ている — 진행형·상태 지속

て形 + いる = "~하고 있다". 정중형은 いる → います.

- 食べています。— 먹고 있습니다.
- 飲んでいます。— 마시고 있습니다.

두 가지 의미:
1. **동작 진행** — 今、勉強しています。지금 공부하고 있습니다.
2. **상태 지속(결과)** — 結婚しています。결혼한 상태입니다. / 窓が開いています。창문이 열려 있습니다.

주의: 結婚しました(결혼한 사건, 완료) vs 結婚しています(결혼한 상태, 지속) — 한국어 "결혼했어요"가 이미 상태 지속 의미를 담고 있어 혼동 포인트.

## ~てください — 요청

て形 + ください = "~해 주세요".

- 待ってください。— 기다려 주세요.
- ここに座ってください。— 여기 앉아 주세요.
- ゆっくり話してください。— 천천히 말해 주세요.

참고(부정 요청): ~ないでください("~하지 마세요")는 て形이 아니라 ない형(부정형) 기반. 예: 写真を撮らないでください。사진을 찍지 마세요.
