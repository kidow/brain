## 기본형 (じしょけい, dictionary form)

일본어 동사는 항상 "u" 소리로 끝남. 사전에 실리는 형태.

- 食べる (たべる) — 먹다
- 飲む (のむ) — 마시다
- 行く (いく) — 가다
- 見る (みる) — 보다

한국어 동사가 항상 "다"로 끝나는 것과 같은 원리.

## 동사 3그룹 분류

활용 규칙이 그룹마다 달라서 분류가 중요.

**1그룹 (u동사)** — 가장 많음. る 이외의 う단으로 끝나거나, る로 끝나도 바로 앞이 a·u·o 소리.
- 書く (かく) — 쓰다
- 飲む (のむ) — 마시다
- 話す (はなす) — 말하다
- 乗る (のる) — 타다 (る 앞이 "o")

**2그룹 (ru동사)** — る로 끝나고 바로 앞이 i·e 소리.
- 食べる (たべる) — 먹다 (る 앞이 "e")
- 見る (みる) — 보다 (る 앞이 "i")
- 起きる (おきる) — 일어나다

**3그룹 (불규칙)** — 딱 2개, 통째로 암기.
- する — 하다
- 来る (くる) — 오다

판별 공식:
1. る로 안 끝남 → 1그룹
2. る로 끝남 → 앞 소리 확인: i·e → 2그룹 / a·u·o → 1그룹
3. する·くる → 3그룹

## ます形 — 정중형 (polite form)

기본형은 반말. ます形은 존댓말 — 한국어 "~습니다"에 해당.

**1그룹** — 어미 う단 → い단으로 바꾸고 + ます
- 書く → 書きます (씁니다)
- 飲む → 飲みます (마십니다)
- 話す → 話します (말합니다)

**2그룹** — る 떼고 + ます
- 食べる → 食べます (먹습니다)
- 見る → 見ます (봅니다)

**3그룹** — 통암기
- する → します (합니다)
- 来る (くる) → 来ます (きます) (옵니다)

감각: 1그룹은 어미 소리 한 단 올림(く→き, む→み, す→し) / 2그룹은 る만 자름 / 3그룹은 암기.

조사 + ます형 결합 예: 私は本を読みます。나는 책을 읽습니다.
