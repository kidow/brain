## 분사구문이란 — 부사절을 분사로 압축

부사절을 `-ing` 분사로 줄인 표현. 뜻 동일, 문장 간결.

만드는 법:
1. 접속사 제거
2. 주어가 주절과 같으면 제거
3. 동사 → `-ing`

```
Because I was tired, I went home.
→ Being tired, I went home.
```

일상 대화보다 글쓰기에서 자주. 단, 관용 분사구문은 구어에서도 흔함:
```
Speaking of that, ...  /  Generally speaking, ...  /  Looking back, ...
```
