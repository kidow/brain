# 중국어 기초 어휘

## 숫자 1~10

| 숫자 | 한자 | 병음 | 성조 |
|------|------|------|------|
| 1 | 一 | yī | 1성 |
| 2 | 二 | èr | 4성 |
| 3 | 三 | sān | 1성 |
| 4 | 四 | sì | 4성 |
| 5 | 五 | wǔ | 3성 |
| 6 | 六 | liù | 4성 |
| 7 | 七 | qī | 1성 |
| 8 | 八 | bā | 1성 |
| 9 | 九 | jiǔ | 3성 |
| 10 | 十 | shí | 2성 |

10 이상: 자릿수 × 단위 나열.
- 11 = 十一 shíyī / 20 = 二十 èrshí / 25 = 二十五 èrshíwǔ / 100 = 一百 yībǎi

주의: 5 wǔ(우) ≠ 我 wǒ(워 = 나)

## 기본 인사 표현

| 상황 | 한자 | 병음 | 뜻 |
|------|------|------|----|
| 만날 때 | 你好 | nǐ hǎo | 안녕하세요 |
| 격식체 | 您好 | nín hǎo | 안녕하세요(경어) |
| 아침 | 早上好 | zǎoshang hǎo | 좋은 아침 |
| 헤어질 때 | 再见 | zàijiàn | 안녕히 가세요 |
| 감사 | 谢谢 | xièxie | 감사합니다 |
| 감사 답변 | 不客气 | bú kèqi | 천만에요 |
| 미안 | 对不起 | duìbuqǐ | 죄송합니다 |
| 괜찮아 | 没关系 | méiguānxi | 괜찮아요 |

핵심: 你好의 好(hǎo)="좋다" → 직역 "당신 좋아요"=안녕.  
谢谢는 같은 글자 반복 — 동사 반복은 가벼운/친근한 표현.

## 인칭대명사

| 인칭 | 한자 | 병음 | 뜻 |
|------|------|------|-----|
| 나 | 我 | wǒ | 나 |
| 너 | 你 | nǐ | 너 |
| 너(존칭) | 您 | nín | 당신 |
| 그 | 他 | tā | 그(남) |
| 그녀 | 她 | tā | 그녀 |
| 그것 | 它 | tā | 그것(사물/동물) |
| 우리 | 我们 | wǒmen | 우리 |
| 너희 | 你们 | nǐmen | 너희 |
| 그들 | 他们 | tāmen | 그들 |

핵심 규칙:
- 他/她/它 = 발음 완전히 동일(tā) — 글자만 다름. 맥락으로만 구별.
- 복수: 단수 + 们(men) → 규칙적. 我→我们, 你→你们, 他→他们
- 您(nín) = 你(nǐ)의 존칭.

주의: 们은 사람에게만. 사물 복수는 별도 표시 없음(중국어 명사는 단복수 무표지).

## 是(shì) — ~이다

주어 + 是 + 명사 구조. 중국어 be동사.

```
我是学生。 Wǒ shì xuéshēng.  나는 학생입니다.
她是老师。 Tā shì lǎoshī.    그녀는 선생님입니다.
这是书。   Zhè shì shū.      이것은 책입니다.
```

부정형: 是 앞에 不(bù).
```
我不是学生。 Wǒ bú shì xuéshēng.  나는 학생이 아닙니다.
```
(不+4성 是 → 변성조 규칙대로 bú)

의문형 — 두 가지:
```
你是学生吗？   Nǐ shì xuéshēng ma?      (吗 붙임)
你是不是学生？ Nǐ shì bu shì xuéshēng?  (是不是 반복, 정반의문문)
```

핵심: 인칭·시제에 따른 동사 변화 없음. 是는 항상 是 — am/is/are 구분 없음.

## 有(yǒu) — 있다/가지다

소유·존재를 나타냄. 是(A=B)와 별개 동사.

```
我有一本书。 Wǒ yǒu yì běn shū.  나는 책 한 권을 가지고 있다.
他有钱。     Tā yǒu qián.        그는 돈이 있다.
```

부정형: 不 아닌 没(méi) 씀 — 중국어 부정사 중 유일한 불규칙.
```
我没有钱。 Wǒ méiyǒu qián.  나는 돈이 없다.
```

의문형:
```
你有钱吗？   Nǐ yǒu qián ma?
你有没有钱？ Nǐ yǒu méiyǒu qián?  (정반의문문)
```

是 vs 有: 是=A는 B다(동일시) / 有=A가 B를 가짐·존재.

## 지시대명사 — 这(zhè) vs 那(nà)

한국어 이/그/저 3분법과 다름. 중국어는 가까움/멂 2분법.

| 한자 | 병음 | 뜻 | 거리 |
|------|------|------|------|
| 这 | zhè | 이(것) | 가까움 |
| 那 | nà | 그(것)/저(것) | 멀음 |

```
这是书。   Zhè shì shū.    이것은 책이다.
那是车。   Nà shì chē.     저것은/그것은 차다.
这个人     zhège rén       이 사람
那个人     nàge rén        그/저 사람
```

个(ge) 필수 삽입: 这/那 뒤에 명사 바로 안 붙고 양사 끼워야 함.
```
这书(X) → 这本书(O) — 本이 책 세는 양사
这人(X) → 这个人(O) — 个가 일반 양사
```

의문형: 这/那 자리에 哪(nǎ)=어느.
```
哪个是你的？ Nǎge shì nǐ de?  어느 것이 네 것이니?
```

핵심: 한국어 이/그/저 3단계를 중국어는 가까움(这)/멂(那) 2단계로만 구분 — "그것"·"저것" 구별 없음.

## 기본 양사(量詞) 5개

명사 앞에 반드시 붙는 세는 단위. 한국어 "책 한 권"의 "권"과 같은 개념.

| 양사 | 병음 | 쓰임 | 예 |
|------|------|------|-----|
| 个 | ge | 범용(대부분 명사) | 一个人 (사람 한 명) |
| 本 | běn | 책류 | 一本书 (책 한 권) |
| 张 | zhāng | 평평한 것 | 一张纸 (종이 한 장) |
| 条 | tiáo | 길고 가는 것 | 一条河 (강 하나) |
| 杯 | bēi | 컵에 담긴 것 | 一杯水 (물 한 잔) |

구조: 숫자 + 양사 + 명사 (한국어와 어순 동일).
```
一个苹果  yí ge píngguǒ   사과 한 개
两本书    liǎng běn shū   책 두 권
三张纸    sān zhāng zhǐ   종이 세 장
```

주의 — 양사 앞 "2"는 二 아닌 两(liǎng):
```
两个人(O) / 二个人(X)
```

个의 위상: 어떤 양사인지 애매하면 个로 대체로 통함 — 만능 양사.
