# 프랑스어 명사 성

## 성의 종류: 남성·여성 2가지

프랑스어 모든 명사 = 남성(masculin) 또는 여성(féminin). 중성 없음.

사물도 성을 가짐. 의미상 성별과 무관 — 역사적으로 굳어진 것.

| 단어 | 뜻 | 성 |
|---|---|---|
| le livre | 책 | 남성 |
| la table | 탁자 | 여성 |
| le soleil | 태양 | 남성 |
| la lune | 달 | 여성 |

성을 틀리면: 소통은 됨. 하지만 관사·형용사 연쇄 오류, 어색함, 일부 단어는 뜻 구분(le tour=여행 / la tour=탑).

**단어 외울 때 항상 le/la와 함께 외울 것.**

## 정관사 le / la / les

| 상황 | 관사 | 예 |
|---|---|---|
| 남성 단수 | le | le livre (그 책) |
| 여성 단수 | la | la table (그 탁자) |
| 복수 (성 무관) | les | les livres / les tables |

모음·묵음 h 앞: le/la → **l'** 로 축약. (l'ami, l'école)
모음 앞에선 l' 하나로 합쳐져 성이 안 보임.

기원: 라틴어 3성(남·여·중) → 중성이 5~9세기 남성으로 흡수 → 프랑스어 2성. 인도유럽조어(PIE)부터 이어진 체계.
