# 프랑스어 명사 성

## 성의 종류: 남성·여성 2가지

프랑스어 모든 명사 = 남성(masculin) 또는 여성(féminin). 중성 없음.

사물도 성을 가짐. 의미상 성별과 무관 — 역사적으로 굳어진 것.

| 단어 | 뜻 | 성 |
|---|---|---|
| le livre | 책 | 남성 |
| la table | 탁자 | 여성 |
| le soleil | 태양 | 남성 |
| la lune | 달 | 여성 |

성을 틀리면: 소통은 됨. 하지만 관사·형용사 연쇄 오류, 어색함, 일부 단어는 뜻 구분(le tour=여행 / la tour=탑).

**단어 외울 때 항상 le/la와 함께 외울 것.**

## 정관사 le / la / les

| 상황 | 관사 | 예 |
|---|---|---|
| 남성 단수 | le | le livre (그 책) |
| 여성 단수 | la | la table (그 탁자) |
| 복수 (성 무관) | les | les livres / les tables |

모음·묵음 h 앞: le/la → **l'** 로 축약. (l'ami, l'école)
모음 앞에선 l' 하나로 합쳐져 성이 안 보임.

## 부정관사 un / une / des

| 상황 | 관사 | 예 |
|---|---|---|
| 남성 단수 | un | un livre (책 한 권) |
| 여성 단수 | une | une table (탁자 하나) |
| 복수 | des | des livres / des tables |

un/une = 영어 a/an. des = 영어 some (프랑스어는 생략 불가).

부정문: un/une/des → **de** (모음 앞 d')
- J'ai un chat. → Je n'ai pas de chat.

## 명사 끝 형태로 성 추측

**여성 높은 끝**: -tion, -sion, -ette, -ée, -ure, -ence/-ance, -té, -ie
**남성 높은 끝**: -ment, -age, -isme, -eau, -eur, -et

패턴이지 법칙 아님. 예외 있음:
- le musée (-ée 끝이지만 남성)
- la photo (-o 끝이지만 여성)

-tion 끝은 거의 무조건 여성. 영어 -tion 단어와 대부분 대응.

## 형용사 성 일치

명사 성·수에 맞춰 형용사 형태 변함. 남성형 기본 + 여성 -e + 복수 -s.

| | 남성 단수 | 여성 단수 | 복수(남) | 복수(여) |
|---|---|---|---|---|
| petit | petit | petite | petits | petites |
| grand | grand | grande | grands | grandes |

발음: 남성 끝 자음 묵음 → 여성 -e 추가 시 살아남.
- grand /그헝/ → grande /그헝드/

불규칙 여성형 주요 예:
- beau → belle, nouveau → nouvelle, vieux → vieille, bon → bonne

## 형용사 위치: 명사 앞 vs 뒤

기본: **명사 뒤** (색·국적·형태 등 대부분).
- une voiture rouge, un homme français

예외 앞에 오는 것: **BAGS** (Beauty·Age·Goodness·Size)
- beau, joli / jeune, vieux, nouveau / bon, mauvais / petit, grand, gros

같은 형용사, 위치 따라 뜻 다름:
- un **grand** homme (위대한) vs un homme **grand** (키 큰)
- **ancien** ami (전 친구) vs ami **ancien** (오래된 친구)

## 명사 복수형

기본: 단수 + **-s** (묵음).

특수 규칙:
- -s/-x/-z 끝 → 변화 없음. le bras → les bras
- -eau/-au 끝 → **-x**. le gâteau → les gâteaux
- -al 끝 → **-aux**. le cheval → les chevaux
- -eu 끝 → **-x**. le jeu → les jeux
- 불규칙: l'œil → les **yeux**

복수 -s는 묵음 → **관사(les/des)가 복수의 진짜 신호**. 연음 있을 때만 s 소리 남(/z/로).

단/복수 구분: 명사 자체론 불가. 관사 발음(le→les), 연음, 동사 형태로 파악.

기원: 라틴어 3성(남·여·중) → 중성이 5~9세기 남성으로 흡수 → 프랑스어 2성. 인도유럽조어(PIE)부터 이어진 체계.
