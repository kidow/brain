## Q&A 대응 — 예상 밖 질문에 시간 벌기

"um..."으로 멈추는 대신, 질문을 받아들이는 짧은 문구로 시간을 벌면서 생각을 정리하는 게 Q&A의 기본기.

**질문 받았다는 신호 + 시간 벌기:**
```
That's a great question.        좋은 질문이네요.
Good point — let me think about that.
I'm glad you brought that up.   그 얘기 나와서 다행이네요.
```
목적은 "침묵을 안 만들면서 1~2초 버는 것" — 한국어 "아, 그 부분이요..."와 같은 기능.

**답 모를 때 — 얼버무리지 않고 정직하게:**
```
That's a great question — I don't have the exact number, but I can follow up.
I'd have to check on that and get back to you.
```
모르는 걸 아는 척하는 것보다 "확인 후 다시 알려주겠다"가 신뢰를 줌.

**질문이 불명확할 때 되묻기:**
```
Just to clarify, are you asking about X or Y?
Could you elaborate on what you mean by that?
```

**공격적인 질문 받았을 때 — 방어적으로 안 들리게:**
```
I see where you're coming from. Here's how we thought about it...
That's a fair concern. Let me address that.
```
상대 관점을 먼저 인정한 뒤 설명으로 넘어가는 구조 — `영어_회화`의 "부드러운 반박" 구조와 같은 원리.

## 슬라이드 언어 — 문장이 아니라 조각으로

슬라이드는 완전한 문장이 아니라 키워드·구(phrase) 위주로 씀. 완전한 문장은 입으로 채움.

**슬라이드 vs 말하는 문장:**
```
슬라이드: Q3 Revenue: +15% YoY
말로:     Our Q3 revenue increased by 15% compared to the same quarter last year.
```

**동사 없는 명사구 헤드라인:**
```
❌ We are increasing our market share.
✅ Increasing Market Share
```
be동사·주어 생략하고 핵심 개념만 — `영어_뉴스·시사_어휘`의 헤드라인체 압축 원리가 슬라이드 제목에도 적용됨.

**병렬 구조 필수 (불릿 포인트):**
```
❌ Improve efficiency / Cutting costs / We will expand
✅ Improve efficiency / Cut costs / Expand market
```
`영어_문장_구조_심화`의 병렬 구조가 슬라이드에서 특히 중요 — 형태가 안 맞으면 청중이 시각적으로 바로 알아챔.

**화살표·기호로 관계 압축:**
```
Cost ↓ → Profit ↑
Q1 → Q2 → Q3: steady growth
```

**숫자 슬라이드 관용구:**
```
YoY (Year over Year)         전년 대비
QoQ (Quarter over Quarter)   전분기 대비
TBD (To Be Determined)       미정
KPI (Key Performance Indicator)   핵심 성과 지표
```

**핵심 원칙:** 슬라이드는 "읽는 용도"가 아니라 "보는 용도" — 청중이 슬라이드를 읽는 동안 발표자 말을 안 듣게 되므로, 최소 정보만 담고 설명은 입으로 채우는 게 원칙.

## 데이터·그래프 설명 표현 — 추이를 말로 그리기

**증가 동사 — 강도 순:**
```
rise / increase     오르다 (중립)
grow                성장하다 (긍정적)
surge / soar / spike   급등하다 (급격함)
```

**감소 동사 — 강도 순:**
```
fall / decrease     내리다 (중립)
decline             감소하다 (점진적)
drop / plunge / plummet   급락하다 (급격함)
```

**속도·양상 부사:**
```
steadily      꾸준히         grew steadily
gradually     점진적으로      declined gradually
sharply       급격히         rose sharply
dramatically  극적으로       increased dramatically
slightly      약간          dropped slightly
```

**추이 설명 문장 패턴:**
```
Revenue rose from $2M to $3M over the past year.   (수치 + from A to B + 기간)
Sales have been steadily increasing since Q1.      (현재완료진행 + 부사 + since)
```

**정체·변동 없음:**
```
remain flat / stay stable / plateau   변동 없이 유지되다
fluctuate                             오르내리다
```

**그래프 특정 지점 가리키기:**
```
As you can see here, ...
This spike in March was due to...
If we look at the right side of the chart, ...
```

**패턴 종합:** 동사(방향) + 부사(속도) + 전치사구(수치·기간) 조합이 기본 골격 — `영어_전치사`의 from/to/since/by가 여기서 활용됨.
