## 프레젠테이션 오프닝

청중 주목 = "Attention please" ❌ → 내용 자체가 끌어당기는 방식.

**오프닝 패턴:**
```
질문:   "How many of you have ever...?"
통계:   "Every 60 seconds, 500 hours of video are uploaded to YouTube."
이야기: "Last year, one of our clients lost $2 million in a single week."
주장:   "Today I want to convince you that [주장]."
```

**도입 표현:**
```
"Let's get started."       시작하죠.
"Let me jump right in."    바로 시작할게요.
"I'll keep this brief."    짧게 말씀드릴게요.
"Before we dive in, ..."   들어가기 전에,
```

첫 30초 안에 "왜 이걸 들어야 하는가"를 답해야 함.  
"Good morning, my name is..." → 청중 관심 끊김.

**Q:** 청중을 주목시킬 때 명령형 표현은 안 쓰나요?  
**A:** 맞음. 행동 자체(말 시작, 슬라이드 전환, 잠깐 침묵)가 주목 신호. 명령형은 권위적·어색함.

## 회의에서 의견 말하기

**의견 제시:**
```
I'd like to suggest that...          제안하고 싶은 건,
From my perspective, ...             제 관점에서는,
One thing worth considering is...    고려할 만한 건,
```

**부드러운 반대 — 인정 + 전환 구조:**
```
That's a valid point, but...
I see where you're coming from, however...
I'd push back on that a little.
Have we considered the possibility that...?
```

**동의:**
```
I'm on board with that.              동의합니다.
Absolutely, and I'd add that...      맞아요, 거기에 덧붙이자면,
```

직접적 반대("No, that's wrong") → 관계 손상.  
반드시 **인정 + 전환** 구조로.

## 회의 진행 표현

**시작·진행:**
```
Let's go over the agenda.              안건을 살펴보겠습니다.
I'd like to hand it over to [이름].   [이름]에게 넘기겠습니다.
Can we take this offline?              회의 후에 따로 얘기할까요?
```

**명확화·확인:**
```
Could you clarify what you mean by...?
Just to confirm, are you saying...?
Let me make sure I understand...
```

**시간 관리:**
```
We're running low on time.     시간이 얼마 안 남았어요.
Let's table this for now.      이건 일단 보류하죠.
Can we come back to this?      이건 나중에 다시 다룰까요?
```

**마무리:**
```
To summarize what we've agreed on...
Let's wrap up.
Action items: [이름] will [동작] by [날짜].
```

## 협상 표현

**제안:**
```
We'd be willing to...              ~할 용의가 있습니다.
What if we...?                     ~하면 어떨까요?
Could we explore the option of...?
```

**조건 걸기:**
```
We can do X, provided that...      ~을 조건으로 X 가능합니다.
That works for us, as long as...   ~한다면 괜찮습니다.
```

**양보·타협:**
```
We can be flexible on...           ~에 대해선 유연하게 볼 수 있어요.
That's a stretch for us, but...    저희에겐 좀 빠듯하지만,
Let's meet halfway.                중간 지점을 찾아봅시다.
```

**거절·보류:**
```
I'm afraid that's not something we can agree to.
We'd need to think about that.
That's outside our budget / scope.
```

"No" 는 가급적 안 씀. "That's difficult, but what if..." 형식으로 대안 제시.

## 프레젠테이션 본문 흐름

**섹션 전환:**
```
Moving on to...                      ~로 넘어가겠습니다.
Now let's look at...                 이제 ~를 살펴보겠습니다.
This brings me to my next point.     다음 포인트로 이어집니다.
```

**강조:**
```
The key takeaway here is...          핵심은,
I want to draw your attention to...  주목해주셨으면 하는 건,
What this means in practice is...    실제로 의미하는 건,
```

**데이터 설명:**
```
As you can see from this chart...    이 차트에서 보시다시피,
This figure represents...            이 수치는 ~를 나타냅니다.
```

**마무리:**
```
To wrap up, ...       마무리하자면,
The bottom line is... 결론은,
```

## 피드백 주고받기

**줄 때 — SBI 모델 (Situation + Behavior + Impact):**
```
"In yesterday's meeting (S), when you interrupted the client (B),
it made it harder for them to finish their point (I)."
```
직접 평가("You were rude") 대신 상황·행동·영향으로.

**긍정:**
```
Great work on...
I really appreciated how you...
That was exactly what we needed.
```

**개선:**
```
One thing that might help is...     도움이 될 것 같은 건,
Have you considered...?
Next time, it might be worth...     다음엔 ~해보는 게 어떨까요?
```

**받을 때:**
```
Thanks for the feedback.
That's helpful — I'll keep that in mind.
Could you give me an example?
```

"That's not fair" / "I disagree" → 방어적으로 들림. 먼저 경청, 나중에 맥락 설명.
