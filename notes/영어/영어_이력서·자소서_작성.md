## 액션 동사(Action Verb) — "책임졌다"가 아니라 "무엇을 했다"

**약한 패턴 — 책임·역할만 서술:**
```
Responsible for managing a team.
In charge of customer service.
Worked on a marketing project.
```
"responsible for", "in charge of", "worked on"은 역할을 말할 뿐 행동을 말하지 않음.

**강한 패턴 — 액션 동사로 시작:**
```
Managed a team of 5.
Led customer service operations.
Launched a marketing campaign.
```
동사 하나로 바로 행동이 드러남.

**액션 동사 카테고리별 예시:**
```
리더십:    led / directed / managed / supervised / coordinated
성과:      achieved / improved / increased / reduced / generated
창조:      built / designed / developed / created / launched
분석:      analyzed / evaluated / researched / identified
```

**핵심 규칙:** 이력서 문장은 거의 항상 동사로 시작. "Responsible for X" 구조를 피하고 X를 행동 동사로 바꿔 문장을 새로 시작.
```
❌ Responsible for increasing sales.
✅ Increased sales by developing a new outreach strategy.
```

한국어 이력서의 "~업무를 담당함"이 전형적인 약한 패턴 — 영어 이력서는 "담당함" 대신 "무엇을 했는가"로 시작하는 게 기본.
