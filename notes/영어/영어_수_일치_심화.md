## the number of vs a number of — 관사 하나가 단복수를 뒤집음

**the number of + 복수명사 → 단수 취급**
```
The number of students is increasing.
```
"학생 수"라는 하나의 숫자·개념 자체가 주어. the는 "그 특정한 수치"를 가리키므로 단수.

**a number of + 복수명사 → 복수 취급**
```
A number of students are absent today.
```
"많은 학생들"이라는 뜻으로 실질적 주어는 students(복수). a number of는 "many"와 같은 수량 형용사처럼 작동.

**판별 원리:**
```
the number of = "그 숫자"라는 단일 개념이 진짜 주어 → 단수
a number of   = "많은 ~"이라는 수식어일 뿐, 뒤 명사가 진짜 주어 → 복수
```

**핵심 이미지:** the number of는 "숫자 그 자체"를 세는 것(하나의 값), a number of는 "많은"이라는 수식어(뒤의 명사가 본질).

## 집합명사(Collective Nouns) — 하나의 집단인가, 구성원들인가

team, family, staff, government 같은 명사는 상황에 따라 단수·복수 둘 다 가능.

**단수 취급 — 집단을 하나의 단위로 볼 때**
```
The team is playing well this season.
```
팀 전체가 한 덩어리로 행동하는 느낌.

**복수 취급 — 집단 구성원 개개인의 행동으로 볼 때**
```
The team are arguing among themselves.
```
팀원들끼리 서로 다투는 상황 — 개별 구성원들의 행동이므로 복수. 하나의 팀이 자기 자신과 싸울 순 없으니 논리적으로도 복수가 맞음.

**미국식 vs 영국식:**
```
미국식: 거의 항상 단수 취급 (The team is...)
영국식: 맥락에 따라 복수도 자연스러움 (The team are...)
```
영국식 스포츠 기사에서 "England are playing"처럼 복수 취급이 흔한 이유.

**판별 테스트:** 집단이 하나로 행동하는가, 그 안의 개인들이 각자 행동하는가.
```
The committee has decided.              (하나의 결정 — 단수)
The committee have different opinions.  (각자 다른 의견 — 복수)
```

**대표적 집합명사:** team, family, staff, government, committee, audience, crew, jury
