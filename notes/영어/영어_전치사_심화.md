## 이동·투과 전치사 — through vs beyond vs over vs onto

**through — 안을 통과**
```
walk through the tunnel   (터널 속을 지나감 — 내부 통과)
```
공간의 안쪽을 관통. 3차원 내부를 지나가는 느낌.

**beyond — 어떤 지점 너머**
```
The store is beyond the bridge.   (다리를 지나 그 너머에)
```
경계선을 넘어선 저쪽. 통과 동작이 아니라 "그 지점 이후"라는 정적인 위치 관계.

**onto — 표면 위로 이동**
```
jump onto the table   (테이블 위로 뛰어오름)
```
to(방향) + on(표면 접촉) 결합. 원래 없던 곳에서 표면 위로 올라가는 이동.

**over — 위를 넘어감**
```
jump over the fence   (울타리를 위로 넘어감)
```
표면에 닿지 않고 위쪽 공간을 지나감. onto는 도착지가 표면 위, over는 도착지가 반대편.

**핵심 구분 축:**
```
through = 내부 통과 (3차원 속)
beyond  = 지점 너머의 위치 (정적, 이동 동사 없어도 됨)
onto    = 표면 위로 도착 (이동 + 표면 접촉)
over    = 위쪽으로 넘어감 (이동 + 접촉 없음)
```
beyond만 유일하게 "통과 동작"이 아니라 "위치 관계". beyond the bridge는 다리를 건너는 동작이 아니라 다리 저편이라는 장소를 가리킴.

## 접촉·부착 전치사 — on vs onto vs against

**on — 접촉 상태 (정적)**
```
The book is on the table.   (책이 테이블 위에 놓여 있음 — 상태)
```
이미 접촉해 있는 상태. 이동 동사 없어도 성립.

**onto — 접촉으로의 이동**
```
The book fell onto the table.   (책이 테이블 위로 떨어짐 — 이동 후 접촉)
```
on의 이동 버전. 원래 접촉해 있지 않다가 접촉하게 되는 과정.

**against — 밀착·저항 접촉**
```
lean against the wall   (벽에 기대다)
```
단순 접촉이 아니라 힘이 작용하는 접촉. 밀거나 기대거나 부딪히는 방향성 있는 압력.

**on vs against:**
```
The picture is on the wall.   (벽에 걸려 있음 — 중력 방향 접촉, 정적)
He pushed against the wall.   (벽을 향해 밂 — 힘의 방향, 저항)
```
on은 중력이 만드는 자연스러운 접촉. against는 누군가의 의도적인 힘이 만드는 접촉.

**정리:**
```
on      = 접촉 상태 (정적)
onto    = 접촉으로 이동 (동적 + on의 결과)
against = 힘이 실린 접촉 (밀착·저항)
```
한국어 "~에" 하나로 뭉뚱그려지는 걸 영어는 접촉의 성질(정적/동적/힘)로 세분화함.

## 시간 전치사 심화 — by vs until vs before

**by — 마감 시한 (완료 시점)**
```
Submit the report by Friday.   (금요일까지 제출 — 금요일 또는 그 전 완료)
```
동작이 그 시점 이전에 끝나야 함. 금요일 자체도 마감일에 포함.

**until — 지속 (계속되다가 그 시점에 끝남)**
```
I'll be here until Friday.   (금요일까지 계속 머묾)
```
동작이 그 시점까지 계속 진행됨. 시점에 동작이 끝남.

**before — 그 이전 (시점 자체는 제외)**
```
Finish it before Friday.   (금요일 되기 전 끝내라 — 금요일 제외, 목요일까지)
```

**동사 성격과 매칭:**
```
완료 동사(submit, finish, arrive) + by/before
지속 동사(stay, wait, work) + until
```
"I'll wait by 5pm" ❌ — wait는 지속 동사라 by와 안 맞음. "I'll wait until 5pm" ✅.

한국어 "~까지"가 완료·지속을 구분 안 하는 것과 대조적. 영어는 동사의 완료성 여부로 전치사가 갈림.

## 원인·수단 전치사 — by vs with vs through vs from

**by — 행위자·방법**
```
The window was broken by John.        (행위자)
I travel by train.                    (교통 수단)
```

**with — 도구 (손에 쥐고 쓰는 것)**
```
I cut the paper with scissors.
```
직접 손에 쥐고 사용하는 물리적 도구.

**through — 매개체·과정 (간접적으로 거쳐감)**
```
I learned it through experience.
We communicate through email.
```
직접 손에 쥐는 게 아니라 그것을 거쳐서 결과에 도달.

**from — 출처·원인 (근본 원인·재료)**
```
The bridge is made from steel.
He suffers from a rare disease.
```

**흔한 실수:**
```
❌ We talked with phone.
✅ We talked on the phone. / through the phone app.
```
전화 통화 자체는 관용적으로 on the phone(접촉면 은유), 앱·화상통화 같은 매개체는 through.

구분 기준: 물리적으로 손에 쥐는가(with) vs 거쳐가는가(through) vs 근본 출처인가(from).
