## 어근 (Root) — 단어의 핵심 의미

어근 하나를 알면 관련 단어 수십 개를 추론 가능.

| 어근 | 의미 | 예시 |
|---|---|---|
| `port` | 운반하다 | transport, import, export, portable, support |
| `dict` | 말하다 | predict, contradict, dictionary, verdict |
| `script/scrib` | 쓰다 | describe, prescription, manuscript, subscribe |
| `rupt` | 끊다·깨다 | interrupt, erupt, corrupt, bankrupt |
| `vis/vid` | 보다 | vision, visible, video, provide, evidence |
| `duc/duct` | 이끌다 | produce, educate, conduct, introduce |

**추론 예시:**
```
"benediction" = bene(좋은) + dict(말하다) + ion(명사) = 축복의 말
```

한국어 한자 어근(生·學·國)으로 단어 추론하는 것과 같은 원리.

## 접두사 (Prefix)

**부정·반대:**

| 접두사 | 의미 | 예시 |
|---|---|---|
| `un-` | 아닌·해제 | unhappy, undo, unlock |
| `in-/im-/ir-/il-` | 아닌 | incorrect, impossible, irregular, illegal |
| `dis-` | 반대·분리 | disagree, disconnect |
| `non-` | 비- | nonprofit, nonsense |
| `anti-` | 반대 | antibiotic, antivirus |

**수·위치·시간:**

| 접두사 | 의미 | 예시 |
|---|---|---|
| `pre-` | 이전 | preview, prehistoric |
| `post-` | 이후 | postwar, postpone |
| `over-` | 초과·위 | overwork, overcome |
| `under-` | 부족·아래 | underestimate, underground |
| `re-` | 다시 | rewrite, reconsider |
| `super-` | 초월 | superhero, supervisor |

**추론 예시:**
```
"unprecedented" = un(아닌) + precedent(선례) + ed → 전례 없는
```

## 접미사 (Suffix) — 품사를 바꾸는 뒷붙이

**동사 → 명사:**

| 접미사 | 예시 |
|---|---|
| `-tion/-sion` | decide→decision, discuss→discussion |
| `-ment` | develop→development, agree→agreement |
| `-er/-or` | teach→teacher, act→actor |
| `-ance/-ence` | perform→performance, exist→existence |

**형용사 → 명사:**

| 접미사 | 예시 |
|---|---|
| `-ness` | happy→happiness, dark→darkness |
| `-ity` | possible→possibility, real→reality |

**명사/동사 → 형용사:**

| 접미사 | 예시 |
|---|---|
| `-ful / -less` | helpful / hopeless |
| `-able/-ible` | readable, flexible |
| `-ous` | dangerous, famous |
| `-al` | national, musical |

**형용사 → 부사:** `-ly` → quickly, carefully

실전: 모르는 단어 끝 접미사로 품사 파악 → 문장 구조 이해 → 의미 추론.

## 콜로케이션 (Collocation)

문법적으로 맞아도 원어민이 안 쓰는 조합이 있음.

**동사 + 명사:**
```
make a decision ✅ / do a decision ❌
do homework ✅ / make homework ❌
take a photo ✅ / make a photo ❌
pay attention ✅ / give attention ❌
```

**형용사 + 명사:**
```
strong coffee / heavy rain / big mistake
(powerful coffee / strong rain / large mistake ❌)
```

**동사 + 부사:**
```
deeply understand / highly recommend / greatly appreciate
```

**핵심:** 번역 직역은 콜로케이션을 모름. "강한 비" → `strong rain` ❌ → `heavy rain` ✅. 단어 하나가 아닌 **짝**으로 암기.

## 다의어 vs 동음이의어

**다의어 — 하나의 단어, 파생된 여러 의미:**
```
"bank" → 은행 / 강둑 / 기울다 / 저장하다
```
어원이 같고 의미가 확장. 사전에서 같은 항목.

**동음이의어 — 형태 같지만 기원이 다른 별개 단어:**
```
"bat" = 야구 방망이 / 박쥐  (어원 다름)
"right" = 오른쪽 / 옳은 / 권리
```

**학습 전략:**
- 단어를 문장 안에서 암기
- 자주 등장하는 맥락 2~3개를 함께 저장
- 새 의미 발견 시 기존 의미와 연결해서 기억

## 어휘 수준 조절 — 격식·비격식·중립

| 비격식 | 중립 | 격식 |
|---|---|---|
| get | obtain | acquire |
| use | utilize | employ |
| show | demonstrate | illustrate |
| need | require | necessitate |
| help | assist | facilitate |
| start | begin | commence |

**선택 기준:**
- 대화·이메일 → 중립 또는 비격식
- 보고서·논문·공문 → 격식

**함정:** 격식 남발이 오히려 명확성을 해침.
```
❌ "I shall endeavour to ascertain the veracity of your assertion."
✅ "I'll check if what you said is true."
```

## 동의어의 미묘한 차이 (Near-synonyms)

**말하다:**
```
say   — 내용 중심: "I said I was tired."
tell  — 청자 중심: "I told him I was tired."
speak — 말하는 행위: "She spoke for an hour."
talk  — 대화·교류: "We talked all night."
```

**보다:**
```
see   — 자연스럽게 눈에 들어옴: "I saw her."
look  — 의도적 시선: "Look at this."
watch — 움직이는 것 집중해서 봄: "Watch the game."
```

**크다:**
```
big   — 일반: "a big house"
large — 격식·규모: "a large company"
great — 위대함·정도: "a great idea" (크기 아님)
huge  — 압도적: "a huge difference"
```

**행복하다:**
```
happy    — 일반
glad     — 특정 이유: "I'm glad you came."
pleased  — 만족·격식: "I'm pleased to meet you."
delighted — 강한 기쁨·격식
```

동의어 사전만 보지 말고 예문에서 차이 체감.
