## 인사와 안부 — How are you 넘어서기

**실제로 쓰는 안부:**
```
How's it going?       어떻게 지내?
How are things?       별일 없어?
What's up?            뭐해? / 어때?
How have you been?    그동안 잘 지냈어?
Long time no see!     오랜만이야!
```

**자연스러운 대답:**
```
Pretty good, thanks.    꽤 잘 지내.
Not bad.                나쁘지 않아.
Can't complain.         그냥저냥.
Same old, same old.     맨날 똑같지.
Busy as always.         항상 바쁘지 뭐.
Could be worse.         더 나쁠 수도 있지.
```

**대답 후 반드시 되묻기:**
```
Pretty good, thanks. You?
Not bad. How about you?
```
되묻지 않으면 대화 끊김.

## 맞장구 표현

**들었다는 신호:**
```
Yeah. / Uh-huh. / Right. / I see.
```

**공감·동의:**
```
Exactly!              맞아!
I know, right?        그러게.
Tell me about it.     내 말이.
That makes sense.     그렇겠네.
Totally.              완전.
```

**놀람·반응:**
```
No way!               설마! / 진짜?
Are you serious?      진심이야?
That's crazy.         말도 안 돼.
Good for you!         잘됐다!
```

**부드러운 반박:**
```
I get what you mean, but...
That's a good point, but...
I'm not sure about that.
```
