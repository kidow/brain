## 인사와 안부 — How are you 넘어서기

**실제로 쓰는 안부:**
```
How's it going?       어떻게 지내?
How are things?       별일 없어?
What's up?            뭐해? / 어때?
How have you been?    그동안 잘 지냈어?
Long time no see!     오랜만이야!
```

**자연스러운 대답:**
```
Pretty good, thanks.    꽤 잘 지내.
Not bad.                나쁘지 않아.
Can't complain.         그냥저냥.
Same old, same old.     맨날 똑같지.
Busy as always.         항상 바쁘지 뭐.
Could be worse.         더 나쁠 수도 있지.
```

**대답 후 반드시 되묻기:**
```
Pretty good, thanks. You?
Not bad. How about you?
```
되묻지 않으면 대화 끊김.

## 맞장구 표현

**들었다는 신호:**
```
Yeah. / Uh-huh. / Right. / I see.
```

**공감·동의:**
```
Exactly!              맞아!
I know, right?        그러게.
Tell me about it.     내 말이.
That makes sense.     그렇겠네.
Totally.              완전.
```

**놀람·반응:**
```
No way!               설마! / 진짜?
Are you serious?      진심이야?
That's crazy.         말도 안 돼.
Good for you!         잘됐다!
```

**부드러운 반박:**
```
I get what you mean, but...
That's a good point, but...
I'm not sure about that.
```

## 시간 벌기

**필러:**
```
Well... / Hmm... / I mean... / You know...
Let me think...     생각해 볼게.
```

**질문 되받아치기:**
```
That's a good question.         (시간 벌기)
What do you mean exactly?       정확히 어떤 의미야?
Could you say that again?       다시 말해줄 수 있어?
Sorry, I didn't catch that.     못 들었어.
```

**모를 때:**
```
I'm not sure, to be honest.     솔직히 잘 모르겠어.
I have no idea.                 전혀 모르겠어.
I'd have to think about that.   생각해봐야 할 것 같아.
```

## 화제 시작·전환·마무리

**화제 시작:**
```
By the way, ...           그런데, / 참,
Guess what!               있잖아, 뭔지 알아?
Oh, that reminds me...    아, 그거 말하니까 생각나는데...
Speaking of which...      말 나온 김에...
```

**화제 전환:**
```
Anyway, ...               아무튼,
On a different note, ...  다른 얘기인데,
Getting back to what you said...  아까 말한 거 다시 말하면,
```

**대화 마무리:**
```
I should get going.           이제 가봐야 할 것 같아.
I'll let you go.              이만 보내줄게. (상대 바쁠 때 배려)
It was great talking to you.  얘기 즐거웠어.
Let's catch up soon.          곧 또 보자.
Take care!                    잘 지내!
```

## 감사·사과·칭찬

**감사:**

I owe you one.                신세 졌어.
You're a lifesaver.           완전 살았다!
```

```
No problem. / Anytime. / Of course! / My pleasure.

**사과 수락:**

```

**칭찬:**
```

## 요청·부탁 — 강도 조절

**공손함 스펙트럼:**
```
Give me water.                     (명령 — 무례)
Can you give me water?             (가벼운 요청)
Could you give me water?           (더 공손)
Would you mind giving me water?    (매우 공손)
```

**허락 구하기:**
```
Can I use your pen?       (구어)
May I use your pen?       (격식)
Is it okay if I sit here? (가장 부드러운)
```

**거절:**
```
I'm sorry, I can't.
I wish I could, but...
I'm a bit tied up right now.   지금 좀 바빠서.
Maybe another time.            다음에.
```
거절 시 짧은 이유 붙이면 관계 안 상함.

## 의견 말하기

**내 생각 표현:**
```
I think... / In my opinion, ... / Personally, ...
As far as I'm concerned, ...   내가 보기엔,
If you ask me, ...             내 생각을 말하자면,
```

**확신 강도:**
```
강함:  I'm convinced that... / I strongly believe...
중립:  I think... / It seems to me...
약함:  I'm not sure, but... / It might be...
```

**동의·부동의:**
```
동의:    I agree. / Absolutely. / That's what I think too.
부분:    I see your point, but... / You have a point, but...
부동의:  I disagree. / I see it differently. (부드러움)
```

**의견 묻기:**
```
What do you think?         어떻게 생각해?
What's your take on this?  이거 어떻게 봐?
```
