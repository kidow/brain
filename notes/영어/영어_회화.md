## 인사와 안부 — How are you 넘어서기

**실제로 쓰는 안부:**
```
How's it going?       어떻게 지내?
How are things?       별일 없어?
What's up?            뭐해? / 어때?
How have you been?    그동안 잘 지냈어?
Long time no see!     오랜만이야!
```

**자연스러운 대답:**
```
Pretty good, thanks.    꽤 잘 지내.
Not bad.                나쁘지 않아.
Can't complain.         그냥저냥.
Same old, same old.     맨날 똑같지.
Busy as always.         항상 바쁘지 뭐.
Could be worse.         더 나쁠 수도 있지.
```

**대답 후 반드시 되묻기:**
```
Pretty good, thanks. You?
Not bad. How about you?
```
되묻지 않으면 대화 끊김.

## 요청·부탁 — 강도 조절

**공손함 스펙트럼:**
```
Give me water.                     (명령 — 무례)
Can you give me water?             (가벼운 요청)
Could you give me water?           (더 공손)
Would you mind giving me water?    (매우 공손)
```

**허락 구하기:**
```
Can I use your pen?       (구어)
May I use your pen?       (격식)
Is it okay if I sit here? (가장 부드러운)
```

**거절:**
```

I wish I could, but...

Maybe another time.            다음에.
```
거절 시 짧은 이유 붙이면 관계 안 상함.
