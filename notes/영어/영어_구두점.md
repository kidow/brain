## 마침표·물음표·느낌표

**마침표 `.`:** 평서문·명령문 종결. 약어(Dr. / Mr. / e.g. / etc.)
- 미국식: 약어 끝 마침표 사용 / 영국식: 생략 가능 (Dr Smith)

**물음표 `?`:** 직접 의문문에만. 간접의문문에는 마침표.
```
Are you coming?              ✅
I wonder if he's coming.     ✅ (마침표)
I wonder if he's coming?     ❌
```

**느낌표 `!`:** 강한 감정·감탄. 남발하면 효과 없어짐. 격식 글에서 절제.
