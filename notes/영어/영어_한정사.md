## each vs every — 개별성 vs 전체성

**each — 하나씩 따로 떼어서 보는 시선**
```
Each student received a certificate.
```
학생들을 한 명 한 명 개별적으로 인식하며 말하는 느낌.

**every — 예외 없이 전부를 하나의 규칙으로 묶는 시선**
```
Every student received a certificate.
```
"학생이라면 다 받았다"는 전체 규칙 진술. 개별성보다 "예외 없음"에 초점.

**결정적 차이 — each는 2개 이상, every는 3개 이상이 자연스러움:**
```
Each of the two brothers has a car.   ✅
Every one of the two brothers... ❌   (어색함)
```
each는 개체 하나하나에 집중해 소규모 집합에도 어울림. every는 전체 집단 뉘앙스가 강해 다수일 때 자연스러움.

**each는 of와 함께 부분 추출 가능, every는 제한적:**
```
Each of us has a role.       ✅
Every one of us has a role.  ✅ (every는 반드시 "every one of"로)
```
`every of us` ❌ — every는 홀로 of 뒤에 못 옴.

**핵심 이미지:** each = 손가락으로 하나씩 짚어가며 확인 / every = 도장 찍듯 전체에 똑같이 적용.

## few/a few, little/a little — 있음 vs 없음의 관점 차이

**few / little — 부정적 관점 ("거의 없다")**
```
I have few friends.        (친구가 거의 없다 — 부족함 강조)
There is little hope.      (희망이 거의 없다 — 절망적 뉘앙스)
```

**a few / a little — 긍정적 관점 ("그래도 조금은 있다")**
```
I have a few friends.      (친구가 몇 명 있다 — 있음에 초점)
There is a little hope.    (희망이 조금 있다 — 그래도 있음)
```

**핵심 — 숫자가 아니라 화자의 태도가 관사를 결정:**
```
Few people came.     (안 왔다는 데 방점 — 실망)
A few people came.   (그래도 왔다는 데 방점 — 다행)
```
같은 상황, 다른 프레이밍.

**countable/uncountable 짝:**
```
few / a few         → 가산명사(friends, people, books)
little / a little   → 불가산명사(hope, time, money)
```

**부정관사 a의 일관된 역할:** the(한정성), a/an(신정보)에 이어 a few/a little까지 — a가 붙으면 "존재를 긍정하는" 방향으로 의미가 기울어짐.
