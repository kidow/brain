# 독일어 대명사

## 인칭대명사 — 주격(Nominativ)

| 인칭 | 독일어 | 뜻 |
|------|--------|-----|
| 1인칭 단수 | ich | 나는 |
| 2인칭 단수(반말) | du | 너는 |
| 3인칭 단수 남성 | er | 그는 |
| 3인칭 단수 여성 | sie | 그녀는 |
| 3인칭 단수 중성 | es | 그것은 |
| 1인칭 복수 | wir | 우리는 |
| 2인칭 복수(반말) | ihr | 너희는 |
| 3인칭 복수 | sie | 그들은 |
| 2인칭 존댓말(단·복수 공통) | Sie | 당신은/당신들은 |

er/sie/es는 명사 성(der/die/das)과 대응:
> der Mann → er / die Frau → sie / das Kind → es

sie는 "그녀는"과 "그들은" 둘 다인데, 동사 형태로 구분:
> Sie ist müde. (그녀는, 단수) / Sie sind müde. (그들은, 복수)

핵심: **인칭대명사 주격 = ich/du/er/sie/es/wir/ihr/sie/Sie. er/sie/es는 명사 성과 대응. sie는 동사 형태로 그녀/그들 구분.**

## 인칭대명사 — 목적격(Akkusativ)

| 주격 | Akkusativ | 뜻 |
|------|-----------|-----|
| ich | mich | 나를 |
| du | dich | 너를 |
| er | ihn | 그를 |
| sie(그녀) | sie | 그녀를 |
| es | es | 그것을 |
| wir | uns | 우리를 |
| ihr | euch | 너희를 |
| sie(그들) | sie | 그들을 |
| Sie | Sie | 당신을 |

er → ihn만 완전히 다른 형태(명사의 der→den 변화에 대응). 나머지는 주격과 동일하거나 규칙적.

> Ich liebe dich. / Ich sehe ihn. (er 아니라 ihn) / Wir brauchen euch.

핵심: **Akkusativ 대명사 — mich/dich/ihn/sie/es/uns/euch/sie/Sie. er→ihn만 형태 완전히 다름.**

## 인칭대명사 — 여격(Dativ)

| 주격 | Akkusativ | Dativ | 뜻 |
|------|-----------|-------|-----|
| ich | mich | mir | 나에게 |
| du | dich | dir | 너에게 |
| er | ihn | ihm | 그에게 |
| sie(그녀) | sie | ihr | 그녀에게 |
| es | es | ihm | 그것에게 |
| wir | uns | uns | 우리에게(Akk와 동일) |
| ihr | euch | euch | 너희에게(Akk와 동일) |
| sie(그들) | sie | ihnen | 그들에게 |
| Sie | Sie | Ihnen | 당신에게 |

패턴: 1·2인칭 단수(mir/dir)는 새 형태. 3인칭 단수(ihm/ihr/ihm)는 명사 Dativ 어미(-em/-er/-em) 흔적. 복수(uns/euch)는 Akk와 동일. sie(그들)→ihnen, Sie→Ihnen은 발음 같고 대문자로만 구분.

> Er gibt mir ein Buch. / Ich helfe ihm. (helfen은 Dativ 요구 동사) / Wir danken Ihnen.

핵심: **Dativ 대명사 — mir/dir/ihm/ihr/ihm/uns/euch/ihnen/Ihnen. 복수는 Akk와 동일. sie들→ihnen, Sie→Ihnen(대문자로 구분).**
