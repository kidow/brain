# 독일어 동사

## 동사 원형의 구조

독일어 동사 원형은 거의 모두 **-en**으로 끝난다.

| 원형 | 뜻 |
|------|----|
| spielen | 놀다, 경기하다 |
| lernen | 배우다 |
| wohnen | 살다, 거주하다 |
| kaufen | 사다 |
| machen | 하다, 만들다 |

원형에서 -en을 떼면 **어간(Stamm)**:
> spielen → 어간 spiel / lernen → 어간 lern

어간에 인칭 어미를 붙여 활용한다.

핵심: **동사 원형 = 어간 + -en. 어간이 활용의 기준.**

## 현재형 인칭 어미 — 규칙동사

lernen(배우다) 어간 lern- 기준:

| 인칭 | 어미 | 활용 |
|------|------|------|
| ich | -e | lerne |
| du | -st | lernst |
| er/sie/es | -t | lernt |
| wir | -en | lernen |
| ihr | -t | lernt |
| sie/Sie | -en | lernen |

외우기: **-e / -st / -t / -en / -t / -en**

- wir·sie·Sie = 원형과 동일
- er·ihr = 둘 다 -t
- 영어(3인칭만 -s)와 달리 6인칭 전부 다름

핵심: **ich=-e / du=-st / er=-t / wir=-en / ihr=-t / sie=-en**
