# 독일어 동사

## 동사 원형의 구조

독일어 동사 원형은 거의 모두 **-en**으로 끝난다.

| 원형 | 뜻 |
|------|----|
| spielen | 놀다, 경기하다 |
| lernen | 배우다 |
| wohnen | 살다, 거주하다 |
| kaufen | 사다 |
| machen | 하다, 만들다 |

원형에서 -en을 떼면 **어간(Stamm)**:
> spielen → 어간 spiel / lernen → 어간 lern

어간에 인칭 어미를 붙여 활용한다.

핵심: **동사 원형 = 어간 + -en. 어간이 활용의 기준.**

## 현재형 인칭 어미 — 규칙동사

lernen(배우다) 어간 lern- 기준:

| 인칭 | 어미 | 활용 |
|------|------|------|
| ich | -e | lerne |
| du | -st | lernst |
| er/sie/es | -t | lernt |
| wir | -en | lernen |
| ihr | -t | lernt |
| sie/Sie | -en | lernen |

외우기: **-e / -st / -t / -en / -t / -en**

- wir·sie·Sie = 원형과 동일
- er·ihr = 둘 다 -t
- 영어(3인칭만 -s)와 달리 6인칭 전부 다름

핵심: **ich=-e / du=-st / er=-t / wir=-en / ihr=-t / sie=-en**

## sein (이다) — 불규칙 동사

| 인칭 | 활용 |
|------|------|
| ich | bin |
| du | bist |
| er/sie/es | ist |
| wir | sind |
| ihr | seid |
| sie/Sie | sind |

> Ich bin müde. / Du bist groß. / Wir sind Freunde.

핵심: **sein = bin/bist/ist/sind/seid/sind. 규칙 없음, 암기.**

## haben (가지다) — 불규칙 동사

| 인칭 | 활용 |
|------|------|
| ich | habe |
| du | hast |
| er/sie/es | hat |
| wir | haben |
| ihr | habt |
| sie/Sie | haben |

du·er에서 b 탈락(habst→hast, habt→hat).

> Ich habe einen Hund. / Er hat Hunger.(배고프다)

관용 표현 — Hunger·Durst는 haben, 나이는 sein:
> Ich habe Hunger. (배고프다) / Ich habe Durst. (목마르다)  
> Ich bin 20 Jahre alt. (나이 20살) — haben 아님

핵심: **haben = habe/hast/hat/haben/habt/haben. du·er에서 b 탈락.**

## 어간 모음 변화 (Stammvokalwechsel)

일부 강변화 동사는 du·er에서만 어간 모음이 바뀐다.

**e → i:** sprechen(du sprichst, er spricht), essen(du isst, er isst), geben(du gibst, er gibt)

**e → ie:** sehen(du siehst, er sieht), lesen(du liest, er liest)

**a → ä:** fahren(du fährst, er fährt), schlafen(du schläfst, er schläft)

> Du sprichst Deutsch. / Er fährt ein Auto. / Ich spreche Deutsch(ich는 변화 없음)

ich·wir·ihr·sie/Sie는 정상. du·er만 모음 변화 + 정상 어미(-st, -t).

핵심: **e→i, e→ie, a→ä — du·er에서만 어간 모음 변화.**

## 분리동사 (Trennbare Verben)

접두사가 평서문에서 문장 맨 끝으로 떨어져 나간다.

> aufstehen(일어나다) = auf + stehen  
> ankommen(도착하다) = an + kommen  
> einkaufen(장보다) = ein + kaufen

> Ich stehe um 7 Uhr auf. (aufstehen 분리)  
> Er kommt heute an. (ankommen 분리)  
> Wir kaufen morgen ein. (einkaufen 분리)

접두사에 강세 있으면 대부분 분리동사(AUFstehen). 원형은 붙여 씀.

핵심: **분리동사 = 접두사가 문장 끝으로 이동. 원형은 붙여 쓰지만 활용 시 떨어진다.**

## 비분리동사 (Untrennbare Verben)

접두사가 절대 떨어지지 않는 동사. 대표 접두사 8개: be-, ge-, er-, ver-, zer-, ent-, emp-, miss-

> bekommen(받다) — bekommt (안 떨어짐)  
> verstehen(이해하다) — versteht

> Ich verstehe das. (안 떨어짐) vs Ich stehe auf. (떨어짐, aufstehen)

구분: 접두사 강세 있으면 분리(AUFstehen), 없으면 비분리(verSTEHEN).

핵심: **be-, ge-, ver-, zer-, er-, ent-, emp-, miss- = 절대 안 떨어짐. 강세 없는 접두사.**

## 화법조동사 (Modalverben)

6개: können(~할 수 있다), müssen(~해야 한다), wollen(~하고 싶다/의지), sollen(~해야 한다/권고), dürfen(~해도 된다), möchten(~하고 싶다/공손)

können 활용:

| 인칭 | 활용 |
|------|------|
| ich | kann |
| du | kannst |
| er/sie/es | kann |
| wir | können |
| ihr | könnt |
| sie/Sie | können |

특이점: **ich = er** (동일 형태, 다른 동사와 다름).

어순 — 조동사 2번째, 본동사 원형이 문장 끝:
> Ich kann gut Deutsch sprechen.  
> Du musst jetzt gehen.  
> Wir wollen ins Kino gehen.

핵심: **화법조동사 6개. ich=er 동일. 본동사는 원형으로 문장 끝.**
