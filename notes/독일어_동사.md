# 독일어 동사

## 동사 원형의 구조

독일어 동사 원형은 거의 모두 **-en**으로 끝난다.

| 원형 | 뜻 |
|------|----|
| spielen | 놀다, 경기하다 |
| lernen | 배우다 |
| wohnen | 살다, 거주하다 |
| kaufen | 사다 |
| machen | 하다, 만들다 |

원형에서 -en을 떼면 **어간(Stamm)**:
> spielen → 어간 spiel / lernen → 어간 lern

어간에 인칭 어미를 붙여 활용한다.

핵심: **동사 원형 = 어간 + -en. 어간이 활용의 기준.**

## 현재형 인칭 어미 — 규칙동사

lernen(배우다) 어간 lern- 기준:

| 인칭 | 어미 | 활용 |
|------|------|------|
| ich | -e | lerne |
| du | -st | lernst |
| er/sie/es | -t | lernt |
| wir | -en | lernen |
| ihr | -t | lernt |
| sie/Sie | -en | lernen |

외우기: **-e / -st / -t / -en / -t / -en**

- wir·sie·Sie = 원형과 동일
- er·ihr = 둘 다 -t
- 영어(3인칭만 -s)와 달리 6인칭 전부 다름

핵심: **ich=-e / du=-st / er=-t / wir=-en / ihr=-t / sie=-en**

## sein (이다) — 불규칙 동사

| 인칭 | 활용 |
|------|------|
| ich | bin |
| du | bist |
| er/sie/es | ist |
| wir | sind |
| ihr | seid |
| sie/Sie | sind |

> Ich bin müde. / Du bist groß. / Wir sind Freunde.

핵심: **sein = bin/bist/ist/sind/seid/sind. 규칙 없음, 암기.**

## haben (가지다) — 불규칙 동사

| 인칭 | 활용 |
|------|------|
| ich | habe |
| du | hast |
| er/sie/es | hat |
| wir | haben |
| ihr | habt |
| sie/Sie | haben |

du·er에서 b 탈락(habst→hast, habt→hat).

> Ich habe einen Hund. / Er hat Hunger.(배고프다)

관용 표현 — Hunger·Durst는 haben, 나이는 sein:
> Ich habe Hunger. (배고프다) / Ich habe Durst. (목마르다)  
> Ich bin 20 Jahre alt. (나이 20살) — haben 아님

핵심: **haben = habe/hast/hat/haben/habt/haben. du·er에서 b 탈락.**

## 어간 모음 변화 (Stammvokalwechsel)

일부 강변화 동사는 du·er에서만 어간 모음이 바뀐다.

**e → i:** sprechen(du sprichst, er spricht), essen(du isst, er isst), geben(du gibst, er gibt)

**e → ie:** sehen(du siehst, er sieht), lesen(du liest, er liest)

**a → ä:** fahren(du fährst, er fährt), schlafen(du schläfst, er schläft)

> Du sprichst Deutsch. / Er fährt ein Auto. / Ich spreche Deutsch(ich는 변화 없음)

ich·wir·ihr·sie/Sie는 정상. du·er만 모음 변화 + 정상 어미(-st, -t).

핵심: **e→i, e→ie, a→ä — du·er에서만 어간 모음 변화.**

## 분리동사 (Trennbare Verben)

접두사가 평서문에서 문장 맨 끝으로 떨어져 나간다.

> aufstehen(일어나다) = auf + stehen  
> ankommen(도착하다) = an + kommen  
> einkaufen(장보다) = ein + kaufen

> Ich stehe um 7 Uhr auf. (aufstehen 분리)  
> Er kommt heute an. (ankommen 분리)  
> Wir kaufen morgen ein. (einkaufen 분리)

접두사에 강세 있으면 대부분 분리동사(AUFstehen). 원형은 붙여 씀.

핵심: **분리동사 = 접두사가 문장 끝으로 이동. 원형은 붙여 쓰지만 활용 시 떨어진다.**

## 비분리동사 (Untrennbare Verben)

접두사가 절대 떨어지지 않는 동사. 대표 접두사 8개: be-, ge-, er-, ver-, zer-, ent-, emp-, miss-

> bekommen(받다) — bekommt (안 떨어짐)  
> verstehen(이해하다) — versteht

> Ich verstehe das. (안 떨어짐) vs Ich stehe auf. (떨어짐, aufstehen)

구분: 접두사 강세 있으면 분리(AUFstehen), 없으면 비분리(verSTEHEN).

핵심: **be-, ge-, ver-, zer-, er-, ent-, emp-, miss- = 절대 안 떨어짐. 강세 없는 접두사.**

## 화법조동사 (Modalverben)

6개: können(~할 수 있다), müssen(~해야 한다), wollen(~하고 싶다/의지), sollen(~해야 한다/권고), dürfen(~해도 된다), möchten(~하고 싶다/공손)

können 활용:

| 인칭 | 활용 |
|------|------|
| ich | kann |
| du | kannst |
| er/sie/es | kann |
| wir | können |
| ihr | könnt |
| sie/Sie | können |

특이점: **ich = er** (동일 형태, 다른 동사와 다름).

어순 — 조동사 2번째, 본동사 원형이 문장 끝:
> Ich kann gut Deutsch sprechen.  
> Du musst jetzt gehen.  
> Wir wollen ins Kino gehen.

핵심: **화법조동사 6개. ich=er 동일. 본동사는 원형으로 문장 끝.**

## 현재완료 (Perfekt) — haben/sein + 과거분사

회화에서 과거를 말할 때 가장 많이 쓰는 시제. 한국어 "~했다"에 대응.

구조: haben/sein(2번째 위치) + ... + 과거분사(문장 끝)
> Ich habe Deutsch gelernt.  
> Er ist nach Berlin gefahren.

과거분사(규칙동사): ge- + 어간 + -t
> lernen → gelernt / spielen → gespielt / machen → gemacht

haben vs sein: 대부분 haben, **이동·상태변화 동사**(fahren, gehen, kommen, werden 등)는 sein.
> Ich habe ein Buch gekauft. (haben)  
> Ich bin nach Hause gegangen. (이동 → sein)

분리동사·화법조동사와 같은 뼈대: 조동사 2번째, 핵심 동사 문장 끝.

핵심: **Perfekt = haben/sein + 과거분사(문장 끝). 규칙동사 과거분사 = ge-어간-t. 이동 동사는 sein.**

## 강변화 동사의 과거분사 — ge-(변형 어간)-en

| 원형 | 과거분사 | 뜻 |
|------|----------|----|
| gehen | gegangen | 갔다 |
| kommen | gekommen | 왔다 |
| sehen | gesehen | 봤다 |
| sprechen | gesprochen | 말했다 |
| essen | gegessen | 먹었다 |
| trinken | getrunken | 마셨다 |
| lesen | gelesen | 읽었다 |
| schreiben | geschrieben | 썼다 |

규칙(약변화) vs 불규칙(강변화):

| | 약변화 | 강변화 |
|-|--------|--------|
| 끝 | -t | -en |
| 어간 | 그대로 | 종종 변형 |
| 예 | gelernt | gegangen |

> Ich habe ein Buch gelesen. / Er ist nach Hause gegangen.

규칙 없어 개별 암기. 원형·과거형·완료형 3단 세트로:
> gehen, ging, ist gegangen

핵심: **강변화 과거분사 = ge-(변형 어간)-en. 원형·과거·완료 3단 세트로 암기.**

## 과거분사 예외 — ge-가 안 붙는 경우

**비분리동사(be-, ver-, er-, ent- 등)** → ge- 없음:
> bekommen → bekommen / verstehen → verstanden / erklären → erklärt

> Ich habe das verstanden. (geverstanden 아님)

**-ieren 동사(외래어 계열)** → ge- 없음:
> studieren → studiert / telefonieren → telefoniert / fotografieren → fotografiert

> Ich habe in Berlin studiert.

**분리동사** → ge-가 접두사와 어간 사이에 삽입:
> aufstehen → aufgestanden / einkaufen → eingekauft

> Ich bin um 7 Uhr aufgestanden.

핵심: **비분리동사·-ieren 동사 = ge- 없음. 분리동사 = ge-가 중간에 삽입.**

## 단순과거 (Präteritum) — sein·haben·화법조동사는 이걸 선호

sein:

| 인칭 | 활용 |
|------|------|
| ich | war |
| du | warst |
| er/sie/es | war |
| wir | waren |
| ihr | wart |
| sie/Sie | waren |

haben:

| 인칭 | 활용 |
|------|------|
| ich | hatte |
| du | hattest |
| er/sie/es | hatte |
| wir | hatten |
| ihr | hattet |
| sie/Sie | hatten |

화법조동사(können): ich konnte, du konntest, er konnte...

> Ich war müde. (Ich bin gewesen müde보다 자연스러움)  
> Ich hatte einen Hund. (Ich habe gehabt보다 자연스러움)

규칙: 일반동사는 Perfekt, sein·haben·화법조동사는 Präteritum이 회화 표준.

핵심: **sein→war, haben→hatte. sein·haben·화법조동사는 Präteritum 선호.**

## 재귀동사 (Reflexive Verben) — sich + 동사

행위가 자기 자신에게 돌아오는 동사. 재귀대명사 필수 동반.

> sich freuen(기뻐하다), sich waschen(씻다), sich setzen(앉다), sich erinnern(기억하다)

재귀대명사 인칭 변화:

| 인칭 | 재귀대명사 |
|------|-----------|
| ich | mich |
| du | dich |
| er/sie/es | sich |
| wir | uns |
| ihr | euch |
| sie/Sie | sich |

> Ich freue mich. / Du freust dich. / Er freut sich. / Wir freuen uns.

사전형은 3인칭 기준 sich로 표기. 재귀대명사 없으면 문장 불완전:
> ❌ Ich freue. / ✅ Ich freue mich.

핵심: **재귀동사 = sich + 동사. 재귀대명사는 인칭마다 변화, 필수 성분.**

## 정동사 제2위 법칙 (V2-Regel)

활용된(정동사) 동사는 평서문에서 항상 **2번째 자리**.

> Ich trinke Kaffee. (Ich=1번, trinke=2번)

1번 자리엔 주어 아닌 것도 올 수 있다 — 그래도 동사는 무조건 2번, 주어는 밀림:
> Heute trinke ich Kaffee. (Heute=1번, trinke=2번, ich=3번)  
> In Berlin wohnt er. (In Berlin=1번, wohnt=2번, er=3번)

조동사·화법조동사가 있으면 본동사/과거분사/분리접두사는 문장 끝:
> Ich kann gut Deutsch sprechen. (kann=2번)  
> Ich habe ein Buch gelesen. (habe=2번)  
> Ich freue mich. (freue=2번)

핵심: **정동사 항상 2번째 자리. 1번 자리가 바뀌어도 동사는 안 움직임, 주어가 밀린다.**

## 종속절에서 동사는 문장 맨 끝 (V-Letzt)

weil(왜냐하면), dass(~라는 것), wenn(만약) 등 종속접속사절에서는 동사가 절 맨 끝으로 이동.

> Ich bleibe zu Hause, weil ich krank bin. (bin이 끝으로)  
> Ich weiß, dass er Deutsch spricht. (spricht가 끝으로)  
> Wenn es regnet, bleibe ich zu Hause. (regnet이 끝으로)

주절 vs 종속절:

| 절 종류 | 동사 위치 | 예 |
|---------|-----------|-----|
| 주절 | 2번째 | Ich bin krank. |
| 종속절 | 맨 끝 | ..., weil ich krank bin. |

조동사 있으면 조동사가 끝으로:
> Ich glaube, dass er kommen kann.

핵심: **주절 = 동사 2번째(V2) / weil·dass·wenn 등 종속절 = 동사 맨 끝(V-Letzt).**

## 명령형 (Imperativ)

du·ihr·Sie 세 상대에 따라 형태가 다름.

**du (반말):** 어간만, -e는 흔히 생략 → lernen→Lern(e)! / kommen→Komm(e)!

**ihr (친한 복수):** 현재형 어미 -t 그대로 → Lernt! / Kommt!

**Sie (존댓말):** 원형 + Sie, 도치 → Lernen Sie! / Kommen Sie!

**sein 불규칙:** Sei still! / Seid still! / Seien Sie still!

**어간모음 e→i/ie 동사는 du형에도 반영(e 생략 안 됨):**
> sprechen → Sprich langsam!  
> lesen → Lies das Buch!

> Komm her! / Öffnen Sie die Tür! / Seid leise!

핵심: **du=어간(+e 생략 가능) / ihr=어미 -t / Sie=원형+Sie 도치. sein 불규칙. e→i 동사는 du형에 반영.**

## 미래형 (Futur I) — werden + 원형

werden 활용:

| 인칭 | 활용 |
|------|------|
| ich | werde |
| du | wirst |
| er/sie/es | wird |
| wir | werden |
| ihr | werdet |
| sie/Sie | werden |

구조: werden(2번째) + ... + 원형(문장 끝)
> Ich werde morgen kommen.  
> Er wird das Buch lesen.

실제로는 현재형+시간부사로도 미래 표현:
> Ich komme morgen. (Futur I 없이도 미래 의미)

Futur I는 추측·확신·강한 의지 강조에 사용:
> Es wird regnen. (추측) / Ich werde das nie vergessen. (강한 의지)

핵심: **Futur I = werden(2번째) + 원형(끝). 일상은 현재형+시간부사가 더 흔함. werden은 추측·의지 강조.**

## 수동태 (Passiv) — werden + 과거분사

능동 → 수동: 능동문 목적어(Akkusativ)가 수동문 주어(Nominativ)로 전환.
> 능동: Der Lehrer korrigiert den Test.  
> 수동: Der Test wird vom Lehrer korrigiert.

행위자: von + Dativ (vom = von dem 축약)

시제별 werden:

| 시제 | 구조 | 예 |
|------|------|-----|
| 현재 | wird + p.p. | Der Test wird korrigiert. |
| 과거 | wurde + p.p. | Der Test wurde korrigiert. |
| 현재완료 | ist + p.p. + worden | Der Test ist korrigiert worden. |

현재완료 수동태의 werden 과거분사는 geworden이 아니라 **worden** (예외).

행위자 불명확해도 되는 경우 자주 사용:
> Das Museum wird um 18 Uhr geschlossen.

핵심: **수동태 = werden + 과거분사. 목적어→주어 전환. 행위자는 von+Dativ. 현재완료 수동은 worden.**
