스페인어 특수 부호 & 키보드 입력

특수 문자 용도 + Mac / Windows / 모바일 입력법

특수 문자 & 역할
문자이름역할예시
ñ 에녜 (eñe)n + 틸데(~) ny 소리. n과 완전히 다른 글자. España = 에스파냐
año(해) ≠ ano(항문)
á é í ó ú 악센트 모음acento (´) ①강세 위치 ②동음이의어 구분 ③히아토 표시 sí(네) vs si(만약)
habló(말했다) vs hablo(말한다)
ü 디에레시스diéresis (¨) gue/gui에서 u를 발음함을 표시 güisqui = 귀스끼
cigüeña = 시귀에냐
¿ 거꾸로 물음표 의문문 시작에 위치 ¿Cómo estás?
¡ 거꾸로 느낌표 감탄문 시작에 위치 ¡Hola! = 안녕!
« » 길예메 (guillemets) 전통 인용부호. 쉼표·마침표는 따옴표 밖. «No tengo tiempo»,
Mac 키보드 단축키
문자입력법
á é í ó ú⌥ Option + e → 모음 입력
ñ⌥ Option + n → n
ü⌥ Option + u → u
¿⌥ Option + ⇧ Shift + ?
¡⌥ Option + 1
«⌥ Option + \
»⌥ Option + ⇧ Shift + \
Windows & 모바일
방법설명
Win키보드 레이아웃 설정 → 언어 → 스페인어 키보드 추가. Win + Space로 전환.
WinAlt 코드 숫자패드 필요: á=Alt+0225, é=Alt+0233, í=Alt+0237, ó=Alt+0243, ú=Alt+0250, ñ=Alt+0241, ü=Alt+0252
모바일길게 누르기 모음(a/e/i/o/u/n) 길게 누르면 á·à·â·ñ 등 선택 팝업 표시
가장 실용적인 방법
• 맥: Option 조합 익히면 자연스러움
• 스페인어 많이 쓴다면: 시스템 언어에 스페인어 키보드 추가 (Mac/Win 공통)
• 모바일: 길게 누르기가 가장 빠름
