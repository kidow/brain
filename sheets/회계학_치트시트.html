회계등식

자산 = 부채 + 자본. 이익 = 수익 - 비용. 자산 늘면 부채 or 자본 같은 만큼 증가 — 절대 안 깨짐.

복식부기 — 차변/대변

차변(왼쪽)대변(오른쪽)
자산증가↑감소↓
부채감소↓증가↑
자본감소↓증가↑

재무제표 3종

표의미
재무상태표특정 날짜 스냅샷 — 자산=부채+자본
손익계산서기간 흐름 — 수익-비용=이익
현금흐름표실제 통장 돈 흐름(영업/투자/재무활동)
건강한 패턴: 영업(+) / 투자(-) / 재무(±). 위험 패턴: 영업(-)+재무(+) = 본업 부진을 빚으로 때움.

이익의 단계

매출액-매출원가=매출총이익(장사 자체) → -판관비=영업이익(본업) → +영업외손익-법인세=당기순이익(최종)

발생주의 vs 현금주의 · 감가상각

회계기준은 발생주의 강제(거래 발생 시점에 기록, 현금수수 무관). 감가상각비는 현금 안 나가는 비용 — 손익계산서 이익 < 실제 현금흐름의 원인.

재무비율

비율공식의미
유동비율유동자산/유동부채단기 상환 능력 (200%↑ 안전)
부채비율부채/자본×100레버리지, 낮을수록 안정
영업이익률영업이익/매출액×100본업 수익성
ROE당기순이익/자본×100주인 돈 대비 수익

EPS · PER · EBITDA

EPS = 당기순이익/발행주식수. PER = 주가/EPS(업종별 비교 필수).
EBITDA = 영업이익+감가상각비+무형자산상각비 — 감가상각 정책 영향 제거해 실제 현금창출력 비교(K-IFRS 공식 항목 아님, 근거 확인 필요).

감사의견 4단계

의견의미
적정정상 — 대부분 상장사
한정경고 — 특정 항목만 확인 불가
부적정심각 — 회계기준 위반
의견거절최악 — 상장폐지 사유 1순위

손익분기점(BEP)

BEP 매출액 = 고정비 / (1 - 변동비율). 예: 고정비 300만, 변동비율 40% → BEP 500만원.

기타 핵심 개념

영업권(Goodwill) = 인수가 - 순자산 장부가치 (매년 손상검토만, 정액상각 없음)
연결재무제표 = 지배·종속 법인을 하나로 합산(내부거래 제거)
자기주식 = 자본의 차감 항목(자산 아님)
이연법인세 = 회계이익과 세법상 과세소득 차이를 담는 항목
