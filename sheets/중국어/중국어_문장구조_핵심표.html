중국어 문장구조 핵심표

어순·부정문·상 표지(了着过)·강조구문·把被자문 — 문장 뼈대 한눈에

기본 어순

주어 + 동사 + 목적어 (SVO)
我吃苹果。 Wǒ chī píngguǒ. 나는 사과를 먹는다.
조사 없이 어순 자체가 문법 — 순서 바뀌면 의미도 바뀌거나 비문이 됨.

부정문

부정사쓰임예
不현재/미래, 습관, 의지, 상태我不去。(안 간다)
没(有)과거 경험, 완료 여부, 有 부정我没去。(안 갔다)

了가 있는 문장을 부정하면 了는 사라지고 没(有)만 남음: 我吃了 → 我没吃

상(aspect) 표지 3종

표지의미예
了완료 또는 상태 변화我吃了。(먹었다) / 天冷了。(추워졌다)
着동작·상태의 지속他坐着。(앉아 있다)
过경험(~해본 적 있다)我去过北京。(가본 적 있다)

了=완료 시점 강조 / 着=완료 후 상태 지속 / 过=경험 유무. 셋 다 "과거"가 아닌 상 범주.

是…的 강조구문

是 + [강조할 정보] + 동사구 + 的
我是去年去北京的。 작년에 베이징에 갔다(작년 강조).
이미 일어난 일의 시간·장소·방법·주체를 짚어줄 때만 씀. 미래엔 못 씀.

把자문 vs 被자문

구문구조예
把주어 + 把 + 목적어 + 동사 + 기타성분我把苹果吃了。(사과를 처리함, 능동 관점)
被목적어 + 被 + [행위자] + 동사 + 기타성분苹果被我吃了。(사과가 당함, 피행위자 관점)

把/被 둘 다 동사만 덜렁 못 씀 — 반드시 뒤에 결과 성분(了/보어 등)이 필요.

정반의문문 3계열

문장 종류형태
일반 동사/형용사V不V / A不A — 去不去？高不高？
조동사조동사不조동사 — 会不会？能不能？
了 있는 완료문V了没(有) — 吃了没有？
