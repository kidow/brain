4성 (표준 중국어)

성조부호설명예
1성ā ¯ 평평높고 길게 유지mā 妈(엄마)
2성á ╱ 올라감낮→높, 의문 어조má 麻(삼)
3성ǎ ∨ 내려갔다 올라옴낮게 내렸다 올림mǎ 马(말)
4성à ╲ 내려감높→낮, 단호mà 骂(꾸짖다)
경성a (부호 없음)빠르고 가볍게ma 吗(의문)
같은 발음 + 성조만 다르면 완전히 다른 단어. 한국어엔 없는 개념 — 프랑스어 악센트(모음 질 변경)와 달리 성조는 피치 곡선 변경.

병음(拼音) 구조

성모(초성) + 운모(중·종성) + 성조부호. 예: b + ei + ˇ → běi(북녘), j + īng + ¯ → jīng(수도). 병음은 한자를 대체하지 않는 발음 도우미(1958년 제정).

중국 대륙·싱가포르=간체자 / 대만·홍콩·마카오=번체자. 한국 한자는 번체자 기반. 발음은 동일, 모양만 다름.

총 214개 부수 기준. 병음 모를 때 부수로 한자 검색 가능.

한 글자 ≠ 한 단어 — 조합 원리

电(전기)+话(말)→电话(전화) / 电+脑(뇌)→电脑(컴퓨터) / 手(손)+机(기계)→手机(휴대폰). 한국 한자어와 완전히 같은 원리(電話·電腦·手機) — 글자 뜻 알면 단어 유추 가능.

한국 한자어 직독 활용

학교(學校)=学校 xuéxiào / 전화(電話)=电话 diànhuà / 문화(文化)=文化 wénhuà. 한국어 어휘 60~70%가 한자어 → 뜻은 이미 알고 발음(병음)+성조만 새로 배우면 됨.
