구조 개요

부품별 요약

CPU
Central Processing Unit — 연산·추론
역할명령 실행, 논리 연산
단위코어 수, GHz (클럭)
특징소수의 강력한 코어
인간으로 치면 뇌. 복잡한 작업을 순서대로 처리.
RAM
Random Access Memory — 작업 메모리
역할실행 중 데이터 임시 저장
특징휘발성 (전원 끄면 소멸)
규격DDR4 / DDR5, GB, CL
책상 위의 작업 공간. 넓을수록 동시 작업 가능.
GPU
Graphics Processing Unit
역할그래픽 렌더링, 병렬 연산
코어수천~수만 개 (단순·소형)
연결PCIe 슬롯
단순 작업을 수천 개 동시에. CPU와 구조 자체가 다름.
메인보드
Motherboard — 연결 기판
역할모든 부품 연결·중계
슬롯PCIe, M.2, RAM, CPU 소켓
규격ATX / mATX / Mini-ITX
척수·신경계. 칩셋이 납땜 내장됨.
NVMe SSD
Non-Volatile Memory Express
역할데이터 영구 저장
특징비휘발성, 3,000~7,000 MB/s
연결M.2 슬롯 (PCIe 직결)
SATA SSD보다 6~14배 빠름.
PSU
Power Supply Unit — 파워서플라이
역할220V AC → DC 변환·공급
단위W (와트)
용량일반 400W / 게이밍 650~850W
변압기 달린 멀티탭. 부족하면 부팅 불가.

클럭 & 오버클럭

클럭 (Clock)
CPU 동작 박자 신호
수십억 트랜지스터 동기화. 단위 GHz.
3.5GHz = 1초 35억 번 박자.
오케스트라 지휘자 역할.
오버클럭
기본 클럭 초과 설정
공식 권장 기능 아님. Z/X 칩셋만 허용.
장점: 성능 향상. 단점: 발열·불안정.
일반 사용자는 부스트 클럭으로 충분.
