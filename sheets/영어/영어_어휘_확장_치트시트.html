영어 어휘 확장 치트시트

Vocabulary Building — 어근·접두사·접미사 / 콜로케이션 / 뉘앙스

어근 (Root)

어근의미예시
port운반하다transport, export, portable
dict말하다predict, contradict, verdict
script/scrib쓰다describe, prescription, subscribe
rupt끊다·깨다interrupt, erupt, corrupt
vis/vid보다vision, visible, evidence
duc/duct이끌다produce, educate, conduct

접두사 (Prefix)

접두사의미
un- / in-/im-/ir-/il- / dis- / non- / anti-부정·반대
pre-이전
post-이후
over- / under-초과 / 부족
re-다시
"unprecedented" = un(부정) + precedent(선례) + ed → 전례 없는

접미사 (Suffix) — 품사 변환

변환접미사예
동사→명사-tion/-sion / -ment / -er/-or / -ance/-encedecide→decision
형용사→명사-ness / -ityhappy→happiness
명사/동사→형용사-ful/-less / -able/-ible / -ous / -alhelp→helpful
형용사→부사-lyquick→quickly

콜로케이션 (Collocation)

make a decisiondo a decision
do homeworkmake homework
take a photomake a photo
heavy rainstrong rain

단어 하나가 아닌 짝으로 암기 — 직역은 콜로케이션을 모름

다의어 vs 동음이의어

다의어하나의 단어, 파생 의미 — "bank"(은행/강둑/기울다)
동음이의어형태 같지만 기원 다름 — "bat"(방망이/박쥐)

어휘 확장 학습 전략

어휘 묶음 학습nation→national→nationality→internationally — 어근 중심 파생 묶음
입력 기반 확장모르는 단어 → 문맥 추론 → 확인
맥락 결합단어 카드 대신 예문으로 암기
간격 반복1일 → 3일 → 1주 → 1개월
