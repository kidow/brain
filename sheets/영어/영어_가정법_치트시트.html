영어 가정법 치트시트

English Conditionals — 구조·용법·비교 한눈에

4가지 유형 핵심 비교

유형 If절 결과절 현실성 예
0차 현재형 현재형 항상 참인 사실 If you heat it, it boils.
1차 현재형 will + 원형 가능한 미래 If it rains, I will stay.
2차 과거형 would + 원형 현재 비현실 If I had time, I would help.
3차 had + p.p. would have + p.p. 과거 비현실 If I had studied, I would have passed.
혼합 had + p.p. would + 원형 과거 조건 → 현재 결과 If I had studied medicine, I would be a doctor now.
⚠️ If절 안에서 미래도 현재형 — If it will rain → If it rains
⚠️ 2차 가정법 be 동사 → 주어 무관 were 고정 — If I was you → If I were you (격식)

if vs when

if when
조건이 일어나지 않을 수도 있음 조건이 반드시 일어남 (기정사실)
If it rains... (비가 안 올 수도) When I retire... (반드시 은퇴할 것)

were 촉발 단어 3가지

if (2차 가정법)
If I were you, I would apologize.
wish
I wish I were taller.
as if / as though
He acts as if he were the boss.
공통 원리
현재 비현실 상황 → were

wish / if only 패턴

시점 구조 예
현재 비현실 wish + 과거형 I wish I were taller.
과거 비현실 wish + had + p.p. I wish I had studied harder.
미래 변화 소망 wish + would + 원형 I wish you would stop smoking.
if only = wish의 강조형 — 구조 동일, 아쉬움·절박함이 더 강함

would rather

대상 구조 예
자신의 선호 would rather + 원형 I'd rather stay home.
타인의 행동 would rather + 주어 + 과거형 I'd rather you came tomorrow.

It's time (that)

구조: It's time + 주어 + 과거형  /  It's high time + 주어 + 과거형 (더 강한 촉구)
It's time you went to bed.        → 이제 잘 시간이야
It's high time he apologized.     → 진작에 사과했어야 해
It's time to sleep. (일반) vs It's time you slept. (특정 주어 → 반드시 과거형)

가정법 도치 (Inversion) — 격식·문어체

원래 형태 도치 형태
If I were you... Were I you...
If I had known... Had I known...
If you should need help... Should you need help...
실전: 비즈니스 이메일 — Should you have any questions, please contact us.

조건 접속사 비교

접속사 뜻 특이사항
unless ~하지 않으면 (= if not) 가정법에선 if not 사용. unless + not = 이중 부정 ❌
as long as ~하는 한 조건이 한정·제한적인 느낌
provided (that) ~한다면 계약·조건부 허락에 자주 등장
even if ~일지라도 (가상·불확실) 조건과 상관없이 결과가 변하지 않음
even though ~임에도 불구하고 (사실) 조건이 이미 알려진 사실
