English Adverbs — 수식·위치·종류 한눈에

부사의 수식 대상 4가지

수식 대상예시비고
동사She runs quickly.
형용사He is very tall.
다른 부사She runs incredibly quickly.
문장 전체Fortunately, no one was hurt.문장 부사
⚠️ 명사 수식 = 형용사 역할 → 부사로 명사 수식 시 문법 오류

-ly 규칙과 예외

-ly 붙어도 형용사인 것
friendly / lovely / lonely / likely
She smiled friendly → in a friendly way
형용사 = 부사 (동형)
fast / hard / late / early
He runs fast. (부사) / a fast car (형용사)
-ly 붙이면 의미 변화 ⚠️
hard (열심히) vs hardly (거의 ~않다)
late (늦게) vs lately (최근에)

위치가 의미를 결정한다 — only 예시

문장뜻
Only I told her the truth.나만 말했다
I only told her the truth.말만 했다
I told only her the truth.그녀에게만 말했다
I told her only the truth.진실만 말했다
부사는 수식 대상 바로 앞(또는 뒤)에 위치 → 위치가 바뀌면 의미 변화

부사 3가지 위치 (Front / Mid / End)

[Front] + 주어 + [Mid] + 동사 + 목적어 + [End]
위치수식 대상예시
Front문장 전체·맥락 설정Suddenly, the door opened.
Mid빈도·확신 부사She always arrives early.
End방법·장소·시간She sings beautifully.
End에 여러 부사: 방법 → 장소 → 시간 순서
She sang beautifully  at the concert  last night.

문장 부사 (Sentence Adverbs)

항상 Front + 쉼표. 사실이 아닌 화자의 태도를 표현.
대표 단어
Fortunately / Unfortunately / Honestly / Clearly / Obviously / Apparently / Surprisingly
문장역할
He spoke honestly.방법 부사 (말하는 방식)
Honestly, he spoke too much.문장 부사 (화자의 평가)

비교급·최상급

짧은 부사 (1음절): -er / -est
fast → faster → fastest
hard → harder → hardest
-ly 부사 (2음절+): more / most
quickly → more quickly → most quickly
원급비교급최상급

연결 동사 뒤엔 형용사 (부사 아님)

연결 동사: be / look / feel / smell / taste / sound / seem / appear / become / get
올바른 표현틀린 표현
She looks beautiful. ✅She looks beautifully.
The soup smells good. ✅The soup smells well.
He felt bad about it. ✅He felt badly about it.
체크법: 동사를 be로 바꿔봄 → 자연스러우면 연결 동사 → 형용사 사용
