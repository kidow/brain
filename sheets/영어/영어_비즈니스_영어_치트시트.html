회의에서 의견 말하기

제시I'd like to suggest that... / From my perspective, ...
부드러운 반대That's a valid point, but... / I'd push back on that a little.
동의I'm on board with that.

회의 진행

Let's go over the agenda. / Can we take this offline?
We're running low on time. / Let's table this for now.
Action items: [이름] will [동작] by [날짜].

협상 표현

제안We'd be willing to... / What if we...?
조건We can do X, provided that...
양보·타협Let's meet halfway.
거절I'm afraid that's not something we can agree to.
"No" 직접 사용 지양 → "That's difficult, but what if..."

프레젠테이션 흐름

전환Moving on to... / This brings me to my next point.
강조The key takeaway here is...
마무리To wrap up, ... / The bottom line is...

피드백 — SBI 모델

Situation + Behavior + Impact — "In yesterday's meeting(S), when you interrupted(B), it made it harder to finish(I)"
개선One thing that might help is... / Have you considered...?
받을 때Thanks for the feedback. / Could you give me an example?

문화 차이

직접성"I recommend we..." (명확) > "I think maybe possibly..." (자신감 없음)
침묵한국=존중 / 영어권=관심 없음 오해 → "That makes sense" 반응 필요
칭찬 수용아니에요, 별거 아니에요 → Thank you, I'm glad it worked out.

영어 면접 표현

자기소개현재(I'm currently...) → 과거(Before that, I...) → 미래(I'm looking to...)
강점강점 + 구체적 상황 + 결과
약점진짜 약점 + 개선 노력
역질문What does success look like in this role?
"No questions" → 관심 없어 보임. 최소 2개 준비.
