영어 동명사 치트시트

English Gerund — 동명사 vs to부정사 / 의미상 주어 / 관용표현

동명사 vs 현재분사

형태역할예문
동명사명사 (주어 자리)Swimming is fun.
현재분사형용사 (명사 수식)The swimming boy is fast.
현재분사동사 (진행형)He is swimming.

의미상 주어

주절 주어와 같으면 생략, 다르면 소유격/목적격 표시
격식: I don't mind your being late.
구어: I don't mind you being late.

동명사 vs to부정사 — 동사 목적어

동명사만to부정사만둘 다 (의미 동일)
enjoy / finish / avoid / mindwant / hope / decide / planlike / love / hate
consider / suggest / admit / denypromise / refuse / expect / needbegin / start / continue
quit / postpone / practice / give upagree / manage / fail / afford

패턴: 동명사 = 이미 일어난 일·회피 / to부정사 = 미래·의지·계획

전치사 + 동명사

전치사 뒤엔 반드시 동명사 — good at to swim → good at swimming
look forward to -ingI look forward to seeing you.
be used to -ingI'm used to waking up early. (used to + 원형 = 과거습관과 다름)
object to -ingHe objected to paying more.

동명사의 시제·태·부정

형태예문
완료 동명사She admitted having stolen it.
수동 동명사I hate being ignored.
완료 수동He denied having been told.
부정(not 동명사 앞)I regret not telling her.
