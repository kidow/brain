수량 한정사 — 가산 vs 불가산

가산 복수불가산
많다manymuch
적다 (부정적)fewlittle
조금 있다 (긍정적)a fewa little
비격식 긍정a lot ofa lot of
있다·없다some / anysome / any
I have few money. → I have little money. (money = 불가산) A few people came. (몇 명 왔다 — 긍정) Few people came. (거의 안 왔다 — 부정)

much / many — 격식·비격식

much / many는 부정·의문문에서 자연스러움. 긍정문엔 a lot of 선호.
상황자연스러운 표현예
긍정 (비격식)a lot ofI have a lot of work.
긍정 (격식·문어)many / muchMany studies suggest…
부정many / muchI don't have much time.
의문many / muchDo you have many friends?

some / any

상황사용예
긍정문someI have some milk.
부정문anyI don't have any milk.
의문 (일반)anyDo you have any questions?
의문 (권유·부탁)someWould you like some coffee?
"Do you have some money?" → 있겠지, 얼마 줄 수 있어? (전제 있음) "Do you have any money?" → 혹시 있어? (없을 수도 있다고 생각)

no / none / not any

구조예
nono + 명사I have no time. / There are no chairs.
not anynot + any + 명사I don't have any time.
none단독 (명사 대체)None of them came.
no = not any (의미 동일, no가 더 강조·간결)  |  none은 대명사로 단독 사용
none of + 복수명사 → 동사 단수(격식) / 복수(일반) 둘 다 허용

형태가 헷갈리는 명사

복수형 → 복수 동사
(a pair of로 셈)
scissors (가위) trousers (바지) glasses (안경) tongs (집게)

The scissors are on the table.

a pair of scissors = 하나

-s로 끝나지만 단수 동사
news (뉴스) mathematics (수학) physics (물리학) economics (경제학) the United States

The news is good.

항상 복수 동사
people (사람들) police (경찰) cattle (소 떼)

The police are looking for him.
