영어 청취 치트시트

Listening Strategy — 두 층위 / 연음 / 예측 / 청킹 / 억양

청취의 두 층위

층위 1 — 음성 인식소리를 단어로 분해. 연음·축약·flap T가 여기 직결
층위 2 — 실시간 처리인식한 소리의 의미를 즉시 처리. 원어민 150~160wpm vs 번역 이해 60~80wpm
두 층위 모두 훈련해야 함 — 하나만 훈련하면 다른 쪽에서 막힘

연음(Linking) 규칙

자음+모음an apple → a-na-pple
같은 자음bad day → ba-day
T 변형want to → wanna / got to → gotta
기능어 소실What do you want? → Whaddaya want?

예측 청취(Predictive Listening)

문법·주제로 다음 단어의 종류를 미리 좁혀서 들음 — "I'm going to the..." → 다음은 장소(명사)

실전: 재생 전 제목·썸네일로 주제 예측 → 예상 어휘 미리 떠올리기

청킹(Chunking) 청취

단어 단위(7번 처리) 대신 의미 덩어리(4번 처리)로 듣기 — 원어민 발화 리듬 자체가 청크 단위

훈련: 문장을 청크 단위로 끊어서 Shadowing

억양(Accent) 차이

영국식R 발음 약함 — car = "카-"
호주식모음이 확 바뀜 — today = "터다이"
인도식강세 패턴, T/D 발음이 다름
전략은 동일 적용, "이 억양은 이 소리를 이렇게 낸다"는 매핑만 추가

학습 루틴

1단계이해도 70%+ 콘텐츠로 시작
2단계자막 켜고 1회 시청
3단계자막 끄고 재시청
4단계안 들린 구간만 반복 + 스크립트 확인

Active Listening: 받아쓰기(Dictation) + 쉐도잉(Shadowing). 콘텐츠 순서: 자막 유튜브 → 스크립트 팟캐스트 → 영화·드라마.

진행 신호: 이해도보다 "몰라도 불안하지 않은" 상태가 먼저 옴
