프랑스어 문자 치트시트

영어 26자 + 악센트 5종 · 표기 ≠ 발음 · 끝 자음은 기본 묵음

악센트(Accent) 5종

악센트 = 별도 글자 아님. 기존 글자 위에 얹는 부호. 알파벳 수(26개)에 포함 안 됨.
기호이름예발음 변화
éaccent aigucafé, étudiant닫힌 "에" — 입술 긴장
èaccent gravetrès, mère열린 "에" — 입 더 벌림
êaccent circonflexefête, être열린 "에" (è와 거의 동일)
â ôaccent circonflexeâge, côte현대 파리 발음 = a/o와 거의 동일
î ûaccent circonflexeîle, sûr발음 변화 없음 — 역사·철자 표시
ë ïtrémaNoël, naïf앞 모음과 분리해서 읽어라
çcédillegarçon, françaisc를 항상 "ㅅ" 소리로 (없으면 "ㄲ")
먼저 익힐 것: é (가장 빈번) → ç (다음) → 나머지

끝 자음 묵음 규칙

기본값: 끝 자음 = 묵음
Paris/파리/ (s 묵음)
chat/샤/ (t 묵음)
grand/그헝/ (d 묵음)
beaucoup/보쿠/ (p 묵음)
예외 — 발음하는 끝 자음: C · a · R · e · F · u · L → CaReFuL
Cavec → /아베크/
Rmer → /메흐/
Fchef → /셰프/
Lbal → /바ㄹ/

연음(Liaison)

끝 자음 묵음이 기본이지만, 다음 단어가 모음으로 시작하면 죽었던 자음이 살아남.
종류예규칙
필수 연음les amis → /레z아미/관사+명사, 대명사+동사
선택 연음격식체에서만문어체·공식 발표
금지 연음고유명사 앞 등절대 불가
s, x 끝 자음 → 연음 시 /z/ 소리
vous avez → /부z아베/ / un enfant → /앙n엉팡/

비모음(Voyelles Nasales)

모음 + n/m 뒤에 자음이 오거나 단어 끝일 때 → 코로 공기 흘리는 비모음
핵심: 모음+n/m 뒤에 모음이 오면 → 비모음 아님 (bonne /본/ vs bon /봉/)
표기발음예
an, am, en, em/ɑ̃/ "앙"enfant /엉팡/, temps /텅/
in, im, ain, ein/ɛ̃/ "앵"vin /뱅/, pain /팡/
on, om/ɔ̃/ "옹"bon /봉/, nom /농/
un, um/œ̃/ "윙"→"앵"un /앵/ (현대 파리: /ɛ̃/와 합류 중)

조합소리예

ph/f/ "프"photo /포토/
gn/ɲ/ "뉴"계열champagne /샹파뉴/, montagne /몽타뉴/

R 발음 — 목젖 마찰음

영어 r = 혀를 뒤로 말음 ↔ 프랑스어 r = 혀 고정, 목젖(uvula) 진동
만드는 법: ① 혀 평평하게 아래 고정 ② "ㅎ"을 목 뒤에서 마찰 ③ 가래 뱉기 직전 소리
예: rue /흐위/, Paris /파히/, merci /메흐시/

강세 (Accent Tonique)

규칙: 항상 마지막 음절. 예외 없음.
café → ca-fé / université → u-ni-ver-si-té
단, 묵음 e는 음절 안 셈: table = ta-ble → 실질 마지막 = ta → /타블/

대소문자 — 영어보다 덜 씀

소문자 (영어는 대문자)
요일: lundi (Monday)
월: janvier (January)
언어: français (French)
국적 형용사: vin français
대문자 유지

제목: 첫 단어만 대문자
