명사 복수형 규칙

단수 끝복수 변화예
일반+ sle livre → les livres
-s / -x / -z변화 없음le bras → les bras
-eau / -au+ xle gâteau → les gâteaux
-al→ -auxle cheval → les chevaux
-eu+ xle jeu → les jeux
불규칙→ yeuxl'œil → les yeux

💡 복수 -s는 묵음. 관사(les/des)가 복수의 진짜 신호. 연음 시 /z/ 소리: les enfants /레z엉팡/
