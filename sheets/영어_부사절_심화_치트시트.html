영어 부사절 심화 치트시트

Advanced Adverb Clauses — 시간·조건·양보·이유·목적·결과·방식

as의 다중 용법

용법예문
시간As the sun set, the temperature dropped.
이유As it was raining, we stayed inside.
비례As you get older, time passes faster.
방식Do it as I showed you.

"~할수록"으로 번역되면 비례, 아니면 시간

시간 부사절

접속사초점
when단순 시점
while진행 중인 배경
as찰나의 동시
once전환점 이후
by the time기준 시점 이전 완료 (결과절에 완료시제)

접속사뉘앙스

유형접속사

even though = 사실 인정 / even if = 가상·불확실

목적 부사절

so that + 조동사구어·격식
in order that격식·문어체
lest + should + 원형~하지 않도록 (부정 목적, 드묾)

결과 부사절 — so...that / such...that

so + 형용사/부사 + thatShe was so tired that she fell asleep.
such + (a/an) + 형용사 + 명사 + thatIt was such a good movie that we watched it twice.

방식 부사절 — as / as if / as though

의미예
as실제 방식(직설법)Do it as I showed you.
as if / as though비현실(가정법)He talks as if he knew everything.

부사절 위치 — 앞 vs 뒤

뒤(기본)She passed because she studied. — 주절이 핵심
앞(강조)Because she studied, she passed. — 이유 먼저
앞에 올 때 쉼표 필수. 글쓰기=앞 배치 격식적, 구어=뒤가 자연스러움
