이중모음

철자발음예시
ei 아이 [aɪ] Wein(포도주) = "바인", mein = "마인"
ie 이(장) [iː] Bier(맥주) = "비어", Liebe = "리베"

발음 핵심 규칙

장모음 vs 단모음

  • 모음 + 자음 1개 → 장모음 (Weg)
  • 모음 + 자음 2개+ → 단모음 (Bett)
  • 모음 + h (묵음) → 장모음 (Bahn = "반")
  • 같은 모음 2개 → 장모음 (See = "제")

위치별 끝소리 규칙

  • -b/d/g 끝 → p/t/k로 경음화 (Dortmund = "도르트문트")
  • -ig 끝 → [ɪç] -히 (Leipzig = "라이프치히")
  • -er 끝 → [ɐ] 약한 어/아 (Müller = "뮐러")
  • -e 끝 (비강세) → [ə] 약한 으/어 (Schule = "슐레")
  • -tion → "치온" (Nation = "나치온")

r 발음

  • 모음 앞: 목젖 마찰 [ʁ] — 혀 말기 금지
  • 단어 끝 / -er: 약한 어/아 [ɐ]
  • 연습: ch([x]) 내는 위치에서 약하게 흘리기
  • Reus = "로이스", Ribéry = "리베리"

강세

  • 기본값: 첫 음절 (München, Dortmund)
  • be- ge- ver- er- ent- 접두사 → 다음 음절으로 점프
  • verstehen = ver-STE-hen
  • 지명·외래어는 원어 강세 따름 (Leverkusen)

v 예외 정리

  • 기본값 v = [f] (Vater, Völler)
  • 이름·외래어에서 v = [v] 예외
  • Leverkusen, Havertz, Hannover → [v]
  • Kevin의 v = [v] / Volland의 v = [f]

묵음 h

  • 모음 뒤 h → 발음 없음, 앞 모음 장모음화
  • ah → "아", eh → "에", oh → "오", uh → "우"
  • Kahn = "칸", Lahm = "람", Uhr = "우어"
  • 모음 앞 h → 약하게 발음 (Haus = "하우스")
