특수 규칙

촉음 っ — 자음 1박 연장

작은 っ → 소리 없이 막히는 1박.
きて (ki-te) = 와줘
きって (kit-te) = 우표
いった (it-ta) = 갔다

장음 — 히라가나 모음 연장

あ단 + あ → おかあさん (o·kā·san)
い단 + い → おにいさん (o·nī·san)
う단 + う → くうき (kū·ki)
え단 + い → せんせい (sen·sē)
お단 + う → とうきょう (Tō·kyō)

발음 주의 글자

し = shi (si ✗)
ち = chi (ti ✗)
つ = tsu (쯔~츠 사이)
ふ = fu (후, 입술 느슨하게)
う = u (입술 덜 내밈)
ら행 = r (혀끝 한 번 튕기기)
