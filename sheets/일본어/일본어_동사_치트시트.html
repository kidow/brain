일본어 동사 치트시트

3그룹 분류 + 활용 8형태(ます·て·ない·가능·의지·사역·수동·たい)

동사 3그룹 판별

그룹규칙예
1그룹(u동사)る 이외 う단 / る로 끝나도 앞이 a·u·o書く, 飲む, 話す, 乗る(o)
2그룹(ru동사)る로 끝나고 앞이 i·e食べる(e), 見る(i), 起きる
3그룹(불규칙)딱 2개, 통암기する, 来る(くる)
판별 순서: ①る로 안 끝남→1그룹 ②る로 끝남→앞 소리(i·e→2 / a·u·o→1) ③する·くる→3그룹

ます形 4형태 (정중체)

긍정부정
현재/미래~ます~ません
과거~ました~ませんでした
1그룹: う단→い단+ます (書く→書きます) / 2그룹: る떼고+ます (食べる→食べます) / 3그룹: します・来ます(きます)

て形 — 1그룹 음편 규칙

어미변화예
う・つ・る→って買う→買って, 待つ→待って, 乗る→乗って
む・ぶ・ぬ→んで飲む→飲んで, 遊ぶ→遊んで, 死ぬ→死んで
く→いて書く→書いて
ぐ→いで泳ぐ→泳いで
す→して話す→話して
2그룹: る떼고+て(食べて) / 3그룹: して・来て(きて) / 예외: 行く→行って(く규칙 안 따름)

ない形 (반말 부정)

그룹ます형ない형
1그룹い단+ますあ단+ない (書かない, 예외: 買う→買わない)
2그룹る떼고+まする떼고+ない
3그룹します/来ますしない/来ない(こない)

주요 문형

~ている: 진행(今、勉強しています) / 상태지속(結婚しています)
~てください: 요청 (待ってください) / 부정요청은 ない형+ないでください

う단 모음 이동 총정리 (예: 飲む)

형태방향결과뜻
ます형い단飲みます마십니다
ない형あ단飲まない안 마신다
가능형え단+る飲める마실 수 있다
의지형お단+う飲もう마시자
사역형あ단+せる飲ませる마시게 하다
수동형あ단+れる飲まれる마셔지다
1그룹은 어미 하나가 い·あ·え·お 네 방향으로 갈라짐. 2·3그룹은 る 떼고 어미만 바뀜(食べられる·食べさせる·食べよう 등).

가능형 — 2·3그룹은 수동형과 동형(문맥 구별)

그룹규칙예
1그룹え단+る飲む→飲める, 話す→話せる
2그룹る떼고+られる食べる→食べられる
3그룹통암기する→できる, 来る→来られる
가능형 문형은 を가 아니라 が: 日本語が話せます(○) / 日本語を話せます(✗, 회화에선 흔히 씀)

사역형 — 시키다

그룹규칙예
1그룹あ단+せる飲む→飲ませる
2그룹る떼고+させる食べる→食べさせる
3그룹통암기する→させる, 来る→来させる
조사: 자동사(목적어 없음)→対象を / 타동사(목적어 있음)→対象に(원래 목적어는 を 유지)

수동형 — 당하다·되다

그룹규칙예
1그룹あ단+れる叱る→叱られる
2그룹る떼고+られる食べる→食べられる(가능형과 동형)
3그룹통암기する→される, 来る→来られる
迷惑の受身(민폐수동): 자동사에도 수동형 가능 — 雨に降られた(비가 와서 곤란했다), 子供に泣かれた(아이가 울어서 난처했다)

사역수동형 — 억지로 ~하게 됨

사역형에서 る 떼고 + られる: 飲ませる→飲ませられる
1그룹 축약(회화체): せられる→される — 飲まされる, 書かされる (단, す로 끝나는 동사는 축약 안 함: 話させられる)
私は母に野菜を食べさせられました。엄마 때문에 억지로 채소를 먹었습니다. — 강한 불만 뉘앙스

たい형 — 희망 (い형용사처럼 활용)

ます형에서 ます 떼고 + たい: 食べます→食べたい
활용: 食べたいです / 食べたくない / 食べたかった (い형용사 규칙 그대로)
3인칭은 たがる: 食べたがる(그가 먹고 싶어한다) — 화자 자신에만 たい 직접 사용
조사: 水が飲みたいです(원칙) — を도 회화에서 흔히 씀
