특수 규칙

ー 장음 부호 — 히라가나에 없는 전용 기호

앞 모음을 1박 연장.
コーヒー = kō·hī (커피)
ケーキ = kē·ki (케이크)
ソウル = so·u·ru (서울 — ー 없음!)
원음이 장음이 아니면 ー 안 씀.

ッ 촉음 — 받침 자음 흡수

작은 ッ → 다음 자음 1박 막기.
ベッド = be-d-do (bed)
ネット = ne-t-to (net)
バッグ = ba-g-gu (bag)
コップ = ko-p-pu (cup)

자음 연속 → 모음 끼워 넣기

일본어는 자음 단독 음절 없음(ン 제외).
"smart" → スマート (su·mā·to)
"strike" → ストライク (su·to·ra·i·ku)
자음 사이에 모음 삽입해 음절 분리.

헷갈리는 글자 쌍

シshi
vs
ツtsu

シ → 점 방향 ↘ (눕힌 느낌)
ツ → 점 방향 ↓ (세운 느낌)

ソso
vs
ンn

ソ → 획이 단순
ン → 끝이 안쪽으로 꺾여 올라감

ウu
vs
ワwa

ウ → 좌우 대칭
ワ → 오른쪽 획이 아래로 길게 내려옴

ノno
vs
メme

ノ → 사선 하나
メ → 두 획 교차 (X에 가까움)

クku
vs
タta

ク → 획 두 개
タ → 획 세 개 (위에 짧은 가로획 추가)

ナna
vs
スsu

ナ → 十자 + 사선
ス → 위 곡선 + 아래 작은 고리
