각 조사 상세

は wa 화제 표지 — 은/는
田中さんは先生です。 다나카 씨는 선생님입니다.
東京は大きいです。 도쿄는 큽니다.
は vs が: は = 화제("~에 대해선"), が = 주체("~이/가 한다")
田中さんは来ました。→ "다나카 씨는요, 왔어요." (화제)
田中さんが来ました。→ "다나카 씨가 왔어요!" (누가 왔냐 답)
공식: 모르는 것 → が / 알려진 것 이야기 → は
が ga 주어 표지 — 이/가
猫が魚を食べます。 고양이가 생선을 먹습니다.
초점 = が 앞 명사. "누가 했냐"를 강조할 때.
を o 목적어 표지 — 을/를
コーヒーを飲みます。 커피를 마십니다.
日本語を勉強します。 일본어를 공부합니다.
に ni 방향·위치·시간
① 방향: 学校に行きます。 학교에 갑니다.
② 위치·존재 (いる/ある와): 家にいます。 집에 있습니다.
③ 시간: 7時に起きます。 7시에 일어납니다.
に vs で:
に = 존재/도착/시간 → 学校にいます (학교에 있다)
で = 행위 장소 → 学校で勉強します (학교에서 공부한다)
で de 행위 장소·수단
① 행위 장소: 図書館で勉強します。 도서관에서 공부합니다.
② 수단·도구: 電車で行きます。 전철로 갑니다. / 箸で食べます。 젓가락으로 먹습니다.
と to 함께·열거
① 함께: 友達と映画を見ます。 친구와 영화를 봅니다.
② 열거(전부): りんごとバナナを買いました。 사과와 바나나를 샀습니다.
に vs と:
に = 일방향 → 先生に聞く (선생님께 묻다)
と = 쌍방향 → 先生と話す (선생님과 이야기하다)
の no 소유·수식
① 소유: 私の本。나의 책. / 田中さんのかばん。다나카 씨의 가방.
② 명사 수식: 日本語の先生。일본어 선생님.
③ 소유 대명사: これは母のです。이건 어머니 것입니다.
연속 사용 가능: 私の友達の本 → 나의 친구의 책
も mo 포함·역시 — 도
私も学生です。나도 학생입니다.
コーヒーも紅茶も好きです。커피도 홍차도 좋아합니다.
부정문: 의문사 + も + 부정 = "아무~도 ~않다"
何も食べません。아무것도 먹지 않습니다.
どこにも行きません。아무 데도 가지 않습니다.
か ka 의문·선택
これは本ですか。이것은 책입니까?
と vs か:
りんごとバナナ → 사과와 바나나 (둘 다)
りんごかバナナ → 사과나 바나나 (하나 선택)
