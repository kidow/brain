모양(´)은 성조 표시 아님. 중국어처럼 음을 올리는 게 아니라 "이 음절을 강하게"라는 위치 표시.
악센트의 세 가지 역할
역할 1
강세 위치 표시
규칙 예외일 때 표시

hablo = HA-blo (나는 말함)
habló = ha-BLÓ (그가 말했음)
역할 2
동음이의어 구분
발음은 같고 뜻만 다름

si (만약) vs sí (네)
el (정관사) vs él (그)
tu (너의) vs tú (너)
역할 3
이중모음 쪼개기
i/u + 강모음도 히아토로

hacia = a-cia (2음절)
hacía = ha-cí-a (3음절)
강세 기본 규칙 (악센트 없을 때)
단어 끝강세 위치예시
모음 / n / s끝에서 두 번째 음절ca-sa → CA-sa, co-men → CO-men
그 외 자음마지막 음절ha-blar → ha-BLAR, ciu-dad → ciu-DAD
악센트 표시된 단어 = 위 규칙의 예외. café (규칙대로면 CA-fe → 예외라 é 표시), árbol (규칙대로면 ar-BOL → 예외라 á 표시)
이중모음 (diptongo) — 두 모음이 한 음절로 합쳐짐
조건: i 또는 u + 다른 모음 (악센트 없을 때)
조합발음예시
ia"야"gracias = 그라-시아스
ie"예"bien = 비엔
ue"웨"bueno = 부에노
au"아우"auto = 아우토
ei"에이"seis = 세이스 (6)
히아토 (hiato) — 두 모음이 별도 음절로 분리됨
조건 1: 강한 모음(a·e·o) + 강한 모음 → 항상 히아토
조건 2: i/u에 악센트(í/ú) 있을 때 → 강제 히아토
