스페인어 발음 규칙

영어와 다른 글자 발음 + 변화 규칙

항상 고정된 발음
글자발음예시
h묵음 (항상 무음)hola = 올라, hombre = 옴브레
b / v동일 — "ㅂ"vino = bino = 비노
j목구멍 마찰음 "ㅎ"José = 호세, jota = 호타
ll"야/여/요" (y와 동일)llama = 야마, llegar = 예가르
ñ"냐/녜/뇨" (ny 소리)España = 에스파냐, año = 아뇨
rr혀 연속 트릴perro = 뻬rrro (개) ≠ pero = 뻬로 (그러나)
r (어두)항상 트릴rosa = rrosa, rico = rrico
c — 뒤 모음에 따라 변화
조합발음예시
c + a / o / u"ㅋ"casa = 카사, cosa = 코사
c + e / i중남미 "ㅅ"   스페인 "th"ciudad = 시우닷 / th유닷
g — 뒤 모음에 따라 변화
조합발음예시
g + a / o / u"ㄱ"gato = 가토, gustar = 구스타르
g + e / i"ㅎ" (j와 동일)gente = 헨테, girar = 히라르
gue / gui"게" / "기" (u = 묵음)guerra = 게라, guitarra = 기타라
güe / güi"궤" / "귀" (ü = 발음)cigüeña = 시귀에냐
x — 위치에 따라 변화
위치발음예시
모음 사이"ks"éxito = 엑시토 (성공)
자음 앞"s" 또는 "ks" 혼용explicar = 에스플리카르
고유명사 (멕시코 지명)"ㅎ" (j 소리)México = 메히코, Oaxaca = 와하카
z — 지역별 차이
발음예시
중남미"ㅅ" (s와 동일)zapato = 사파토
스페인영어 th 소리zapato = th아파토
z는 e/i 앞에 안 씀 → c 사용: cero(영), ciudad(도시) / a·o·u 앞엔 z: zapato, zona
que / qui — 묵음 u
조합발음예시
que"케"queso = 케소 (치즈), porque = 뽀르케
qui"키"quién = 키엔 (누구)
q는 항상 qu 세트. u는 항상 묵음.
